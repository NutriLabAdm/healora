# FOOD_DESCRIPTION.md — Нутрициологическое описание блюд

## Структура описания

Каждое блюдо описывается по схеме:

```
## [Название]
- **Тип приёма**: завтрак / обед / ужин / перекус
- **Ингредиенты**: [...]
- **КБЖУ**: ккал | белки | жиры | углеводы
- **NDI**: 0–10
- **Health Score**: 0–10

### 🧪 Микронутриентный профиль
- *Вещество* — источник — функция — кол-во (мг/мкг)

### 🔄 Биодоступность
- **Копробукты** (синергисты, повышают усвоение)
- **Ингибиторы** (снижают усвоение)

### 📊 Гарвардская тарелка
- Оценка по 7 осям (0–100): белки, овощи, углеводы, жиры, клетчатка, нутри-индекс, приготовление

### ⚙️ Метаболическая нагрузка
- Лёгкая / Средняя / Высокая — влияние на ЖКТ, печень, поджелудочную

### ❤️ Влияние на системы
- Сердечно-сосудистая
- Пищеварительная
- Нервная / Энергия
- Эндокринная (гликемическая нагрузка)

### 🔧 Оптимизация
- Замены для улучшения профиля
- Рекомендуемые комбинации
```

### 01. Котлеты с макаронами и салат

**Тип**: Обед  
**КБЖУ**: 620 ккал | 32г белки | 26г жиры | 58г углеводы  
**NDI**: 6.5 | **Health Score**: 6.5

**Ингредиенты**: котлеты (говядина/свинина), макароны, помидоры, огурцы, салат, масло растительное, лук

**Микронутриенты**:
- Железо гемовое (котлеты) ~2.5 мг — 2+

- Калий (огурцы, томаты) ~400 мг — сердце
- Витамин K (салат) ~80 мкг — свёртываемость

**Биодоступность**:

**Метаболическая нагрузка**: Средняя — сочетание жареного мяса и рафинированных углеводов

**Рекомендация**: Добавить больше свежих овощей до 50% тарелки. Заменить макароны на цельнозерновые. Котлеты запекать, а не жарить.

## Общие принципы комбинирования

### Золотые правила усвоения

1. **Железо** — мясо/рыба + витамин C (лимон, перец, зелень)
2. **Кальций** — молочные + витамин D + магний (избегать со шпинатом/щавелем)
3. **Каротиноиды** — овощи + масло/авокадо
4. **Куркумин** — чёрный перец (пиперин ↑ усвоение ×20)
5. **Пробиотики** — ферментированные продукты + клетчатка (пребиотики)
6. **Цинк** — устрицы/мясо + лимон/кислая среда (избегать со злаками без замачивания)
7. **Омега-3** — рыба + оливковое масло + антиоксиданты (зелень)
8. **Витамин D** — жирная рыба/яйца + жиры (сметана, масло) + магний

### Чего избегать

- Чай/кофе сразу после еды (танины ↓ железо на 60–80%) — пить через 1–1.5 ч
- Молочка + мясо (кальций ↓ гемовое железо)
- Шпинат/ревень + молочка (оксалаты ↓ кальций)
- Цельнозерновые без замачивания (фитаты ↓ Zn, Fe, Ca)
- Алкоголь + жирная пища (окислительный стресс ↓ усвоение B1, B9, B12, Zn)
