# Протокол: Контроль закупок
### ID: `PTL_PROCURE` (КБЖУ-баланс)

### Category: Питание

---

## 1. Цель

Обеспечение сбалансированного набора КБЖУ (калории, белки, жиры, углеводы) через структурированное планирование и контроль закупок продуктов. Исключение импульсивных покупок, снижение доли ультра-обработанных продуктов, гарантия наличия нутриентно-плотных продуктов дома.

---

## 2. Области применимости

- Любой протокол питания, требующий контроля макронутриентов (средиземноморская диета, кетодиета, интервальное голодание, DASH)
- Пользователи с дефицитом времени на ежедневное планирование
- Семейные закупки (синхронизация КБЖУ нескольких членов семьи)
- Переход с хаотичного питания на структурированное

## 4. Детальное описание интервенций

### PR_01: Планирование меню на неделю

**Когда:** Воскресенье, 20–30 мин
**Что:** Составить таблицу завтрак/обед/ужин/перекусы на 7 дней с учётом цели КБЖУ.

**КБЖУ-референс (Дмитрий, 57, 88→75 кг):**
- Калории: 1900–2000 ккал/день (дефицит −500)
- Белки: 110–130 г (1.3–1.5 г/кг, ~25% калорий)
- Жиры: 55–65 г (~25% калорий)
- Углеводы: 200–220 г (~45% калорий, с акцентом на сложные)

**Шаблон:**
```
| День | Завтрак       | Обед           | Ужин          | Перекус       |
|------|---------------|----------------|---------------|---------------|
| Пн   | Овсянка+ягоды | Салат+курица   | Рыба+овощи    | Яблоко+орехи  |
| Вт   | Яйца+авокадо  | Гречка+индейка | Тофу+брокколи | Груша+йогурт  |
| ...  |               |                |               |               |
```

### PR_02: Составление списка закупок по КБЖУ

**Когда:** Сразу после меню
**Что:** Выписать все ингредиенты, сгруппировать по отделам магазина.

**Группы:**
1. Белковые: курица, индейка, рыба, яйца, тофу, греческий йогурт, творог
2. Овощи/фрукты: листовая зелень, брокколи, цв. капуста, огурцы, помидоры, яблоки, ягоды (заморозка)
3. Сложные углеводы: овсянка, гречка, киноа, цельнозерновой хлеб
4. Жиры: оливковое масло, авокадо, орехи, семена чиа
5. Молочка: кефир, творог, твёрдый сыр (ограниченно)
6. Снеки: хумус, овощные палочки, протеиновый батончик (без сахара)

**Критерий:** Каждый пункт списка должен быть объяснён: «белок для завтрака», «клетчатка для обеда», «Омега-3 для ужина».

### PR_03: Чтение этикеток

**Правило 3 строк:**
1. Белки > 15 г на 100 г (для основного продукта)
2. Сахар < 5 г на 100 г
3. Длина состава не более 5 ингредиентов (чем короче — тем лучше)

**Что избегать:**
- Гидрогенизированные жиры (trans fats)
- Сироп глюкозно-фруктозный / HFCS
- Е-консерванты (E211, E320, E321)
- «0% жира» — обычно компенсируется сахаром

### PR_04: Правило корзины «70/30»

- **70% объёма корзины** — продукты с периметра магазина (овощи, фрукты, мясо, рыба, молочка)
- **30% объёма корзины** — продукты из центральных рядов (крупы, масла, специи, консервация без сахара)

**Визуальный чек-лист:**
- [ ] В корзине есть зелёные овощи
- [ ] Есть источник белка (животный или растительный)
- [ ] Есть цельнозерновые
- [ ] Нет сладких газировок и пакетированных соков
- [ ] Нет чипсов, крекеров, вафель

### PR_05: Заморозка порций (batch prep)

**Когда:** Суббота, 1–2 ч
**Что:** Приготовить 3–4 позиции впрок, разложить по контейнерам с наклейкой (дата + КБЖУ).

**Пример batch prep:**
- Запечённая куриная грудка × 4 порции (белок)
- Отварная гречка × 4 порции (сложные углеводы)
- Тушёные овощи (баклажаны, кабачки, перец) × 4 порции (клетчатка)
- Заправка: оливковое масло + лимон — отдельно

**Формат наклейки:** `18.05 — К300 Б25 Ж8 У32`

### PR_06: Челлендж «0 ультра-обработанных»

**1 раз в месяц — 7 дней без:** колбас, сосисок, магазинных соусов, сладких йогуртов, хлопьев для завтрака, газировки, пакетированных соков, фастфуда.

**Альтернатива:**
| Вместо | Берём |
|--------|-------|
| Колбаса | Запечённая курица/индейка |
| Магазинный соус | Оливковое масло + лимон + травы |
| Сладкий йогурт | Греческий + ягоды |
| Хлопья | Овсянка + орехи |

### PR_07: Аудит холодильника

**Чек-лист:**
- [ ] Проверить сроки годности — просрочку выбросить
- [ ] Убрать соусы/заправки с сахаром в первой тройке состава
- [ ] Пополнить запас ключевых продуктов: яйца, зелень, йогурт
- [ ] Оценить разнообразие: есть ли 5 разных овощей?

## 6. Красные флаги

| Показатель | Порог | Действие |
|------------|-------|----------|
| Импульсивные покупки | > 2/неделя | Подключить PR_01 + PR_02 |
| Пустой холодильник на утро | ≥ 3 раз/нед | Подключить PR_05 (batch prep) |
| Расход на доставку > бюджета | > 3 раза/нед | План закупок на 2 недели |
| «Я не знаю, что есть» в чате | > 2 раз/нед | Открыть список меню из PR_01 |

## 7. Интеграция с Дмитрием (57 лет, профиль)

**Уровень до внедрения:** Хаотичные закупки, доставка 3–4 раза/нед, в холодильнике нет структурированного набора продуктов, перекусы на работе — офисная еда.

**План внедрения (4 недели):**
- Нед 1: PR_01 + PR_02 (освоить планирование меню)
- Нед 2: PR_03 + PR_04 (научить читать этикетки и выбирать продукты)
- Нед 3: PR_05 (batch prep — сэкономить время в будни)
- Нед 4: PR_06 + PR_07 (аудит и контроль качества)

**Ключевые изменения:**
- До: 50% калорий из обработанных продуктов, доставка, спонтанные перекусы
- После: 80% калорий из цельных продуктов, готовый план на неделю, экономия ~3 ч/нед

## 9. За и Против

### За
- Исключает главную причину срывов — отсутствие правильных продуктов дома
- Переносит когнитивную нагрузку «что есть» на 1 раз в неделю
- Batch prep экономит 5–7 ч/нед на готовке

### Против
- Требует дисциплины в начале (первые 2 недели)
- Batch prep подходит не всем (нужен морозильник, контейнеры)
- Семейные закупки сложнее синхронизировать

---

## 10. Дискуссия

Протокол контроля закупок решает проблему, которую не адресуют классические диеты: **доступность правильных продуктов в момент голода**. Пока холодильник заполнен в соответствии с КБЖУ-планом, срыв на доставку или уличную еду менее вероятен. Протокол не требует знания КБЖУ от пользователя — все ориентиры рассчитываются цифровым двойником и передаются в список закупок.
