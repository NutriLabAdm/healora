# Протокол: Диагностика дефицитов витаминов и микроэлементов (МИИН)
### ID: `PTL_MIIN_DEF` (Алгоритм диагностического поиска МИИН)

### Category: Диагностика / Нутрициология

---

## 1. Цель

Системное выявление дефицитов витаминов и микроэлементов у условно здорового клиента в рамках компетенции нутрициолога с использованием четырёхшагового алгоритма диагностического поиска МИИН (Московский Институт Интегративной Нутрициологии).

---

## 2. Области применимости

- Клиенты с хронической усталостью, сниженной энергией, неспецифическими жалобами (выпадение волос, сухость кожи, мышечные подёргивания, тяга к сладкому)
- Период после беременности и грудного вскармливания (истощение запасов Fe, B12, йода, магния)
- Офисные работники с низкой инсоляцией и неструктурированным питанием
- Лица с высоким уровнем стресса (истощение магния, витамина C, группы B)
- Ситуации, когда врач исключил диагнозы (1-я группа здоровья), но клиент продолжает испытывать симптомы неоптимального состояния

---

## 3. Алгоритм диагностического поиска (4 шага)

### Шаг 1 — Первичный опрос и определение границ компетенции

| Действие | Описание |
|----------|----------|
| 1.1 | Сбор первичных негативных ощущений клиента |
| 1.2 | Оценка анамнеза (беременности, ГВ, диспансеризация, заболевания) |
| 1.3 | Выявление «красных флагов» — направление к врачу при подозрении на анемию, гипотиреоз, воспаление |
| 1.4 | Запрос базовой лабораторной диагностики (ОАК, ТТГ, ферритин, CRP) не старше 3 мес |
| 1.5 | Направление на УЗИ ОБП (исключение проблем с жёлчным пузырём) |

### Шаг 2 — Расширенный опросник + оценка ЖКТ

| Действие | Описание |
|----------|----------|
| 2.1 | Сбор дополнительных маркеров неоптимального состояния |
| 2.2 | Оценка признаков нарушения всасывания (жирный стул, вздутие, кровоточивость дёсен, гематомы) |
| 2.3 | У женщин — оценка менструального цикла (объём кровопотери, длительность, сгустки) |
| 2.4 | Формирование гипотез по дефицитам на основе кластеров симптомов |

### Шаг 3 — Анализ дневника питания

| Действие | Описание |
|----------|----------|
| 3.1 | Оценка дневника за 3–7 дней со шкалой голода 0–10 |
| 3.2 | Анализ макронутриентов (белок, жиры, углеводы, клетчатка) |
| 3.3 | Анализ микронутриентов (Fe, B12, йод, Омега-3, витамин C, D, цинк, магний) |
| 3.4 | Оценка режима питания, перекусов, качества еды |
| 3.5 | Выявление дефицитных паттернов |

### Шаг 4 — Лабораторное подтверждение (в рамках компетенции или через врача)

| Действие | Описание |
|----------|----------|
| 4.1 | Назначение целевых анализов для подтверждения гипотез |
| 4.2 | Интерпретация результатов в рамках референсных интервалов |
| 4.3 | Разработка плана коррекции (нутрицевтики + питание) |
| 4.4 | Контрольная диагностика через 3 мес |

## 4. Карта симптомов → дефициты (на примере кейса Светланы)

| Симптом | Возможные дефициты | Приоритет |
|---------|-------------------|-----------|
| Выпадение волос | Fe, биотин, витамин A, цинк, Омега-3, витамин D | 1 |
| Сухость кожи | Витамин A, цинк, биотин, Омега-3, витамин C, гидратация | 1 |
| Мышечные подёргивания | Mg, Ca, K, B6 | 2 |
| Тяга к сладкому | Mg, хром, B-комплекс, дисбаланс глюкозы | 2 |
| Кровоточивость дёсен | Витамин C | 1 |
| Лёгкое образование гематом | Витамин C, биофлавоноиды | 2 |
| Зябкость, холодные конечности | Fe, йод, селен, цинк (функция ЩЖ) | 2 |

| Ломкие ногти | Биотин, кремний, цинк, Fe | 2 |

## 5. Факторы истощения нутриентов

| Фактор | Истощаемые нутриенты |
|--------|---------------------|
| Беременность + ГВ (2+ года) | Fe, B12, йод, Ca, Mg, Омега-3 |
| Хронический стресс | Mg, витамин C, B-комплекс |
| Низкая инсоляция | Витамин D |
| Кофеин (>3 чашек/день) | Mg, B1, Fe (снижение абсорбции) |
| Высокое потребление сахара | Mg, Cr, B-комплекс |
| Неструктурированное питание («на бегу») | Белок, Fe, B12, клетчатка, Омега-3 |
| Вегетарианство/веганство | B12, Fe, Zn, Ca, Омега-3 |
| Приём ОК | B6, B12, фолаты, Mg, Zn |

## 6. Интервенции коррекции (нутрицевтики + питание)

| ID | Название | Дозировка | Примечание |
|----|----------|-----------|------------|
| MI_01 | Железо (хелат) | 20–40 мг/день | Контроль ферритина, приём с витамином C |
| MI_02 | Магний (цитрат/глицинат) | 300–500 мг/день | Вечером при нарушении сна |
| MI_03 | B-комплекс | Форма с активными кофакторами | B6 (P-5-P), B12 (метилкобаламин), фолат (5-MTHF) |
| MI_04 | Биотин | 1000–5000 мкг/день | Курс 3 мес при выпадении волос |
| MI_05 | Витамин D3 | 2000–5000 МЕ/день | С K2, контроль через 3 мес |
| MI_06 | Омега-3 (EPA+DHA) | 1000–2000 мг/день | Соотношение EPA:DHA 2:1 |
| MI_07 | Витамин C (липосомальный) | 500–1000 мг/день | При стрессе, кровоточивости дёсен |
| MI_08 | Цинк (пиколинат) | 15–30 мг/день | Краткосрочно, с медью 1:10 |
| MI_09 | Йод/селен | 150 мкг йода + 100 мкг селена | При подозрении на гипофункцию ЩЖ |
| MI_10 | Программа коррекции ЖКТ | Индивидуально | При нарушении всасывания |

## 7. Критерии эффективности (через 3 мес)

- Нормализация ферритина >50 мкг/л
- Витамин D >50 нг/мл
- Прекращение выпадения волос, появление нового роста
- Исчезновение сухости кожи и мышечных подёргиваний
- Стабилизация уровня энергии (отсутствие послеобеденной усталости)
- Исчезновение тяги к сладкому
- Нормализация стула (отсутствие жирного стула)

---

## 8. Красные флаги (направление к врачу)

- Анемия (Hb ниже нормы) — к терапевту/гематологу
- Выраженная меноррагия (>80 мл/цикл) — к гинекологу
- Наличие антител к ЩЖ (Ат-ТПО, Ат-ТГ) — к эндокринологу
- Кератоз (фолликулярный гиперкератоз) — к дерматологу
- Выпадение >35% волос в тесте натяжения — к трихологу
- Положительные маркеры воспаления (CRP >5) — к терапевту
- Облысение, залысины — к трихологу
