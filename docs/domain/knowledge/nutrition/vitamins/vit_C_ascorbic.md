# Витамин C (Аскорбиновая кислота)

**Категория:** Витамин  
**Тип:** Водорастворимый  
**Суточная норма (РФ, взрослые):** 90 мг (мужчины), 75 мг (женщины)  
**Суточная норма (США, RDA):** 90 мг (мужчины 19+), 75 мг (женщины 19+), +35 мг при курении  
**Верхний допустимый уровень (UL):** 2000 мг/сут (США), 2000–3000 мг (РФ)

## Что это такое

Витамин C (L-аскорбиновая кислота) — мощный водорастворимый антиоксидант, который человек не может синтезировать сам (мутация гена L-gulonolactone oxidase).

**Формы:**
- L-аскорбиновая кислота (основная, биоактивная)
- Натрия аскорбат, кальция аскорбат (мягкие формы для желудка)

## Функции в организме

- **Синтез коллагена:** кофактор пролил- и лизилгидроксилазы (здоровье кожи, сосудов, ран)
- **Иммунитет:** стимуляция фагоцитоза, выработка интерферона, защита лейкоцитов от окисления
- **Усвоение железа:** восстанавливает Fe3+ → Fe2+, повышая биодоступность негемового железа в 3–6 раз
- **Антиоксидант:** нейтрализация свободных радикалов, регенерация вит. E
- **Синтез нейромедиаторов:** норадреналина, серотонина
- **Карнитин:** синтез L-карнитина для энергетического обмена

## Признаки дефицита

- **Цинга:** кровоточивость дёсен, выпадение зубов, синяки, медленное заживление ран
- Сухая, шелушащаяся кожа
- Снижение иммунитета (частые простуды)
- Анемия (из-за плохого усвоения железа)
- Утомляемость, депрессия

## Признаки избытка

Витамин C не токсичен (выводится с мочой), но высокие дозы >1000 мг/сут могут вызвать:
- Диарею, тошноту, спазмы желудка
- Увеличение риска камней в почках (оксалатные камни) у предрасположенных
- У беременных: >2000 мг/сут — риск зависимости плода ( rebound скорбут)

## Биодоступность

**Усвоение:**
- Из пищи: **70–90%** (при дозах до 180 мг/сут)
- При дозах >1000 мг: усвоение падает до **50%** (кишечник насыщается)
- Из добавок: аналогично пище

**Энхансеры:**
- **Железо (негемовое):** вит. С повышает усвоение в 3–6 раз (обязательно сочетать для веганов!)
- **Биофлавоноиды (рутин, гесперидин):** содержатся в цитрусовых, усиливают эффект

**Ингибиторы:**
- **Термическая обработка:** 30–70% разрушается при варке, тушении
- **Хранение:** овощи/фрукты теряют 15–55% за 3–7 дней хранения
- **Курение:** требует +35 мг/сут (окислительный стресс)

## Взаимодействие

| Тип | Вещество | Описание |
|-----|----------|----------|
| Синергисты | Железо (Fe) | Снижает Fe3+ до Fe2+, повышает всасывание в 4–6 раз |
| Синергисты | Витамин E | Регенерирует окисленный вит. Е (цикл антиоксидантной защиты) |
| Антагонисты | Аспирин, НПВС | Повышают выведение вит. С с мочой |
| Антагонисты | Оральные контрацептивы | Снижают уровень вит. С в плазме |

## Особые группы риска дефицита

- **Курящие:** оксидативный стресс повышает потребность на 35 мг/сут
- **Пожилые:** низкое потребление свежих овощей/фруктов
- **Люди с мальабсорбцией:** целиакия, болезнь Крона, язвенный колит
- **Гемодиализ:** выводит вит. С из крови
- **Веганы и вегетарианцы:** при недостатке разнообразия растительной пищи

## Научные факты

- **Мета-анализ (Cochrane, 2013):** Вит. С не предотвращает простуду у большинства людей, но снижает **длительность болезни на 8% у взрослых и 14% у детей**.
- **Исследование (JAMA, 2020):** Высокие дозы вит. С (>500 мг/сут) не снижают риск сердечно-сосудистых заболеваний.
- **Российские данные:** Среднее потребление в РФ — 70–80 мг/сут (едва покрывает норму).

## Лайфхак для пользователя (Nutrition Tip)

> 🍊 **Съел бобовые или шпинат? Запей их стаканом апельсинового сока!** Витамин С повышает усвоение растительного железа в 4–6 раз. Для веганов это критично — без вит. С растительное железо почти не усваивается!
