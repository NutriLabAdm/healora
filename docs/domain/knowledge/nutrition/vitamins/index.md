## Ключевые синергии для NutriChat

| Группа | Взаимодействие | Значение для пользователя |
|--------|----------------|------------------------|
| **Fe + Vit C** | Вит. C повышает усвоение негемового Fe в 4–6 раз | Для веганов критично: добавлять лимон/перец к бобовым |
| **Ca + Vit D + K2** | D3 нужен для всасывания; K2 направляет Ca в кости | Без K2 высокие дозы D3 могут кальцифицировать сосуды |
| **B12 + B9 + B6** | Снижают гомоцистеин (риск инсульта) | Принимать как комплекс B-группы |
| **Vit A + Zn** | Цинк нужен для синтеза ретинол-связывающего белка | Дефицит Zn маскирует дефицит вит. A |
| **Вит. D + Mg** | Mg нужен для активации вит. D в организме | 50% населения не получают Mg, блокируя работу D3 |

## Методология

1. **Источники:** Только научные (NIH, USDA, Роспотребнадзор, WHO, публикации в рецензируемых журналах)
2. **Нормы:** Приоритет — РФ (МР 2.3.1.0253-21), дублируются нормы США (RDA/AI/UL)
3. **Биодоступность:** Указаны % усвоения, энхансеры и ингибиторы для каждого нутриента
4. **Синхронизация:** MD-файлы и JSON содержат одинаковые данные (JSON генерируется на основе MD)

---

> 📌 **Для разработчиков NutriChat:** Используйте `data.json` для подстановки норм в профиль пользователя. Используйте `foods_crosswalk.json` для расчёта микронутриентов в дневнике питания.
