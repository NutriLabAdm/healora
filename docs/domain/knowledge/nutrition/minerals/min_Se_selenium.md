# Селен (Selenium, Se)

**Категория:** Микроэлемент  
**Тип:** Минерал  
**Суточная норма (РФ, взрослые):** 70 мкг  
**Суточная норма (США, RDA):** 55 мкг (взрослые 19+)  
**Верхний допустимый уровень (UL):** 400 мкг/сут

## Что это такое

Селен — микроэлемент, входящий в состав селенопротеинов (глутатионпероксидаза, дейодиназа). Критичен для щитовидной железы и антиоксидантной защиты.

## Функции в организме

- **Щитовидная железа:** селенопротеины (дейодиназы) активируют Т4 → Т3
- **Антиоксидант:** глутатионпероксидаза (защита от перекисного окисления)
- **Иммунитет:** активация Т-клеток, вирусная нагрузка
- **Репродукция:** высокая концентрация в семенниках и сперме

## Признаки дефицита

- **Болезнь Кешана:** кардиомиопатия (эндемическая в Китае, где <10 мкг/сут)
- **Микседема (у детей):** пятнистая депигментация кожи
- **Снижение иммунитета** (частые вирусные инфекции)
- **Гипотиреоз** (нарушение активации Т4 → Т3)

## Признаки избытка

⚠️ **Избыток из добавок (>400 мкг/сут длительно):**
- **Селеноз:** ломкость ногтей, выпадение волос, дерматит, «чесночный» запах изо рта
- **Диабет 2 типа** (высокие дозы >800 мкг/сут, противоречиво)

## Биодоступность

**Усвоение:** 50–90% (из пищи), зависит от формы (селенометионин > селенат).

**Энхансеры:**
- **Витамин E:** совместная антиоксидантная защита
- **Белок в рационе:** улучшает усвоение

**Ингибиторы:**
- **Бериллий, мышьяк, кадмий:** тяжёлые металлы вытесняют Se
- **Почва с низким содержанием Se** (в РФ: большинство регионов)

## Взаимодействие

| Тип | Вещество | Описание |
|-----|----------|----------|
| Синергисты | Витамин E | Совместная антиоксидантная защита |
| Синергисты | Йод (I) | Кофактор дейодиназ (активация гормонов щитовидной железы) |
| Антагонисты | Тяжёлые металлы (Hg, Pb, Cd) | Вытесняют Se из ферментов |
| Антагонисты | Высокие дозы вит. C | Могут снижать статус Se |

## Особые группы риска дефицита

- **Жители регионов с низким содержанием Se в почве** (большинство регионов РФ)
- **Пациенты на гемодиализе** (выведение Se)
- **ВИЧ-инфицированные** (повышенная потребность)
- **Пациенты с болезнями щитовидной железы** (нужен для активации гормонов)

## Научные факты

- **Болезнь Кешана (Китай):** Кардиомиопатия при Se <10 мкг/сут. Добавки Se снизили смертность на 50%.
- **Рак простаты:** Низкий статус Se ассоциирован с повышенным риском рака простаты (снижение на 30–40% при добавках 200 мкг/сут).
- **РФ:** Содержание Se в почвах низкое (большинство регионов), дефицит имеют 40–60% населения.

## Лайфхак для пользователя (Nutrition Tip)

> 🥜 **Бразильский орех — суперфуд, но С ОСТОРОЖНО!** Всего 1–2 ореха покрывают суточную норму Se. Не ешьте >3 шт/сут — риск селеноза (выпадение волос, ломкость ногтей). Также отличные источники: морепродукты, семена подсолнеча, чеснок.
