# N-ацетилцистеин (NAC)

**Категория:** Витаминоподобное вещество  
**Тип:** Водорастворимое  
**Эндогенный синтез:** Нет (производное L-цистеина)  
**Основное действие:** Предшественник глутатиона

---

## Что это такое

NAC — ацетилированная форма L-цистеина, которая гидролизуется в организме до цистеина — лимитирующего субстрата синтеза **глутатиона** (ключевого внутриклеточного антиоксиданта). Также NAC напрямую восстанавливает дисульфидные связи за счёт свободной SH-группы.

## Функции в организме

- **Синтез глутатиона:** лимитирующий предшественник (цистеин)
- **Мукоактивное действие:** разжижение слизи (восстановление SS-связей муцина)
- **Детоксикация:** хелатирование тяжёлых металлов (Hg, Pb), конъюгация с электрофилами
- **Нейропротекция:** модуляция глутамата (NMDA-рецептор), снижение окислительного стресса
- **Противовоспалительное:** подавление NF-κB, снижение IL-6, TNF-α
- **Гепатопротекция:** антидот при передозировке парацетамола

## Признаки дефицита

- Снижение глутатиона → окислительный стресс
- Повышение маркеров воспаления (IL-6, CRP)
- Снижение детоксикационной способности печени
- Лёгочные заболевания (снижение мукоцилиарного клиренса)

## Признаки избытка

⚠️ NAC имеет широкий терапевтический диапазон. Редкие побочные эффекты:
- Тошнота, рвота (при дозах >2000 мг/сут per os)
- Головная боль, крапивница
- Бронхоспазм (редко, при ингаляции)

## Применение (терапевтическое)

- **Парацетамол (ацетаминофен) — антидот:** 140 мг/кг нагрузка, затем 70 мг/кг × 4-17 доз
- **ХОБЛ, бронхит:** муколитик (600-1200 мг/сут)
- **Психиатрия:** ОКР, трихотилломания, зависимость (1200-2400 мг/сут, уровень B)
- **Бесплодие (мужское):** улучшение качества спермы (600 мг/сут)
- **Синдром поликистозных яичников:** снижение инсулинорезистентности (уровень C)
- **Неалкогольная жировая болезнь печени:** снижение АЛТ/АСТ (1800 мг/сут)
- **Профилактика нефропатии:** после контрастных веществ (в/в)

## Продукты-источники

NAC не содержится в пище напрямую. Источник цистеина:
- Сывороточный протеин (WPC, WPI)
- Яйца (желток)
- Курица, индейка
- Йогурт, творог
- Чеснок (алицин → цистеин)

## Биодоступность

**Per os:** 10-30% (высокий эффект первого прохождения).  
**В/в:** 100% (при острой интоксикации).

**Энхансеры:**
- **Силимарин:** повышает синтез глутатиона синергично
- **Селен (Se):** кофактор глутатионпероксидазы
- **Витамин C:** регенерирует глутатион

## Взаимодействие

| Тип | Вещество | Описание |
|-----|----------|----------|
| Синергисты | Силимарин | Повышение синтеза глутатиона |
| Синергисты | Se, C, E | Синергия антиоксидантной защиты |
| Антагонисты | Алкоголь | Снижение синтеза глутатиона |
| Антагонисты | Активированный уголь | Связывание (интервал 2 ч) |

## Научные факты

- **Мета-анализ (2020):** NAC снижает IL-6 и CRP при метаболическом синдроме (PMID: 32686679)
- **ОКР:** 2400 мг/сут NAC снижает симптомы Y-BOCS на 30% (PMID: 23397850)
- **Глутатион:** пероральный NAC повышает уровень глутатиона в эритроцитах на 30-40% через 2 нед

## Лайфхак для пользователя

> NAC лучше принимать натощак за 1 ч до еды. Сочетать с Se 55-100 мкг и C 500 мг для максимального синтеза глутатиона.

## Промпты для презентации

### Для клиента
> **Заголовок:** «N-ацетилцистеин (NAC): антиоксидант, муколитик, защитник печени — что нужно знать»
>
> **Теория:**
> - NAC — предшественник глутатиона, главного антиоксиданта организма
> - Применение: ХОБЛ, ОКР, защита печени, детоксикация
> - Особенность: низкая биодоступность per os (10-30%)
> - Синергия: Se + C + силимарин
>
> **Практика:**
> - Стандартная доза: 600-1200 мг/день per os
> - Приём натощак, за 1 ч до еды
> - Курс 2-3 мес при хроническом воспалении
> - Сочетать с Se 55-100 мкг/день
> - Красный флаг: тошнота при >2000 мг/день
>
> **Дисклеймер:** NAC — пищевая добавка. При приёме антикоагулянтов — консультация врача.

### Для специалиста
> **Заголовок:** «NAC: клинический протокол — глутатион, детоксикация, воспаление»
>
> **Теория:**
> - NAC → цистеин → γ-ГКЦ → глутатион (GSH): лимитирующая стадия (PMID: 32686679)
> - Муколитик: восстановление дисульфидных связей муцина (N-ацетил-цистеин)
> - Нейропротекция: модуляция глутамата через cystine-glutamate antiporter (PMID: 23397850)
> - Антидот: связывание NAPQI — токсичного метаболита парацетамола
>
> **Практика:**
> - Дозировка: 1200-2400 мг/сут (разделить на 2 приёма)
> - Длительность: 8-12 нед для оценки эффекта (ОКР)
> - Мониторинг: трансаминазы (редкая гепатотоксичность)
> - Комбинации: NAC + Se + C + силимарин (синергия GSH)
> - Противопоказания: бронхиальная астма (ингаляция), цистеинурия
>
> **Дисклеймер:** NAC может снижать МНО при комбинации с варфарином. Мониторинг.
