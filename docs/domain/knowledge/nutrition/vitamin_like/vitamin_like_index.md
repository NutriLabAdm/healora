## Витаминоподобные вещества (Vitamin-like substances)

Витаминоподобные вещества — органические соединения, которые по химической структуре и биологической активности близки к витаминам, но не относятся к ним по одному из критериев:

- **Синтезируются в организме** (частично или полностью покрывая потребность)
- **Не являются незаменимыми** для всех возрастных групп
- **Требуются в бóльших количествах**, чем типичные витамины
- **Выполняют структурную или фармакологическую**, а не коферментную роль

> **Для NutriChat:** Эти вещества часто встречаются в BAD-добавках. Важно корректно классифицировать их как «витаминоподобные», а не витамины, и указывать, что для здорового человека синтез/поступление с пищей обычно достаточны.

## Ключевые синергии для NutriChat

| Группа | Взаимодействие | Значение для пользователя |
|--------|---------------|--------------------------|
| **CoQ10 + Жиры** | Усвоение убихинона требует жиров (в 4–6× больше при приёме с едой) | Принимать CoQ10 с жирной пищей, а не натощак |
| **Карнитин + АТФ** | L-карнитин необходим для β-окисления жирных кислот | Сжигание жира замедляется при дефиците карнитина |
| **Липоевая кислота + Глюкоза** | Активирует GLUT4 и AMPK, улучшает чувствительность к инсулину | Нутрицевтический подход к предиабету |
| **Флавоноиды + Витамин C** | Взаимно потенцируют антиоксидантное действие (30–40% усиление) | Есть цитрусовые целиком (с белой мезгой), а не только сок |
| **Инозитол + Холин** | Синергия в синтезе фосфатидилинозитола и липотропном эффекте | Важно для профилактики жирового гепатоза |
| **Флавоноиды + Железо** | Танины флавоноидов хелатируют Fe, снижая его усвоение на 50–70% | Чай/кофе — через 1–1.5 ч после еды с железом |

## Методология

1. **Источники:** Только научные (NIH, PubChem, Роспотребнадзор, EFSA, IOM, рецензируемые журналы)
2. **Статус:** Чёткое разделение на витамины, витаминоподобные вещества, фитонутриенты и БАД
3. **Нормы:** Для веществ с неопределённым RDA указан «ориентировочный адекватный приём» с пометкой
4. **Синтез:** Для каждого вещества указана способность организма к эндогенному синтезу
5. **Синхронизация:** MD-файлы и JSON содержат одинаковые данные (JSON генерируется на основе MD)

> 📌 **Для разработчиков NutriChat:** Используйте `data.json` для подстановки норм в профиль пользователя.
