# Пробиотики (Probiotics)

**Категория:** БАД / Микробиом  
**Тип:** Живые микроорганизмы  
**Основные роды:** Lactobacillus, Bifidobacterium, Saccharomyces boulardii  
**Синонимы:** Probiotics, пробиотические микроорганизмы, бактерии для кишечника  

## Что это такое

Пробиотики — живые микроорганизмы, которые при введении в адекватных количествах приносят пользу здоровью хозяина (определение ВОЗ/FAO). Наиболее изученные роды — Lactobacillus, Bifidobacterium и дрожжи Saccharomyces boulardii. Применяются для коррекции микробиома, восстановления барьерной функции кишечника и модуляции иммунного ответа.

## Внешний вид

Капсулы (белые/цветные — сухой лиофилизат), жидкие концентраты (мутная жидкость с осадком), порошок (белый/кремовый). Дрожжевые (S. boulardii) — капсулы/саше с кристаллической структурой

## Метаболический цикл

- **Прохождение через ЖКТ:** живые микроорганизмы должны выдержать кислоту желудка (pH 1–3, 30–60 мин), жёлчные кислоты (двенадцатиперстная кишка) и панкреатические ферменты. Выживаемость — 1–10% (Lactobacillus кислотоустойчивее Bifidobacterium). Капсулы с кишечнорастворимой оболочкой повышают выживаемость в 10–20 раз.
- **Колонизация:** транзиторная (не постоянная). Прекращение приёма → исчезновение из стула через 1–2 недели. Взаимодействие с резидентной микробиотой: конкуренция за рецепторы и субстраты, продукция бактериоцинов.
- **Метаболизм:** ферментация углеводов → КЦЖК (ацетат, лактат — Lactobacillus; ацетат, пропионат — Bifidobacterium). Синтез витаминов (B12, K2). Гидролиз жёлчных кислот (B SH). Антимикробная активность.
- **Выведение:** 90–99% погибают в ЖКТ. Живые — с калом (короткое окно, 1–3 дня).
- **Особенность:** для выживаемости — принимать натощак или с едой (pH 4–5), запивать тёплой водой (не кислым соком). Саше/порошок эффективнее капсул (кроме кислотоустойчивых капсул). Курс — 2–4 недели минимум.

## Механизмы действия

- **Конкурентное подавление патогенов** — занимают рецепторы на эпителии
- **Продукция антимикробных пептидов** — бактериоцины, лактоферрин
- **Укрепление плотных контактов** — снижение кишечной проницаемости
- **Модуляция иммунитета** — увеличение IgA, регуляция Treg
- **Продукция КЦЖК** — бутират, ацетат, пропионат (при наличии пребиотиков)
- **Синтез витаминов** — K2, B12, фолаты
- **Деконъюгация жёлчных кислот** — регуляция липидного обмена
- **Метаболизм пищевых волокон** — продукция КЦЖК

## Особенности приёма

- **Кол-во:** не менее 10⁹ КОЕ/день (для большинства штаммов)
- **Штаммоспецифичность** — эффект одного штамма не переносится на другие
- **Курс:** 4–12 недель (для хронических состояний — дольше)
- **Приём:** натощак или с едой (зависит от штамма)
- **Антибиотики:** разносить на 2–3 часа
- **Температура:** хранить согласно инструкции (не все требуют холодильника)

## Форма приема и порядок

Капсулы (кишечнорастворимые — 10⁹–10¹¹ КОЕ), порошок (саше), жидкие (ампулы/флаконы). Принимать во время или перед едой (максимальная выживаемость). При антибиотиках — разносить на 3 ч. Курс: 4–12 недель. Жидкие хранить в холодильнике. Хронические — 2–4 р/день

## Противопоказания

- Тяжёлый иммунодефицит (только под наблюдением врача)
- Центральные венозные катетеры (риск фунгемии с S. boulardii)
- Тяжёлый панкреатит (S. boulardii)
- Недоношенные с массой < 1000 г (только под контролем)


## Топ-5 продуктов, в которых содержится

Кефир (натуральный, живой), йогурт (без сахара, живой), квашеная капуста, мисо-суп (паста мисо), комбуча (чайный гриб, напиток)

## Взаимодействие

### Комплименты (синергисты)

| Ранг | Вещество | Эффект |
|------|----------|--------|
| +++ | Пребиотики (клетчатка, псиллиум, инулин) | Питательная среда для пробиотиков → рост колоний |
| ++ | Костный бульон | Глицин+глютамин → восстановление среды для микробиома |
| ++ | L-глютамин | Восстановление барьера → пробиотики лучше колонизируют |
| ++ | Бутират | Синергия: пробиотики продуцируют бутират, бутират питает колоноциты |
| ++ | Колострум | Иммуноглобулины + пробиотики — синергия иммунитета |
| + | Цинк | Кофактор для многих штаммов (Lactobacillus) |

### Ингибиторы (антагонисты)

| Ранг | Вещество | Эффект |
|------|----------|--------|
| +++ | Антибиотики | Уничтожение пробиотических штаммов — разносить на 2–3 ч |
| +++ | Высокая температура (> 40°C) | Гибель живых микроорганизмов |
| +++ | Противогрибковые | Уничтожение S. boulardii (дрожжи) |
| ++ | Алкоголь | Подавляет рост пробиотиков |
| + | ИПП (длительно) | Снижение pH → изменение выживаемости штаммов |

## Научные факты

- Кокрановский мета-анализ (2022): пробиотики снижают риск антибиотик-ассоциированной диареи на 50% (RR 0.53).
- Мета-анализ (2023): мультиштаммовые пробиотики достоверно уменьшают боль и вздутие при СРК.
- S. boulardii снижает риск рецидива C. difficile на 66%.

## Лайфхак для пользователя (Nutrition Tip)

> 🦠 **Пробиотики — микробиомная «инъекция».** Но это не лекарство на один раз: колонизация длится, пока вы кормите бактерии пребиотиками (клетчатка, овощи, фрукты). Курс 4–12 недель, а после — диета с достаточным количеством клетчатки. Для СРК ищите B. infantis 35624 или L. plantarum 299v; для профилактики диареи — S. boulardii.
