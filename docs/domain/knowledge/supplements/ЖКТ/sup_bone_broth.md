# Костный бульон, желатин, коллаген (Bone broth, Gelatin, Collagen)

**Категория:** Пищевой продукт / Нутрицевтик  
**Тип:** Заживление слизистых / Источник аминокислот  
**Основные действующие вещества:** Глицин, глутамин, пролин, гидроксипролин  

## Что это такое

Костный бульон — отвар из костей, хрящей и соединительной ткани животных. Содержит желатин (денатурированный коллаген), глицин, глутамин и другие аминокислоты, необходимые для восстановления слизистой ЖКТ.

## Внешний вид

Жидкость — от золотистого до янтарного цвета (в зависимости от костей и времени варки); при охлаждении образует гелеобразную массу (желатин)

## Метаболический цикл

- **Всасывание:** гидролизованные коллагеновые пептиды (глицин, пролин, гидроксипролин) всасываются в тонкой кишке через дипептидил-транспортеры (PEPT1/2, SLC15A1). Cmax гидроксипролина через 1–2 ч.
- **Распределение:** гидроксипролин — маркер доставки коллагена в соединительную ткань. Глицин — системно, через ГЭБ (транспортер GlyT1). Минералы (Ca, Mg, P) — ионный транспорт.
- **Метаболизм:** аминокислоты — включение в пул эндогенного коллагена (фибробласты, остеобласты). Глицин — участие в синтезе глутатиона и гема.
- **Выведение:** избыток аминокислот — дезаминирование в печени → мочевина → почки. T½ гидроксипролина 4–6 ч.
- **Особенность:** гидроксипролин в крови — косвенный биомаркер поступления коллагена. Усвоение улучшается при совместном приёме с витамином C.

## Механизмы действия

- **Даёт глицин** — противовоспалительная аминокислота, улучшает качество сна
- **Даёт глутамин** — топливо для энтероцитов
- **Содержит коллаген/желатин** — субстрат для соединительной ткани
- **Укрепляет слизистую** — делает её более «жирной» и эластичной
- **Поддерживает целостность эпителиального барьера**

## Показания (ЖКТ)

| Показание | Эффект |
|-----------|--------|
| Повышенная кишечная проницаемость | Укрепляет барьер |
| Восстановление после болезни | Глутамин + глицин |
| Дефицит белка / аминокислот | Дополнительный источник |
| Заживление слизистой | Субстрат для регенерации |

## Особенности приёма

- **Домашний бульон:** варить кости (говяжьи/куриные) 8–24 ч с овощами и травами
- **Желатин (порошок):** 10–15 г/сут, растворить в воде или бульоне
- **Коллаген (гидролизованный):** 10–20 г/сут, хорошо растворяется в холодной воде
- Для веганов/вегетарианцев — альтернатива: аминокислоты по отдельности

## Форма приема и порядок

Бульон (пить тёплым, 200–300 мл/день), порошок (коллагеновый — растворить в воде/супе), кубики (заморозка). Варить: 12–24 часа (говяжьи), 6–12 часов (куриные) с добавлением уксуса. Принимать натощак или с едой. Курс: длительно

## Противопоказания

- **Гистаминовая непереносимость** — длительная варка костей повышает содержание гистамина
- **Индивидуальная непереносимость** (редко)
- **При gout (подагре)** — пурины могут усугубить


## Топ-5 продуктов, в которых содержится

Говяжьи кости (суставные, мозговые), куриные кости/лапки, рыбий скелет (головы, хребты), свиные ножки/хвосты, бараньи кости

## Взаимодействие

### Комплименты (синергисты)

| Ранг | Вещество | Эффект |
|------|----------|--------|
| +++ | L-глютамин | Синергия заживления (глицин+пролин+глютамин) |
| ++ | Колострум | IGF-1 + глицин — синергия регенерации |
| ++ | Цинк-L-карнозин | Мембрана + коллаген — синергия восстановления |
| ++ | Пробиотики | Восстановление микробиома + барьера |
| ++ | Витамин C | Кофактор синтеза коллагена (гидроксилирование пролина/лизина) |

### Ингибиторы (антагонисты)

| Ранг | Вещество | Эффект |
|------|----------|--------|
| ++ | Алкоголь | Снижает синтез коллагена и регенерацию |
| ++ | ГКС | Катаболическое действие — разрушение коллагена |
| + | Оксалаты (шпинат, щавель) | Связывание кальция из костного бульона |

## Лайфхак для пользователя (Nutrition Tip)

> 🍲 **Костный бульон — «суп для кишечника».** Глицин, глутамин и коллаген в одной тарелке. Варить минимум 8 часов. При гистаминовой непереносимости — ограничить (длительная варка накапливает гистамин). Альтернатива: гидролизованный коллаген + глутамин по отдельности.
