# Мята перечная (Peppermint Oil, Mentha × piperita)

**Категория:** БАД / Фитотерапия  
**Тип:** Эфирное масло (спазмолитик)  
**Основное действующее вещество:** Ментол (30–50%), ментон (15–30%)  
**Синонимы:** Peppermint oil, масло перечной мяты, мятное масло  

## Что это такое

Эфирное масло мяты перечной (Mentha × piperita) — стандартизированный фитопрепарат с доказанным спазмолитическим действием на гладкую мускулатуру ЖКТ. Основной активный компонент — ментол, блокирующий кальциевые каналы (L-тип) в гладкомышечных клетках.

## Внешний вид

Эфирное масло: прозрачная жидкость от бесцветной до светло-жёлтой, с характерным мятным запахом и холодящим вкусом

## Метаболический цикл

- **Всасывание:** ментол (40–50% масла) и ментон — быстро всасываются в тонкой кишке (пассивная диффузия, липофильны). Cmax через 1–2 ч. Капсулы с кишечнорастворимой оболочкой — доставка в подвздошную/толстую кишку.
- **Метаболизм:** ментол → гидролиз до ментона/ментилацетата в печени (CYP2A6, CYP2B6). Глюкуронирование (UGT1A9, UGT2B7) → выведение с мочой.
- **Распределение:** ментол — связывание с TRPM8 (холодовые рецепторы) в ЖКТ, мышцах (местное спазмолитическое). Системно — через ГЭБ (частично).
- **Выведение:** T½ ментола ~2–4 ч. 70% почек (как глюкуронид ментола), 15% с выдыхаемым воздухом (ментол — летуч), 15% — трансдермально.
- **Особенность:** без кишечнорастворимой оболочки — расслабляет нижний пищеводный сфинктер (изжога). Ключевое правило: капсула с оболочкой = для кишечника, капли/чай = для тошноты/верхних отделов.

## Механизмы действия

- **Спазмолитическое** — блокада Ca²⁺ каналов L-типа → расслабление гладкой мускулатуры
- **Ветрогонное** — уменьшение газообразования и вздутия
- **Антиноцицептивное** — активация TRPM8-рецепторов → анальгезия
- **Прокинетическое** (тонкая кишка) — ускорение транзита (TRPM8)
- **Антимикробное** — ментол подавляет рост бактерий и грибков (SIBO, C. albicans)
- **Противовоспалительное** — снижение NF-κB, IL-6
- **Расслабление НПС** — GЭРБ (с осторожностью — может усиливать рефлюкс)

| Показание | Эффект | Уровень |

| Функциональная диспепсия | 1B | Уменьшает тяжесть, тошноту |

## Особенности приёма

- **СРК:** 180–225 мг × 2–3 р/день (кишечнорастворимые капсулы)
- **Курс:** 4–8 недель, при необходимости длительно
- **Приём:** за 30–60 мин до еды
- **Запивать водой, не разжёвывать** (высвобождение в тонкой кишке)
- **При ГЭРБ** — с осторожностью (ментол расслабляет НПС)

## Форма приема и порядок

Капсулы кишечнорастворимые (180–225 мг × 2–3 р/день при СРК), масло (для ароматерапии), чай (листья), капли (3–6 капель с водой). Только кишечнорастворимые капсулы — для действия в тонкой кишке. Принимать за 30–60 мин до еды. Курс: 4–8 недель

## Противопоказания

- ГЭРБ средней/тяжёлой степени (расслабление НПС)
- Грыжа пищеводного отверстия диафрагмы (с осторожностью)
- Желчнокаменная болезнь (ментол — желчегонное, может спровоцировать)
- Дети < 8 лет (ментол может вызвать апноэ/ларингоспазм)
- Язвенная болезнь (обострение)

## Научные факты

- Кокрановский мета-анализ (2020): масло перечной мяты в кишечнорастворимых капсулах достоверно уменьшает боль и вздутие при СРК (RR 2.12 по сравнению с плацебо, NNT=3–5).
- Мета-анализ (2018): масло перечной мяты более эффективно, чем спазмолитики и плацебо при СРК (NNT=3 vs NNT=5 для спазмолитиков).
- NICE Guideline (2022): масло перечной мяты — терапия первой линии при СРК.

## Лайфхак для пользователя (Nutrition Tip)

> 🌱 **Мятное масло — №1 при СРК (спазмы+вздутие).** Выбирайте только кишечнорастворимые капсулы (180–225 мг ментола), иначе будет изжога. NNT=3 — один из лучших показателей среди всех средств для СРК. При ГЭРБ — с осторожностью (ментол расслабляет нижний пищеводный сфинктер). Не для самолечения детям до 8 лет.
