# PEA — Пальмитоилэтаноламид (Palmitoylethanolamide)

**Категория:** Жирная кислота / Эндогенный медиатор  
**Тип:** Нутрицевтик / Противовоспалительное / Нейропротектор  
**Основное действующее вещество:** Пальмитоилэтаноламид  
**Синонимы:** PEA, Palmitoylethanolamide  

## Что это такое

PEA — эндогенный медиатор, синтезируемый в организме из пальмитиновой кислоты. Относится к семейству ацетилэтаноламидов (NAEs). Используется как мягкое и безопасное средство для уменьшения воспаления, особенно нейровоспаления и активации микроглии.

## Внешний вид

Белый кристаллический порошок (PEA) без запаха, со сладковатым вкусом

## Метаболический цикл

- **Всасывание:** липофильная молекула — пассивная диффузия в тонкой кишке. Частичное гидролизование ферментом FAAH в энтероцитах. Мицеллярная/липосомальная форма повышает биодоступность в 3–4 раза.
- **Метаболизм:** PEA — субстрат NAAA (N-ацилэтаноламин-гидролизующая кислая амидаза) → расщепление до этаноламина + пальмитиновой кислоты. Альтернативный путь — включение в мембранный пул (депо). PEA действует через PPAR-α и каналы TRPV1.
- **Распределение:** Vd — ограничен. Накопление в мембранах клеток (мозг, ЖКТ, кожа). Проникновение через ГЭБ — 5–10% (часть гидролизуется до входа).
- **Выведение:** T½ PEA 2–4 ч. Метаболиты — включение в β-окисление. Выведение — минимально (полностью метаболизируется).
- **Особенность:** для GI — местное действие на TRPV1 (висцеральная гиперчувствительность). Микронизированная/ультрамикронизированная форма — в 3–5 раз эффективнее стандартной (PeaPure, Normast).

## Механизмы действия

- **Активирует PPAR-α** — ядерный рецептор, регулирующий воспалительный ответ
- **Снижает активацию микроглии** — ключевой механизм при нейровоспалении
- **Укрепляет гематоэнцефалический барьер** (ГЭБ)
- **Модулирует тучные клетки** — снижает высвобождение провоспалительных медиаторов
- **Поддерживающая терапия аутоиммунных состояний**

## Показания

| Показание | Эффективность | Примечание |
|-----------|---------------|-----------|
| Нейровоспаление (постковид, ЧМТ) | Высокая | Укрепляет ГЭБ |
| Болезнь Альцгеймера | Поддерживающая | Снижает прогрессирование |
| Болезнь Паркинсона | Поддерживающая | Защита дофаминовых нейронов |
| Рассеянный склероз | Умеренная | Нейропротекция |
| Аутоиммунные заболевания | Поддерживающая | Безопасное противовоспалительное |
| Хроническая боль | Хорошая | Модуляция болевых сигналов |

## Особенности приёма

- **Действует медленно** — эффект через ~2 недели после начала приёма
- **Безопасный профиль** — практически нет побочных эффектов
- **Начинать с малых доз** для оценки переносимости

## Форма приема и порядок

Капсулы/таблетки (200–400 мг), порошок (сублингвально/орально). Формы: микронизированный PEA (лучшая биодоступность), PEA с лютеолином, ультрамикронизированный PEA. Принимать натощак или с едой. Курс: 8–12 недель

## Противопоказания

- Индивидуальная непереносимость (крайне редко)


## Топ-5 продуктов, в которых содержится

Соя (лецитин — следовые количества PEA), яичный желток (следы), не является диетическим компонентом; синтезируется эндогенно — не содержится в значимых количествах в продуктах питания

## Взаимодействие

### Комплименты (синергисты)

| Ранг | Вещество | Эффект |
|------|----------|--------|
| +++ | Омега-3 (EPA/DHA) | Синергия разрешения воспаления (SPMs + PEA) |
| ++ | Куркумин | Усиление противовоспалительного (разные рецепторы) |
| ++ | Лютеолин/кверцетин | Синергия стабилизации тучных клеток |
| + | Магний | Синергия при нейропатической боли |
| + | Витамин B12 | Дополнение при нейропатии |

### Ингибиторы (антагонисты)

| Ранг | Вещество | Эффект |
|------|----------|--------|
| + | Иммуносупрессоры | Теоретическое ослабление (PEA модулирует иммунитет) |
| + | Антикоагулянты | Минимальное потенцирование (теоретически) |
| + | Алкоголь | Снижает эффективность PEA (окислительный стресс) |

## Научные факты

- PEA медленно работает (2 недели до эффекта) — важно объяснить клиенту для комплаенса
- Укрепляет ГЭБ — особенно актуально после ковида (расшатанный ГЭБ)

## Лайфхак для пользователя (Nutrition Tip)

> 🧠 **PEA — мягкий нейропротектор при нейровоспалении.** Работает медленно (2 недели), но безопасно. Особенно хороша при постковиде, когнитивных нарушениях и аутоиммунных состояниях. Укрепляет гематоэнцефалический барьер. Принимать минимум 2 недели для оценки эффекта.
