# Бетаин (Betaine, Trimethylglycine, TMG)

**Категория:** Аминокислота (производное)  
**Тип:** Нутрицевтик / Метильный донор / Пищеварение  
**Основные формы:** Бетаин HCl, бетаин ангидрид (TMG)  
**Синонимы:** Триметилглицин, TMG, бетаин HCl  

## Что это такое

Бетаин — производное аминокислоты глицина с тремя метильными группами. В контексте ЖКТ имеет два применения: как **донор метильных групп** (бетаин ангидрид/TMG) и как **источник соляной кислоты** (бетаин HCl — для повышения кислотности желудка).

## Внешний вид

Белый кристаллический порошок (бетаин HCl) — гигроскопичный, с кисловатым вкусом; бетаин без HCl — сладковатый

## Метаболический цикл

- **Всасывание:** быстрое и полное в тонком кишечнике (натрий-хлор-зависимый транспортер BGT1, SLC6A12). Cmax через 30–60 мин.
- **Метаболизм:** донор метильных групп (→ метионин → S-аденозилметионин/SAMe → гомоцистеин → метионин). Метилирование — ключевая функция (эпигенетика, синтез карнитина, креатина). Альтернативный путь: окисление до диметилглицина (бетаин-гомоцистеин-метилтрансфераза, BHMT).
- **Распределение:** внутриклеточно (печень, почки). В плазме — минимально (регулируемый уровень).
- **Выведение:** T½ ~14 ч. Почечная экскреция (канальцевая реабсорбция — до 98%). Избыток (диетический) выводится с мочой.
- **Особенность:** двойное действие — осмопротектор (клетки мозгового вещества почек) + метаболический донор метила. При избытке — риск повышения холестерина (через SAMe).

## Механизмы действия

- **Донор метильных групп** — участвует в цикле метилирования (гомоцистеин → метионин)
- **Осмопротектор** — защищает клетки от осмотического стресса
- **Бетаин HCl** — временно повышает кислотность желудочного сока (при гипоацидности)
- **Пребиотик** — в форме свекольного кваса работает как пребиотик
- **Улучшает усвоение железа** (за счёт повышения кислотности)

## Показания

| Показание | Форма | Эффективность |
|-----------|-------|---------------|
| Гипоацидность / гипохлоргидрия | HCl | Временная, только после заживления слизистой |
| Нарушение метилирования (высокий гомоцистеин) | TMG | Хорошая |
| Нарушение усвоения железа | HCl | Условная (за счёт кислотности) |
| Дисбиоз | В пище (квас) | Пребиотическое действие |

## Особенности приёма

- **Бетаин HCl** — только после заживления слизистой! При гипоацидности — начинать с малых доз
- **При появлении дискомфорта** — отменить, поддержать слизистую, пройти доп. диагностику
- **Пищевые источники:** свёкла (свекольный сок, квас), шпинат, цельнозерновые

## Форма приема и порядок

Капсулы (бетаин HCl 500–650 мг + пепсин), порошок. Принимать в начале еды. Курс: только при недостаточности желудочной кислоты. Начинать с 1 капсулы, увеличивать до появления чувства тепла. Противопоказан при гастрите/язве

## Противопоказания

- Активная язва желудка
- Эрозивный гастрит (обострение)
- ГЭРБ (может усилить рефлюкс)
- Приём ингибиторов протонной помпы (бессмысленно)
- Гиперацидность


## Топ-5 продуктов, в которых содержится

Свёкла (самый богатый источник бетаина), шпинат, киноа, цельнозерновой хлеб, морепродукты

## Взаимодействие

### Комплименты (синергисты)

| Ранг | Вещество | Эффект |
|------|----------|--------|
| +++ | Панкреатин/пищеварительные ферменты | Восстановление кислотности → активация пепсина + ферментов |
| ++ | L-глютамин | Синергия восстановления слизистой (кислота — барьер) |
| ++ | Солодка DGL | Защита слизистой при восстановлении кислотности |
| + | Цинк-L-карнозин | Мембранная защита при повышенной кислотности |

### Ингибиторы (антагонисты)

| Ранг | Вещество | Эффект |
|------|----------|--------|
| +++ | ИПП (омепразол, пантопразол) | Противоположное действие (ИПП снижают, бетаин повышает кислотность) |
| +++ | Антациды | Нейтрализация эффекта бетаина |
| ++ | Н2-гистаминоблокаторы | Ослабление эффекта бетаина |
| ++ | НПВС | Риск повреждения слизистой при гиперацидности |

## Ключевое предупреждение

Бетаин HCl используется **только** после работы со слизистыми и снижения воспаления. Если начать с кислотности при изъязвлённой слизистой — будет болезненно и вредно.

## Лайфхак для пользователя (Nutrition Tip)

> 🧪 **Бетаин HCl — не для всех!** Только при подтверждённой гипоацидности и ТОЛЬКО после заживления слизистой. Свекольный квас — безопасная альтернатива для мягкой стимуляции пищеварения с пребиотическим эффектом. Не пить уксус натощак — лучше использовать лимонную воду через трубочку.
