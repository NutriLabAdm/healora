# Глутамин (L-Glutamine)

**Категория:** Аминокислота  
**Тип:** Нутрицевтик / Заживление слизистых  
**Основная форма:** L-глутамин  
**Суточная потребность (условно):** 5–10 г/сут (эндогенный синтез), в стрессовых состояниях — до 20 г/сут  

## Что это такое

Глутамин — условно незаменимая аминокислота. В нормальных условиях синтезируется в организме, но при стрессе, болезнях и повреждениях ЖКТ его собственная продукция становится недостаточной. Является основным топливом для энтероцитов (клеток кишечного эпителия) и иммунных клеток.

## Внешний вид

Белый кристаллический порошок без запаха, с нейтральным вкусом (слегка сладковатый, напоминает сахарную пудру)

## Метаболический цикл

- **Всасывание:** активный натрий-зависимый транспорт (B⁰AT1, SLC6A19; SN2, SLC38A5) в тонкой кишке. Наиболее активно — в тощей кишке. Cmax через 30–45 мин. До 70–80% абсорбированного глутамина метаболизируется в энтероцитах.
- **Метаболизм:** энтероциты: глутамин → глутамат (глутаминаза) → α-кетоглутарат → цикл Кребса (энергия). Печень: глутамин → синтез мочевины. Почки: глутамин → аммоний (регуляция pH). Иммунные клетки: субстрат для пролиферации.
- **Распределение:** самая распространённая свободная аминокислота в крови (500–700 мкМ). 60% в скелетных мышцах (главное депо). Транспорт через ГЭБ — через систему L (LAT1).
- **Выведение:** почечная экскреция (канальцевая реабсорбция — практически полная, порог ~600 мкМ). При распаде мышц — увеличивается экскреция.
- **Особенность:** при стрессе (травма, сепсис, интенсивные тренировки) потребность возрастает в 3–5 раз. Для ЖКТ — предпочтителен приём натощак (эффект на энтероциты). Для мышц — после тренировки.

## Механизмы действия

- **Топливо для энтероцитов** — основное питательное вещество для клеток слизистой кишечника
- **Субстрат синтеза глутатиона** — главного антиоксиданта организма
- **Участвует в аммониогенезе в почках** — выведение кислоты при ацидозе
- **Субстрат для синтеза мочевины в печени** — отведение аммиака
- **Укрепляет межклеточные контакты** — уменьшает кишечную проницаемость (leaky gut)
- **Предшественник ГАМК** — через глутамат (при нарушении баланса глутамат-ГАМК → кошмары, гиперактивность)

## Показания

| Показание | Эффективность | Примечание |
|-----------|---------------|-----------|
| Повышенная кишечная проницаемость | Высокая | Укрепляет плотные контакты |
| Стресс | Хорошая | Синтез глутатиона, защита от окислительного стресса |
| Сниженный уровень белка | Хорошая | Аминокислота для поднятия общего белка |
| Гипераммониемия | Высокая | Синтез мочевины |
| Ацидоз | Высокая | Аммониогенез в почках |
| Профилактика повреждения ЖКТ при химио/лучевой терапии | Доказанная | Предотвращает атрофию ворсинок |
| Бактериальная транслокация | Высокая | Укрепляет стенку → уменьшает проникновение бактерий в кровоток |

## Особенности приёма

- **Профилактическая дозировка:** 1 г/сут (натощак)
- **Терапевтическая дозировка:** 5–15 г/сут, разделить на 2–3 приёма (иногда до 20–30 г/сут при критических состояниях — только под наблюдением)
- **Принимать натощак** — для лучшего всасывания
- **Курс:** 1–3 месяца, при необходимости длительнее (безопасен при нормальной переносимости)

## Форма приема и порядок

Порошок (растворяется в воде/соке, 5–10 г на приём), капсулы (500–1000 мг). Принимать натощак (за 30–40 мин до еды или через 2 ч после), отдельно от других аминокислот. Курс: 4–12 недель. Хорошо сочетается с пробиотиками и колострумом

## Побочные эффекты и меры предосторожности

| Симптом | Действие |
|---------|----------|
| Кошмары, гиперактивность | Нарушение баланса глутамат-ГАМК — отменить и работать с этим соединением альтернативно |
| Запоры | Глутамин может крепить |
| Серная чувствительность | Глутамин — серная аминокислота, при непереносимости серы не подходит |

## Противопоказания

- Тяжёлые заболевания печени (цирроз, печёночная энцефалопатия) — может усилить гипераммониемию
- Серная чувствительность
- Индивидуальная непереносимость


## Топ-5 продуктов, в которых содержится

Костный бульон (самый богатый источник), говядина, курица, рыба, яйца, молочные продукты (творог, сыр), капуста, шпинат, бобовые

## Взаимодействие

### Комплименты (синергисты)

| Ранг | Вещество | Эффект |
|------|----------|--------|
| +++ | Коллоидный серебро/BERBERIN (антимикробные) | Подготовка среды → глютамин восстанавливает |
| ++ | Костный бульон | Глицин+пролин+глютамин — синергия заживления |
| ++ | Цинк-L-карнозин | Разные механизмы: метаболизм (Gln) + мембрана (ZnC) |
| ++ | Колострум | IGF-1 + глютамин — синергия пролиферации энтероцитов |
| + | Пробиотики | Восстановление среды → глютамин питает энтероциты |

### Ингибиторы (антагонисты)

| Ранг | Вещество | Эффект |
|------|----------|--------|
| ++ | Алкоголь | Снижает всасывание и утилизацию глютамина |
| ++ | НПВС | Повреждают слизистую → увеличивают потребность в глютамине |
| + | Цитостатики (метотрексат) | Конкуренция за транспортёры |
| + | ГКС | Катаболическое действие → снижение пула глютамина |

## Лайфхак для пользователя (Nutrition Tip)

> 🥩 **Глутамин — лучшее топливо для кишечника.** При стрессе, после болезни, при нарушении пищеварения — это первая добавка для восстановления слизистой. Профилактическая доза: 1 г натощак. Если появились кошмары или гиперактивность — отмените (нарушение баланса глутамат-ГАМК).
