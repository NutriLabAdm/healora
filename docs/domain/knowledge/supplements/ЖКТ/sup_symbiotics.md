# Симбиотики (Symbiotics — комбинации пробиотических штаммов)

**Категория:** БАД / Микробиом  
**Тип:** Комбинация микроорганизмов (несколько штаммов/видов)  
**Основные компоненты:** Несколько штаммов Lactobacillus, Bifidobacterium, Enterococcus, Streptococcus, Saccharomyces и др. в одном препарате  
**Синонимы:** Симбиотики, поликомпонентные пробиотики, мультиштаммовые пробиотики  

## Что это такое

Симбиотики (от греч. «symbiosis» — сожительство, совместная жизнь) — это комбинации нескольких видов и/или штаммов пробиотических микроорганизмов в одном препарате. В отличие от синбиотиков (пробиотики + пребиотики) и монопробиотиков (один штамм), симбиотики содержат сразу несколько культур, которые теоретически дополняют друг друга, занимая разные экологические ниши. Важно: термин «симбиотики» часто путают с «синбиотиками» — это разные понятия.

## Внешний вид

Капсулы (желатиновые/растительные — сухой лиофилизат, порошок кремового/бежевого цвета), жидкие концентраты (мутная жидкость с осадком), порошок/саше (для растворения)


## Метаболический цикл

- **Прохождение через ЖКТ:** пробиотическая составляющая — выживание в кислой среде желудка (кислотоустойчивые штаммы + капсулы с кишечнорастворимой оболочкой). Пребиотическая составляющая — не всасывается в тонкой кишке, достигает толстой кишки интактной.
- **Метаболизм:** ферментация пребиотиков микробиотой → КЦЖК (ацетат, пропионат, бутират). Синергия: пробиотики используют пребиотики как субстрат → усиление колонизации.
- **Распределение:** преимущественно локально в толстой кишке. КЦЖК — 5–10% в системный кровоток.
- **Выведение:** пробиотики — транзиторная колонизация (1–3 недели после прекращения приёма). Пребиотики — 5–10% с калом (неферментированная часть).
- **Особенность:** синергия пре- + пробиотиков. Выбор штаммов под конкретную задачу (иммунитет — Lactobacillus, барьер — Bifidobacterium). Курс 4–12 недель.

## Механизмы действия

- **Поликомпонентная колонизация** — разные штаммы занимают разные участки/ниши кишечника
- **Разные механизмы** — каждый штамм имеет свой спектр активности (продукция бактериоцинов, лактата, КЦЖК, витаминов)
- **Синергия штаммов** — один штамм может создавать среду для другого
- **Более широкий спектр действия** — охват разных показаний (запоры + иммунитет + противовоспалительное)
- **Конкурентное ингибирование патогенов** — разные штаммы подавляют разные патогены
- **Снижение риска доминирования** — меньший риск дисбиоза при длительном приёме

## Показания

| Показание | Эффект | Уровень |
|-----------|--------|---------|
| Дисбиоз (неуточнённый) | 2A | Широкий охват |
| Антибиотик-ассоциированная диарея (профилактика) | 1A | Лучше монопробиотиков |
| Синдром раздражённого кишечника | 1A | Мультиштаммовые > моноштаммовые |
| Восстановление после гастроэнтерита | 1B | Комплексное восстановление |
| Атопический дерматит | 1B | В перинатальном периоде |
| Функциональные запоры | 1B | Разные штаммы — разные механизмы |

## Топ-5 продуктов, в которых содержится

Симбиотики как мультиштаммовые препараты не встречаются в природе в готовом виде. Ближайшие природные аналоги (естественные мультивидовые сообщества): кефир (10+ видов бактерий + дрожжи), комбуча (симбиотическая культура бактерий и дрожжей — SCOBY), квашеная капуста (спонтанное брожение — несколько видов Lactobacillus), закваска для хлеба (молочнокислые бактерии + дрожжи), сыры с живой плесенью и бактериями

## Особенности приёма

- **Дозировка:** 10⁹–10¹¹ КОЕ/день (суммарно по всем штаммам)
- **Форма:** капсулы (кишечнорастворимые), жидкие, саше
- **Приём:** во время или перед едой
- **Курс:** 4–12 недель
- **С антибиотиками:** разносить на 3 часа
- **Выбор:** отдавать предпочтение штаммам с доказанной эффективностью (не просто «10 штаммов», а клинически изученные)
- **Потенциальный риск:** при СИБР мультиштаммовые пробиотики могут усугубить вздутие — пробный приём 3–5 дней

## Противопоказания

- Тяжёлый иммунодефицит
- Острый панкреатит (S. boulardii — осторожно)
- СИБР (тяжёлый — пробный приём с осторожностью)
- Индивидуальная непереносимость компонентов

## Форма приема и порядок

Капсулы (кишечнорастворимые), порошок (саше), жидкие концентраты. Принимать во время или перед едой (максимальная выживаемость). При антибиотиках — разносить на 3 ч. Жидкие — только из холодильника. Курс: 4–12 недель. При хронических заболеваниях — дробный приём 2–4 р/день. При повышенной кислотности — предпочесть капсулированную форму.

## Взаимодействие

### Комплименты (синергисты)

| Ранг | Вещество | Эффект |
|------|----------|--------|
| +++ | Пребиотики (клетчатка, инулин) | Питание для всех штаммов |
| ++ | Метабиотики | Восстановление среды → симбиотики лучше колонизируют |
| ++ | Бутират | Дополнительная энергия для колоноцитов |
| + | Колострум | Иммуноглобулины + пробиотики |

### Ингибиторы (антагонисты)

| Ранг | Вещество | Эффект |
|------|----------|--------|
| +++ | Антибиотики | Уничтожение части/всех штаммов — разносить на 3 ч |
| ++ | Противогрибковые | Уничтожение дрожжевых штаммов (S. boulardii) |
| ++ | Алкоголь | Подавление роста пробиотиков |
| + | ИПП (длительно) | Изменение pH → снижение выживаемости |

## Научные факты

- Мета-анализ (2022): мультиштаммовые пробиотики эффективнее монопробиотиков при СРК (RR 1.7 vs RR 1.3, p=0.02).
- Систематический обзор (2021): комбинация L. rhamnosus GG + B. lactis BB-12 эффективнее каждого по отдельности при антибиотик-ассоциированной диарее.
- РКИ (2020): препарат VSL#3 (8 штаммов) достоверно снижает рецидивы язвенного колита.

## Лайфхак для пользователя (Nutrition Tip)

> 🧬 **Симбиотики — пробиотическая «команда».** В отличие от монопробиотиков (один штамм) и синбиотиков (бактерии + еда для них), симбиотики — это несколько штаммов в одной капсуле. Теория: чем больше штаммов, тем шире охват. Практика: важны не количество, а клинически доказанные штаммы. При СИБР — осторожно (может усиливать вздутие). Самый изученный мультиштаммовый — VSL#3. Аналог из еды — кефир (10+ видов).
