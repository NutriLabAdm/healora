# Метабиотики / Постбиотики (Postbiotics / Metabiotics)

**Категория:** БАД / Микробиом  
**Тип:** Метаболиты пробиотических микроорганизмов  
**Основные компоненты:** Короткоцепочечные жирные кислоты (КЦЖК — бутират, пропионат, ацетат), ферменты, бактериоцины, витамины (K₂, B₁₂, фолаты), аминокислоты, пептидогликаны, липотейхоевые кислоты, экзополисахариды  
**Синонимы:** Postbiotics, метабиотики, постбиотики, парапробиотики, инактивированные пробиотики, пробиотические метаболиты, «какашки» бактерий  

## Что это такое

Метабиотики (постбиотики) — метаболиты жизнедеятельности пробиотических микроорганизмов, продукты их ферментации, а также инактивированные (убитые) пробиотические клетки и их компоненты. Термин «метабиотики» подчёркивает, что это «то, что бактерии производят, а не сами бактерии». Определение ISAPP (2021): «препарат из неживых микроорганизмов и/или их компонентов, который приносит пользу здоровью хозяина».

Ключевое преимущество: метабиотики работают сразу (не требуют колонизации), не зависят от кислотности желудка, не могут размножаться, безопасны при СИБР.

## Внешний вид

Капсулы (порошок светлого/кремового цвета — сухие метаболиты), жидкие концентраты (жёлтая/коричневая жидкость с кисломолочным/сырным запахом), порошок (для растворения)

## Метаболический цикл

- **Всасывание:** КЦЖК (бутират, пропионат, ацетат) — пассивная диффузия и транспорт через MCT1 (SLC16A1) в эпителии толстой кишки. Бутират — 70–80% утилизируется колоноцитами. Пропионат — в портальный кровоток (печень). Ацетат — системно.
- **Распределение:** бутират — локально (толстая кишка). Пропионат — глюконеогенез в печени. Ацетат — холестерин, синтез жиров (печень). Ферменты (супероксиддисмутаза) — местно, в просвете кишки.
- **Метаболизм:** бутират → β-окисление в колоноцитах (85% энергии). Пропионат → глюконеогенез. Ацетат → липогенез.
- **Выведение:** CO₂ (дыхание), минимально — почки (<5%). T½ бутирата — 6–10 мин в плазме (быстрый клиренс).
- **Особенность:** в форме постбиотиков (инактивированные бактерии + метаболиты) — стабильнее и безопаснее живых пробиотиков. Не требуют холодильника. Безопасны при иммунодефиците.

## Механизмы действия

- **Создание мукозного слоя** — стимуляция продукции слизи
- **Энергия для колоноцитов** — КЦЖК (бутират) — 70% энергии толстой кишки
- **Укрепление плотных контактов** — снижение кишечной проницаемости
- **Иммуномодуляция** — сигналинг через GPR41, GPR43, GPR109A → Treg
- **Противовоспалительное** — ингибирование NF-κB (бутират — HDACi)
- **Антимикробное** — бактериоцины (лактоцин, низин, реутерин)
- **Пребиотическое** — создание среды для роста нормофлоры
- **Витаминообразование** — витамин K₂, B₁₂, фолаты
- **Нейромодуляция** — ГАМК, серотонин, дофамин (психобиотики)

## Показания

| Показание | Эффект | Уровень |
|-----------|--------|---------|
| СИБР (избыточный бактериальный рост) | 2A | Безопасны, не усугубляют |
| Синдром повышенной проницаемости | 1B | Укрепление плотных контактов |
| Воспалительные заболевания кишечника | 2A | HDACi — противовоспалительное |
| Антибиотик-ассоциированная диарея | 1B | Профилактика + восстановление |
| Тотальный дисбиоз (3+ курса антибиотиков) | 2B | Подготовка для пробиотиков |
| Метаболический синдром | 2A | КЦЖК → улучшение инсулинорезистентности |
| Атопический дерматит | 2A | Иммуномодуляция |
| Функциональные запоры | 2B | Стимуляция моторики |
| Тревожность / депрессия (психобиотики) | 2A | ГАМК, серотонин через ось «кишечник-мозг» |

## Топ-5 продуктов, в которых содержится

Кефир (метаболиты кефирных грибков — КЦЖК, витамины, бактериоцины), комбуча (органические кислоты, витамин B, ферменты), квашеная капуста (молочная кислота, витамин K₂, бактериоцины), мисо-паста (ферментированная соя — метаболиты Aspergillus), сыры выдержанные (пармезан, гауда — КЦЖК, аминокислоты, бактериоцины из заквасок), кимчи (молочная кислота, витамины, пептиды)

## Особенности приёма

- **Форма:** капсулы, жидкие концентраты, порошок
- **Дозировка:** по инструкции производителя (стандартизация отсутствует)
- **Приём:** независимо от еды (не требуют защиты от желудочного сока)
- **Курс:** 2–4 недели (базовый курс) → затем синбиотики или пробиотики
- **Не нагревать** (жидкие формы — можно только тёплое)
- **Лучшая стратегия:** сначала метабиотики (восстановление среды) → затем пробиотики/синбиотики (заселение)
- **Хранение:** капсулы — комнатная температура; жидкие — холодильник

## Противопоказания

- Индивидуальная непереносимость компонентов
- Нет ограничений при иммунодефиците (в отличие от живых пробиотиков)
- Безопасны при СИБР (отличительная особенность)

## Форма приема и порядок

Капсулы, жидкий концентрат, порошок. Принимать независимо от еды. Курс: 2–4 недели (первый этап) → затем переход на пробиотики/синбиотики. При СИБР — принимать без ограничений. Схема: метабиотики (2–4 нед, создать мукозный слой) → синбиотики (4–12 нед, заселение) → пребиотики (длительно, поддержка).

## Взаимодействие

### Комплименты (синергисты)

| Ранг | Вещество | Эффект |
|------|----------|--------|
| +++ | Синбиотики | Метабиотики создают среду → синбиотики заселяют |
| +++ | Пребиотики | Естественная продукция метабиотиков из пребиотиков |
| ++ | L-глютамин | Метабиотики + глютамин — синергия восстановления барьера |
| ++ | Колострум | Иммуноглобулины + метабиотики — синергия иммунитета |

### Ингибиторы (антагонисты)

| Ранг | Вещество | Эффект |
|------|----------|--------|
| ++ | Антибиотики | Не влияют на метабиотики (не живые) — преимущество |
| ++ | ИПП (длительно) | Изменение pH толстой кишки — снижение эффекта КЦЖК |
| + | Слабительные | Ускорение транзита — снижение времени контакта |

## Научные факты

- Консенсус ISAPP (2021): постбиотики имеют доказанную пользу при инфекционной диарее, иммуномодуляции и СРК.
- РКИ (2022): метабиотики снижают водородный дыхательный тест (СИБР) на 40% через 4 недели без антибиотиков.
- Исследование (2020): метабиотики (postbiotics) быстрее и безопаснее пробиотиков при восстановлении после антибиотиков (создание среды за 2 нед vs 4 нед).

## Лайфхак для пользователя (Nutrition Tip)

> 🦠 **Метабиотики — «полезные отходы» бактерий.** В отличие от пробиотиков (живые бактерии) не требуют колонизации — работают сразу. Идеальны для: 1) СИБР (не усугубляют); 2) тотального дисбиоза (подготовка среды); 3) старта восстановления микробиома. Стратегия: метабиотики (2–4 нед) → синбиотики (4–12 нед) → пребиотики (длительно). Кефир и комбуча — природные аналоги метабиотиков.
