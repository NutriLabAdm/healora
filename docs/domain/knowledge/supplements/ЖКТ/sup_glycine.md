# Глицин (Glycine)

**Категория:** БАД / Аминокислота  
**Тип:** Протеиногенная аминокислота  
**Основное действующее вещество:** Аминоуксусная кислота (Glycine)  
**Синонимы:** Glycine, аминоуксусная кислота, гликокол  

## Что это такое

Глицин — простейшая протеиногенная аминокислота. Играет множественные роли в организме: конъюгация жёлчных кислот (образует гликохолевую кислоту), синтез глутатиона, нейромедиатор (ГАМК-ергический и NMDA-рецепторный сигналинг), участие во II фазе детоксикации печени (конъюгация с глицином), синтез гемоглобина и метаболизм головного мозга.

## Внешний вид

Белый кристаллический порошок, сладковатый на вкус (в 0.8 раз слаще сахара), хорошо растворим в воде

## Метаболический цикл

- **Всасывание:** натрий-зависимые транспортеры глицина (GlyT1, SLC6A9; IMINO, SLC6A20) в тонкой кишке. Cmax через 30–60 мин. Всасывание — быстрое и полное.
- **Метаболизм:** три основных пути: (1) синтез глутатиона (γ-глутамилцистеин → GSH) — ключевой антиоксидант; (2) синтез гема (δ-аминолевулиновая кислота); (3) синтез креатина (глицин + аргинин → гуанидиноацетат). Катаболизм: глицин → серин → пируват (глицин-расщепляющая система).
- **Распределение:** глицин — нейротрансмиттер (спинной мозг, ствол мозга). Транспорт через ГЭБ — через глициновые транспортеры (GlyT1). В плазме — 200–400 мкМ.
- **Выведение:** T½ глицина 0.5–2 ч. Почечная экскреция (канальцевая реабсорбция — пороговое значение). Избыток — окисление до CO₂ + H₂O.
- **Особенность:** для ЖКТ — действие через синтез глутатиона + коллагена. Для сна — агонист NMDA-глицинового сайта спинного мозга (миорелаксация). Доза для сна 3 г перед сном.

## Механизмы действия

- **Конъюгация жёлчных кислот** — образование гликохолевой кислоты (вторичные жёлчные кислоты)
- **Детоксикация II фаза** — конъюгация с глицином → выведение бензойной кислоты и её производных
- **Предшественник глутатиона** — субстрат для синтеза GSH
- **ГАМК-ергическое** — активирует процессы торможения в ЦНС, седативный эффект
- **NMDA-модуляция** — ко-агонист NMDA-рецепторов (в паре с глутаматом)
- **Синтез гемоглобина** — входит в состав гема
- **Регуляция сна** — снижение температуры тела, улучшение засыпания
- **Метаболизм оксалатов** — глицин — один из предшественников синтеза оксалатов
- **Коллагенообразование** — каждая 3-я аминокислота коллагена

## Показания (ЖКТ + детокс + нервная система)

| Показание | Эффект | Уровень |
|-----------|--------|---------|
| Застой желчи (улучшение реологии) | 1B | Конъюгация + таурин/глицин |
| Детоксикация (II фаза, бензойная кислота) | 2A | Конъюгация с глицином |
| Стресс / тревожность | 1B | Седативный эффект |
| Бессонница | 1B | Улучшение засыпания |

| Высокий гомоцистеин (как донор) | 2B | Метилирование |

## Особенности приёма

- **Стандартная дозировка:** 1000–3000 мг на ночь
- **Форма:** сублингвальные таблетки (100–300 мг), порошок, капсулы
- **Приём:** сублингвально (лучшая биодоступность, быстрый седативный эффект)
- **Важный нюанс:** у части людей глицин может не успокаивать, а бодрить — это связано с NMDA-рецепторами (глицин + глутамат открывают Na/Ca-каналы). Возможные причины возбуждающей реакции:
  - Нейровоспаление (после COVID)
  - Дефицит магния (Mg — антагонист Ca) → избыток Ca в клетке
  - Избыток натрия / дефицит калия
  - Избыток глутамата в пище (синдром китайского ресторана)
- **При возбуждающей реакции:** перенести на первую половину дня или отменить
- **Противопоказан при склонности к оксалатным камням** (оксалаты синтезируются из глицина, глюкозы и аскорбиновой кислоты)

## Форма приема и порядок

Сублингвальные таблетки (100–300 мг), порошок (1000–3000 мг на ночь). На ночь — сублингвально (седация). При возбуждающей реакции — перенести на утро. Курс: 4–8 недель

## Противопоказания

- Оксалатные камни в почках
- Нейровоспаление с гипервозбудимостью
- Беременность, лактация (безопасность не установлена)


## Топ-5 продуктов, в которых содержится

Костный бульон (желатин — глицин составляет 1/3 аминокислот), кожа курицы, свиная шкурка (холодец), коллагеновые продукты (желе, мармелад — на желатине), бобовые (нут, фасоль, чечевица), тыквенные семечки

## Взаимодействие

### Комплименты (синергисты)

| Ранг | Вещество | Эффект |
|------|----------|--------|
| +++ | Таурин | Синергия конъюгации жёлчных кислот (гликохолевая + таурохолевая) |
| +++ | Магний | Предотвращает возбуждающий эффект глицина (антагонист Ca) |
| ++ | NAC | Синергия синтеза глутатиона (глицин + цистеин) |
| ++ | Лецитин | Синергия работы желчного пузыря |
| ++ | Силимарин | Гепатопротекция + детокс |

### Ингибиторы (антагонисты)

| Ранг | Вещество | Эффект |
|------|----------|--------|
| +++ | Глутамат натрия (пищевой) | Связывание → открытие NMDA-каналов → гипервозбудимость |
| ++ | Алкоголь | Снижает уровень глицина (повышенный расход на детокс) |
| ++ | Бензойная кислота (косметика, Бенфотиамин B1) | Расходует глицин на конъюгацию → снижение пула |
| ++ | Кофеин | Конкуренция за рецепторы сна |
| + | Оксалаты (шпинат, щавель) | Предшественник синтеза оксалатов |

## Научные факты

- РКИ (2018): 3 г глицина перед сном улучшают качество сна и уменьшают дневную сонливость (PSQI −2.5 балла).
- Исследование (2016): глицин улучшает метаболизм глюкозы и чувствительность к инсулину у пациентов с метаболическим синдромом.
- Мета-анализ (2020): глицин снижает нейровоспаление и оксидативный стресс через синтез глутатиона.

## Лайфхак для пользователя (Nutrition Tip)

> 💤 **Глицин — универсальный помощник для желчи и сна.** Работает на двух фронтах: жёлчные кислоты (гликохолевая) + седативный (ГАМК). Стандартная схема — 1000–3000 мг на ночь сублингвально. Если бодрит — проверьте глутамат в пище и магний (дефицит Mg — частая причина возбуждения). Важно: не назначать при оксалатных камнях (глицин — предшественник оксалатов).
