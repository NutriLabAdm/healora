## Ключевые принципы

Первая встреча с клиентом состоит из трёх этапов:
1. **Предварительная (продающая) встреча** — знакомство, выяснение запроса, прошлого опыта, договорённость
2. **Первичная консультация** — сбор данных, постановка цели по SMART, план работы
3. **Инструкция к себе** — правила взаимодействия, которые делают работу комфортной
