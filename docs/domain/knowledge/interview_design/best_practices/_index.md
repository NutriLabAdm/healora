## Сводная таблица по артефактам и взаимодействию

| Параметр | MIIN | NBC-HWC | Clinical RD |
|---|---|---|---|
| **Длительность встречи** | 45–60 мин | 50–60 мин | 30–90 мин |
| **Формат** | Онлайн / офлайн / чат | Онлайн / офлайн | Преимущественно офлайн (клиника) |
| **Анкета предзаписи** | 2–5 вопросов | health history + life story | intake form (full medical) |
| **Основной метод сбора** | Открытые вопросы, дневник | Wheel of Life, шкалы | 24h recall, FFQ, labs |
| **Артефакты** | Визитка, список услуг, инструкция к себе | Goal setting worksheet, коуч-контракт | Nutritional assessment form, план MNT |
| **Постановка цели** | SMART + глубинные вопросы | SMART + self-directed change | PES statement (проблема-этиология-симптом) |
| **Способ взаимодействия** | Чат + онлайн созвоны | Видео-сессии + email | Приёмы в клинике + онлайн |
| **План работы** | Протокол + интервенции | Goal-based action plan | MNT protocol (дозировки, лаборатории) |
