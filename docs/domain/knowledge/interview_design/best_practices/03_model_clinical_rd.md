## Общее описание

Clinical RD-модель — стандарт **медицинской нутрициологии** (США, Европа). Использует Nutrition Care Process (NCP) — 4-шаговый цикл: Nutritional Assessment → Nutrition Diagnosis (PES) → Nutrition Intervention → Nutrition Monitoring & Evaluation.

Основное отличие от MIIN и NBC-HWC: клинический фокус — диагноз, лаборатории, доказательные протоколы. Нутрициолог = медицинский специалист, назначающий терапию.

## Артефакты интервью

| Артефакт | Формат | Когда используется | Назначение |
|---|---|---|---|
| **Intake Form / Nutritional Assessment Form** | Подробная анкета (20–60 вопросов) | До первой встречи | Медицинский анамнез, лекарства, аллергии, ЖКТ |
| **24-hour dietary recall** | Структурированное интервью | На первой встрече | Оценка текущего питания |
| **Food Frequency Questionnaire (FFQ)** | Анкета (частота потребления продуктов за месяц) | На первой / второй встрече | Долгосрочные паттерны питания |
| **Антропометрия** | Замеры (вес, рост, ИМТ, окружность талии, % жира) | На каждой встрече | Объективные метрики |
| **Лабораторная диагностика** | Направление на анализы | На старте | Биомаркеры (глюкоза, HbA1c, липиды, ферритин, витамин D и др.) |
| **PES Statement** | Nutrition Diagnosis (проблема — этиология — симптомы) | После сбора данных | Медицинский «диагноз» нутрициолога |
| **MNT Protocol** | План питания с дозировками (ккал, белки, жиры, углеводы) | На второй встрече | Лечебное питание |

## Способы взаимодействия

| Формат | Применение | Частота | Стоимость (входит в бюджет) |
|---|---|---|---|
| **Офлайн-приём (клиника)** | Первичный приём, замеры, антропометрия | 1 раз в 1–4 недели | Основной формат |
| **Онлайн-консультация** | Разбор лабораторий, коррекция плана | По необходимости | Включено или частично |
| **Чат / телефон** | Только для вопросов по плану | По необходимости (не ежедневно) | Не входит |
| **Email** | Отправка плана питания, отчётов | Раз в неделю / по плану | Включено |

## Список услуг и что входит в бюджет

### Вариант 1. Medical Nutrition Therapy (1 приём)

| Компонент | Входит |
|---|---|
| Nutritional Assessment (анкета + 24h recall + антропометрия) | Да |
| PES-диагноз | Да |
| Индивидуальный план питания (КБЖУ рассчитаны) | Да |
| **Лабораторная диагностика** | Нет (назначается, оплачивается отдельно) |
| **Сопровождение** | Нет |
| **Цена** | 5 000–10 000 руб |

### Вариант 2. NCP-ведение (1 месяц)

| Компонент | Входит |
|---|---|
| Всё из разового приёма | Да |
| Еженедельный мониторинг (замеры, коррекция) | Да |
| **Разбор лабораторий** | Да |
| **Меню на неделю (рассчитанное по КБЖУ)** | Да |
| **Список покупок по отделам** | Да |
| **Цена** | 15 000–30 000 руб |

### Вариант 3. Полный протокол (3–6 месяцев)

| Компонент | Входит |
|---|---|
| Всё из NCP-ведения | Да |
| Лабораторный контроль (входной + промежуточный + итоговый) | Да |
| **Нутрицевтическая коррекция** (дефициты, дозировки) | Да |
| **Daily tracking через приложение** | Да |
| **Заключительный отчёт с динамикой** | Да |
| **Цена** | 45 000–90 000 руб |

## Пример плана-протокола

### Протокол «Метаболический синдром — коррекция» (12 недель)

**PES:** Избыточное потребление энергии (E-01) связанное с нерегулярным питанием и преобладанием простых углеводов, проявляющееся ИМТ 32, окружностью талии 102 см, HbA1c 6.4%.

**Целевые метрики:** ИМТ <28, талия <94 см, HbA1c <5.7%, глюкоза натощак <5.6 ммоль/л.

| Неделя | Интервенция | Дозировка | Артефакт / Инструмент |
|---|---|---|---|
| 0 | Первичный приём: антропометрия + 24h recall + направление на лабораторию | 1 раз | Nutritional Assessment Form |
| 0 | Сбор дневника питания (3 рабочих + 1 выходной день) | 4 дня | Шаблон дневника |
| 1 | Разбор лабораторий, PES-диагноз, план питания | 1 приём | PES Statement |
| 1 | Дефицит 500 ккал от поддерживающей калорийности | Ежедневно | Индивидуальный план КБЖУ |
| 1–12 | Белок 1.2–1.5 г/кг, жиры 25–30%, углеводы 40–45% | Ежедневно | Меню на неделю + список покупок |
| 2–12 | Замена простых углеводов на сложные (крупы, овощи) | Ежедневно | Гайд по углеводам |
| 4 | Контрольные замеры + повтор антропометрии | 1 раз | Таблица динамики |
| 4–12 | Добавление физической активности (10 000 шагов/день) | Ежедневно | Шагомер / приложение |
| 8 | Контрольная лаборатория | 1 раз | Сравнение с входными данными |
| 12 | Итоговый приём: замеры, лаборатория, заключение | 1 раз | Итоговый отчёт |
