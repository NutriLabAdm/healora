## Общее описание

MIIN-модель — практический подход, разработанный для российских нутрициологов. Сочетает **коучинговые техники** (открытые вопросы, активное слушание, постановка цели) с **нутрициологической диагностикой** (анализ дневников питания, анамнез, работа с лабораториями).

Основное отличие: фокус на **выбор «своего» клиента** через анкету предзаписи и предварительную встречу, а также на **инструкцию к себе** как инструмент управления ожиданиями.

## Артефакты интервью

| Артефакт | Формат | Когда используется | Назначение |
|---|---|---|---|
| **Анкета предзаписи** | 2–5 открытых вопросов, форма Google / Telegram | До встречи | Отбор целевых клиентов |
| **Визитка нутрициолога** | Текст / презентация | На предварительной встрече | Кто я, моя ниша, опыт, результаты |
| **Список услуг с описанием** | Документ (PDF / гайд) | После предварительной встречи | Форматы работы, стоимость |
| **Инструкция к себе** | Текст (1 страница) | На старте работы | Время ответа, мессенджер, особенности |
| **Дневник питания** | Шаблон (7 дней, ПН–ВС) | Во время первичной консультации | Сбор данных о питании |
| **SMART-карта цели** | Таблица / устно | На первичной консультации | Формализация результата |

## Способы взаимодействия

| Формат | Применение | Частота | Стоимость (входит в бюджет) |
|---|---|---|---|
| **Чат (Telegram / WhatsApp)** | Ежедневная поддержка, ответы на вопросы, отчёты | Ежедневно в рабочее время | Включено в ведение |
| **Онлайн-созвоны** | Первичная консультация, еженедельная сверка | 1 раз в 1–2 недели | Включено |
| **Офлайн-встречи** | Первичный приём, замеры | 1 раз в месяц / на старте | Дополнительно или включено |
| **Голосовые сообщения** | Разбор дневника, обратная связь | По необходимости | Включено |

## Список услуг и что входит в бюджет

### Вариант 1. Разовая консультация

| Компонент | Входит |
|---|---|
| Анкета предзаписи | Да |
| Предварительная встреча (15–20 мин) | Да |
| Первичная консультация (60 мин) | Да |
| Анализ дневника питания | Да |
| Рекомендации (текстовый гайд) | Да |
| **Чат-поддержка** | Нет |
| **Сопровождение** | Нет |
| **Цена** | 3 000–6 000 руб |

### Вариант 2. Ведение (1 месяц)

| Компонент | Входит |
|---|---|
| Всё из разовой консультации | Да |
| Еженедельные созвоны (4 × 30 мин) | Да |
| **Чат-поддержка ежедневно** | Да |
| Корректировка плана по результатам | Да |
| **Список покупок на неделю** | Да |
| **Инструкция к себе** | Да |
| **Цена** | 12 000–20 000 руб |

### Вариант 3. Полное ведение (3 месяца)

| Компонент | Входит |
|---|---|
| Всё из ведения на 1 месяц | Да |
| Лабораторная диагностика (разбор) | Да |
| **Глубокий протокол** (дефициты, нутрицевтики) | Да |
| **План Б на форс-мажоры** (командировки, праздники) | Да |
| **Контрольные замеры каждые 2 недели** | Да |
| **Заключительный отчёт** | Да |
| **Цена** | 30 000–60 000 руб |

## Пример плана-протокола

### Протокол «Коррекция веса + гидратация» (4 недели)

**Цель по SMART:** снизить вес на 3 кг за 4 недели, увеличив потребление воды с 0.8 л до 2 л/день.

| Неделя | Интервенция | Дозировка / Частота | Артефакт |
|---|---|---|---|
| 1 | Дневник питания с фото | Ежедневно | Шаблон дневника |
| 1 | Водный режим: +1 стакан утром | Каждый день после пробуждения | Трекер воды |
| 2 | Замена ужина на белково-овощной боул | Ежедневно | 3 шаблона боулов |
| 2 | Прогулка 20 мин | 4 раза в неделю | Чек-лист |
| 3 | Добавить второй приём воды (перед обедом) | Ежедневно | Трекер воды |
| 3 | Исключить сладкие напитки | Ежедневно | Дневник контроля |
| 4 | Завтрак с белком (яйца / тофу / рыба) | Ежедневно | 7 рецептов завтраков |
| 4 | Контрольное взвешивание + замеры | 1 раз | Таблица замеров |
