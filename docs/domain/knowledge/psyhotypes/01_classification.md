## Конституциональный подход

В основе лежит **конституция человека** — совокупность устойчивых морфологических и функциональных (в т.ч. психических) свойств, обусловленных наследственностью и влиянием среды.

- **Морфологические типы:** астеник, нормостеник, гиперстеник
- **Психологические типы:** сангвиник, холерик, флегматик, меланхолик
- **Вегетативная регуляция:** баланс симпатической (активация, стресс) и парасимпатической (расслабление, восстановление) нервной системы

> Чистые типы встречаются редко — есть склонность к определённым реакциям.

## 4 психотипа

### 1. Сангвиник

| Характеристика | Описание |
|---|---|
| **Поведение** | Последовательно идёт к цели при минимальной поддержке. Не теряет тонус при неудачах. Легко справляется с искушениями |
| **Адаптация** | Быстро реагирует на изменения, не испытывает значительного стресса от нововведений |
| **Навыки** | Легко отказывается от старых стереотипов, быстро приобретает новые привычки |
| **ВНС** | Уравновешенная симпатика и парасимпатика |
| **Морфотип** | Нормостеник |
| **Телосложение** | Атлетическое, широкий плечевой пояс, узкие бедра, розовая кожа |
| **Движения** | Плавные, чёткие |
| **Обхват запястья** | Ж: 15–17 см / М: 18–20 см |

### 2. Холерик

| Характеристика | Описание |
|---|---|
| **Поведение** | Лидер по натуре. Вынослив, работоспособен. Нацелен на результат |
| **Эмоции** | Плохо контролирует, может быть взрывным, вспыльчивым, критичным, резким |
| **ВНС** | Активная симпатика, парасимпатика недоактивирована |
| **Особенность** | Нетерпелив — нужен результат «здесь и сейчас» |
| **Морфотип** | Нормостеник |
| **Телосложение** | Атлетическое, желтоватая кожа |
| **Движения** | Резкие, порывистые, торопливые |
| **Обхват запястья** | Ж: 15–17 см / М: 18–20 см |

### 3. Флегматик

| Характеристика | Описание |
|---|---|
| **Поведение** | Спокойный, тихий, добрый, медлительный |
| **Изменения** | Не любит перемен, постоянен. Любит взвешивать «за и против» |
| **ВНС** | Низкая активность симпатики, сильная парасимпатика |
| **Риски** | Склонен к бездействию, быстро утомляется, сложно поддерживать мотивацию к новому |
| **Морфотип** | Гиперстеник |
| **Телосложение** | Плотный, склонный к полноте, широкие плечи и бёдра, широкое лицо |
| **Движения** | Плавные, медлительные |
| **Обхват запястья** | Ж: >18 см / М: >20 см |

### 4. Меланхолик

| Характеристика | Описание |
|---|---|
| **Поведение** | Застенчивый, ответственный, мягкий, тактичный |
| **Особенность** | Легко выходит из равновесия. Избыточно ранимый, критичный к себе |
| **Среда** | Хорошо чувствует себя в привычной обстановке; изменения = высокий стресс |
| **ВНС** | Неустойчивый баланс, склонность к дисбалансу |
| **Морфотип** | Астеник |
| **Телосложение** | Высокий рост, низкая мышечная масса, узкие плечи и бёдра, склонность к сухости кожи |
| **Обхват запястья** | Ж: <15 см / М: <18 см |

## Вегетативная регуляция

| Состояние | Симпатика | Парасимпатика |
|---|---|---|
| **Норма** | Мобилизация, активность, энергичность | Концентрация на цели, устойчивость изменений, спокойствие |
| **Отклонение** | Избыточная тревожность, подозрительность, раздражительность | Лень, инертность, отсутствие мотивации |

> В работе важно учитывать как врождённые конституциональные признаки, так и текущие отклонения (дисбаланс).

---

## Обоснование по обхвату запястья

Обхват запястья остаётся стабильным при изменении веса (в отличие от талии/бёдер) и коррелирует с толщиной кости, что позволяет определить морфотип даже при избыточной массе тела.

| Тип | Женщины | Мужчины |
|---|---|---|
| Астеник (меланхолик) | <15 см | <18 см |
| Нормостеник (сангвиник/холерик) | 15–17 см | 18–20 см |
| Гиперстеник (флегматик) | >18 см | >20 см |


## Промпты для презентации

### Для клиента
> **Заголовок:** «Без названия: как внедрить полезные привычки без силы воли»
>
> **Теория:**
> - Основная идея: научный подход к изменению поведения
> - Почему сила воли не работает (и что работает вместо неё)
> - 3 ключевых принципа изменений
> - Как не срываться и возвращаться после срывов
>
> **Практика:**
> - Техника «маленьких шагов»: с чего начать сегодня
> - Дизайн среды: как обустроить пространство
> - Чек-лист на 21 день
>
> **Дисклеймер:** Изменение поведения — процесс. Не корите себя за срывы.

### Для специалиста
> **Заголовок:** «Без названия: модели поведения и техники коучинга»
>
> **Теория:**
> - Теоретические модели: поведенческие модели
> - Нейрофизиологическая основа привычек
> - Роль среды, триггеров и подкрепления
>
> **Практика:**
> - Техники интервью для выявления барьеров
> - Инструменты: CJM, MIIN, Fogg Behavior Model
> - Протокол сопровождения клиента на 30/60/90 дней
>
> **Дисклеймер:** Коучинговые инструменты. Не заменяют психотерапию.
