## 1. Дофаминовая основа мотивации

### Короткий vs длинный дофамин

| | Короткий дофамин | Длинный дофамин |
|---|---|---|
| **Путь** | Лимбическая система («хочу») | Лобная кора («надо») |
| **Источник** | Вкусная еда, соцсети, быстрые удовольствия | Планирование, достижение этапов, отсроченная награда |
| **Цель** | «Хочу» — избежать опасности, получить сейчас | «Надо» — долгосрочная цель по здоровью |

### 4 квадранта мотивации (по скорости и типу стимула)

| | Угроза («от») | Награда («к») |
|---|---|---|
| **Быстрый** | Опасность здесь и сейчас (собака) | Съел вкусное — получил удовольствие |
| **Медленный** | Отчёт, чтобы не уволили | Здоровье через год (самый сложный) |

> Цели по здоровью почти всегда в квадранте «медленная награда» — поэтому их так трудно достигать.

## 2. Инструменты мотивации по психотипам

### Сангвиник

| Аспект | Рекомендации |
|---|---|
| **Тип дофамина** | Длинный + немного быстрого. Получает удовольствие от процесса |
| **План** | Чёткий пошаговый план. Хорошо работает на долгосрок |
| **Поддержка** | Минимальная. Регулярные созвоны для сверки, постановка еженедельных/ежемесячных целей |
| **Инструменты** | Трекер привычек, графики, схемы на 3 месяца |
| **Мотивация** | Наградой (а не угрозой). Идёт за энергией, активным долголетием |
| **Роль нутрициолога** | Партнёр с письменными пошаговыми рекомендациями |

### Холерик

| Аспект | Рекомендации |
|---|---|
| **Тип дофамина** | Яркие быстрые результаты. Нужно весомое вознаграждение |
| **План** | Чёткий план + скорость и эффективность. Здоровье = бизнес-проект |
| **Поддержка** | Без критики и осуждения. В балансе — показывать альтернативы и давать выбор |
| **Инструменты** | Карта желаний, партнёрский договор, полезные бонусы (гайды, лайфхаки), график эффективности |
| **Мотивация** | Наградой (а не угрозой). Важно завоевать авторитет — убедить на логике и фактах |
| **Роль нутрициолога** | Бизнес-партнёр: «Давайте добавим в программу это — какой результат хотите?» |

### Флегматик

| Аспект | Рекомендации |
|---|---|
| **Тип дофамина** | Частые быстрые награды. Нужен запасной план на случай потери мотивации |
| **План** | Пошаговое движение, дробление целей. Недельные задачи. Схема на 1–2 месяца |
| **Поддержка** | Размеренная постоянная работа. Объяснять логику, смысл действий |
| **Инструменты** | Трекер привычек, готовые списки покупок, сопровождение (телефон), читмил как награда |
| **Мотивация** | Через боль и угрозу здоровью (часто приходят из боли). Важна научность, доказательность |
| **Роль нутрициолога** | Помощник: «Я помогу разобраться — будет несложно». Сервисность и комфорт |

### Меланхолик

| Аспект | Рекомендации |
|---|---|
| **Тип дофамина** | Частая и быстрая награда (раз в неделю или чаще). Награда с яркой положительной окраской |
| **План** | Краткосрочное планирование (максимум месяц). Подведение итогов |
| **Поддержка** | Эмоциональная поддержка, похвала, личное общение. Не превращаться в «спасателя» |
| **Инструменты** | Трекер привычек, дневник питания в фото (визуальная оценка «до/после»), мудборд/карта желаний |
| **Мотивация** | Через вдохновение и увлекательную идею. Важно обоснование необходимости перемен |
| **Роль нутрициолога** | Заботливый старший товарищ / тренер: «Давайте вернёмся к целям — представьте результат» |

## 3. Роли нутрициолога в зависимости от состояния клиента

### В балансе

| Тип | Роль | Стратегия |
|---|---|---|
| Сангвиник | **Партнёр** | Последовательная работа, письменные рекомендации, самостоятельная регуляция |
| Холерик | **Предлагающий** | Давать выбор, показывать альтернативы, выгоды, инструменты эффективности |
| Флегматик | **Помощник** | Научность, доказательность, сервисность, комфорт, готовые решения |
| Меланхолик | **Тренер** | Эмоциональное вдохновение, заряжающие цели, визуализация результата |

### В дисбалансе

| Тип | Роль | Что делать / не делать |
|---|---|---|
| Сангвиник | *(редко выходит из баланса)* | Обоснование рекомендаций, спокойная работа |
| Холерик | **Психолог** | Не критиковать, не осуждать. «Отлично, что попробовали — давайте альтернативу» |
| Флегматик | **Эксперт** | Научность, авторитетная позиция, чёткое объяснение «зачем» |
| Меланхолик | **Старший товарищ** | Хвалить, указывать на положительное. Не подчёркивать неудачи. Не быть «жилеткой» |

## 4. Цикл развития осознанности (компетентности)

### 4 уровня

```
Неосознанная некомпетентность
  ↓ (информирование, дневники, фиксация)
Осознанная некомпетентность
  ↓ (знания, коучинг, гайды, связь с практикой)
Осознанная компетентность
  ↓ (умения, трекеры привычек, дизайн поведения)
Неосознанная компетентность (навык)
  ↓ (новая цель → новый цикл)
```

### Этапы и инструменты

| Этап | Состояние клиента | Инструменты |
|---|---|---|
| **1. Неосознанная некомпетентность** | Не видит связи проблемы и образа жизни. «Меня привели». Ждёт таблетку | Дневник питания, дневник образа жизни, запись дня по 15 мин |
| **2. Осознанная некомпетентность** | Понял, что проблема есть. Нужны знания | Коучинг-сессии, гайды, дневник изменений, связь знаний с самочувствием |
| **3. Осознанная компетентность** | Внедряет привычки, тренируется | Трекеры привычек, дизайн поведения (конструкторы, органайзер кухни), упрощение |
| **4. Неосознанная компетентность** | Навык автоматизирован (как чистить зубы) | Постановка новой, более высокой цели → цикл повторяется |

## 5. Общая схема работы с клиентом

```
Шаг 1. Определить психотип (сангвиник / холерик / флегматик / меланхолик)
        + определить уровень дофамина (быстрый / долгий)
        + подобрать инструменты мотивации

Шаг 2. Определить уровень осознанности:
        - Неосознанный → информировать, вести по циклу
        - Осознанный → поддерживать, закреплять, выводить на новый уровень
        - Гиперконтроль → снижать тревожность, подключать психолога

Шаг 3. Развивать компетентность на трёх уровнях:
        - Знания → информирование, обучение
        - Умения → тренировка, коучинг
        - Навыки → автоматизация, новая цель
```
