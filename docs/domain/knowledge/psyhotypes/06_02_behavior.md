## 1. Что такое дизайн поведения

**Дизайн поведения** — изучение систематических ошибок, которые люди совершают при выборе, и проектирование среды, которая помогает принять полезное решение или совершить правильное действие.

**Ключевые принципы:**
- Наши **намерения ≠ наши действия** (будущее «Я» всегда лучше «Я» настоящего)
- **Восприятие ≠ реальности** (карта не равна территории)
- **Поведение ≠ убеждениям** (человек может знать о пользе, но действовать иначе)
- Хотите влиять на человека — меняйте **факторы**, определяющие его поведение

### Эксперимент Либета (1950-е)

Нейрохирург Бенджамин Либет доказал: мозг принимает решение **за доли секунды до того**, как человек осознаёт это решение. Современные МРТ-исследования показали: решение можно предсказать **за 7 секунд** до его осознания.

> Свободы воли в классическом понимании не существует. Решение принимает мозг как биохимическая машина. Дизайн поведения не оставляет мозгу шанса на неправильное решение.

## 2. Что определяет поведение человека

```
ФАКТОРЫ ПОВЕДЕНИЯ (схема):
  ┌──────────────────────────────────┐
  │  5%  Рациональное мышление       │
  │      (потребности, ценности,     │
  │       убеждения)                 │
  ├──────────────────────────────────┤
  │ 30%  Среда                       │
  │      (доступность продуктов,     │
  │       организация пространства)  │
  ├──────────────────────────────────┤
  │ 30%  Эмоции                      │
  │      (окраска поступка: приятно  │
  │       / неприятно)              │
  ├──────────────────────────────────┤
  │ 25%  Триггеры                    │
  │      (предметы, контекст,        │
  │       напоминания)              │
  ├──────────────────────────────────┤
  │ 10%  Опыт                        │
  │      (повторение, привычка)      │
  └──────────────────────────────────┘
```

> Только **5%** поведения определяется сознательными ценностями и убеждениями. Остальное — среда, эмоции, триггеры и опыт.

### Как мозг принимает решения

В мозге одновременно конкурируют разные нервные сигналы (системы возбуждения и торможения). Где сигнал интенсивнее — туда и направляется действие.

**Пример:** Вы голодны (зона голода активна), но слушаете интересную лекцию (зона познавательного интереса активна). Пока интерес перевешивает — вы остаётесь. Когда голод достигает критической точки — вы идёте есть.

**Задача дизайна поведения:** сделать сигнал к полезному действию более интенсивным, чем сигнал к вредному.

|---|---|---|
| **Среда (30%)**<br>Доступность продуктов, организация пространства | Правило вытянутой руки | Фрукты на столе, печенье — в верхний ящик |
| | Зонирование кухни (зелёная / красная зона) | Тарелка-боул на видном месте, сладости не покупать |

| | Социальное удовольствие (полезное + общение) | Групповая прогулка, спорт с другом |
| | Музыка / аудиоконтент как фон | Подкаст во время заготовки, музыка — на зарядке |
| | **Техника для коуча:** «Вспомните, когда полезное действие было радостью. Что сделало его приятным?» | |
| **Триггеры (25%)**<br>Предметы, контекст, напоминания | Визуальный триггер (предмет на видном месте) | Бутылка воды на столе → напоминание пить |
| | Контекстный триггер / habit stacking | «После чистки зубов — стакан воды» |
| | Цифровой триггер (пуш, напоминание) | Напоминание в телефоне каждый час |
| | Социальный триггер (другой человек) | Чат с коучем «Сделал зарядку ✅» |
| | Эмоциональный триггер (состояние → действие) | «Стресс → дыхательная практика 4-7-8» |
| | Магнитный триггер (на пути, нельзя не заметить) | Коврик для зарядки на полу |
| | Триггер-напоминание (стикер, магнит) | Магнит «Вода!» на холодильник |
| | **Техника «Триггер-аудит»:** запись всех триггеров за день → перепроектировать | |
| **Опыт (10%)**<br>Повторение, привычка | Правило 21 дня (минимальный срок закрепления) | 21 день подряд пить стакан воды утром |
| | Микро-привычки (< 2 мин на старт) | 1 приседание, 1 глоток воды |
| | Серийный эффект (пропуск не обнуляет) | Пропустил день — продолжаю на следующий |
| | Трекер галочек (визуализация streak) | Календарь с крестиками |
| | Наращивание сложности (+10% в неделю) | 1 ст → 2 ст → 3 ст воды |
| | Награда за повторение (дофамин) | После 7 галочек — маленькая награда (не еда) |
| | План Б на срыв (заранее) | «Пропустил → делаю на следующий день, без вины» |
| | **Техника «21/7»:** 21 день трекинг → 7 дней без трекера (проверка автоматизма) | |
| **Приверженность и убеждения (10%)**<br>Внутреннее согласие | Публичное обещание (объявить цель) | «Я похудею на 5 кг за 3 мес, спросите меня» |
| | Контракт с собой (письменно, с подписью) | «Обязуюсь пить 2 л воды 30 дней» |
| | Штраф за срыв (финансовый / социальный) | 500 руб в общий котёл группы |
| | Принцип последовательности (напомнить успехи) | «Вы бросили курить — вы умеете достигать целей» |
| | Научное обоснование (исследования, механизмы) | «2 л воды снижают головные боли на 40%» |
| | Авторитетный источник (ВОЗ, МИИН) | «ВОЗ рекомендует...» |
| | Противодействие когнитивному диссонансу | «Вы говорите здоровье — приоритет, но спите 5 ч. Как сочетается?» |
| | **Техника «Почему?»:** 5 вопросов «почему» до глубинной ценности | Семья → долголетие → здоровье |
| **Потребности (5%)**<br>Базовые драйверы | Иерархия (сон → вода → белок → питание → активность → БАДы) | Не начинать с диеты, если клиент спит 4 ч |
| | Карта потребностей (что стоит за вредным поведением) | Сладкое → потребность в энергии |
| | Замена способа удовлетворения | Энергия: шоколад → белок + сложные углеводы |
| | Автономия (выбор из 2–3 вариантов) | «Боул с курицей или средиземноморская тарелка?» |
| | Компетентность (показать прогресс) | «7 дней водного режима — вы профессионал гидратации» |
| | Связанность (группа, поддержка) | Чат клиентов, парное выполнение |
| | **Техника «Иерархия»:** чек-лист базовых потребностей перед планом | |
| **Ценности (5%)**<br>Глубинные ориентиры | Ценностный якорь (привязать действие к ценности) | «Зарядка — вклад в долголетие, чтобы быть с детьми» |
| | Дневник ценностей (еженедельно) | «На этой неделе пила воду ради ценности — здоровье» |
| | Визуализация ценностей (коллаж, доска) | Фото семьи + «Ради них я забочусь о себе» |
| | Ценностный конфликт (осознать разрыв) | «Говорите, здоровье — ценность, но едите фастфуд. Почему?» |
| | Ритуал ценностей (ежедневное действие) | Каждое утро вслух: «Я выбираю здоровье, потому что люблю себя» |
| | Ценностное интервью (5 «почему») | От «хочу похудеть» до «хочу быть с семьёй долго» |
| | **Техника «7 ценностей»:** выбрать, приоритизировать, записать действие для каждой сегодня | |

## 3. Инструменты дизайна поведения

### 3.1. Трудность — лёгкость

**Принцип:** желаемое поведение сделать **лёгким**, нежелательное — **максимально трудным**.

| Что делаем | Пример |
|---|---|
| **Списки покупок** | Клиент идёт в магазин с готовым списком, а не придумывает на ходу |
| **В магазин — сытым** | Голодный клиент с вероятностью 90% купит вредное |
| **Простые заготовки** | Рецепты на 10–15 мин, а не на 1.5 часа |
| **Организация кухни** | Полезное — на видном месте, вредное — в дальний ящик |
| **Органайзеры нутрицевтиков** | На видном месте (на столе), а не в шкафу |
| **Подготовка к утру с вечера** | Спортформа рядом с кроватью, замоченные бобовые, коврик на полу |

**Пример эксперимента:** В корпоративной столовой полезную еду поставили в первый ряд, сладости убрали под крышку на дальний ряд. Потребление полезных продуктов выросло на 30% без единой лекции.

### 3.2. Изменение самоидентичности

**Принцип:** клиент начинает думать о себе через желаемое качество, а не через действие.

| Вместо | Говорим |
|---|---|
| «Я хожу заниматься спортом» | **«Я — спортивный человек»** |
| «Я пытаюсь правильно питаться» | **«Я — человек, который заботится о себе»** |
| «Я должен пить воду» | **«Я — человек, который внимателен к своему телу»** |

**Цикл самоидентичности:**
```
Утверждение («я — спортивный»)
  ↓
Действие (иду на пробежку, потому что «я спортивный»)
  ↓
Убеждение (это действие укрепляет мою идентичность)
  ↓
Новое утверждение (ещё более сильное)
```

### 3.3. Растим доминанту

#### 3.3.1. Привычки группами

Формирование одной цепочки из нескольких привычек требует **столько же энергии**, сколько одна привычка по отдельности.

**Пример утренней группы привычек:**
```
Встаю → коврик (аппликатор Кузнецова 2 мин) → умывание прохладной водой
  → стакан воды → зарядка 5 мин → завтрак (боул)
```

#### 3.3.2. Критическая масса информации

Клиент должен понимать, **зачем** он делает то или иное действие. Объясняем механизмы:
- «Почему утром стоять на колючках?» — активация точек, микро-стресс → иммунитет
- «Почему пить тёплую воду?» — парасимпатика, снижение стресса
- «Почему белок в каждый приём?» — сытость, гликемический контроль

**Инструменты:** книги, брошюры, гайды, видео, лекции.

#### 3.3.3. Фокус внимания

| Инструмент | Назначение |
|---|---|
| Магнитики на холодильник | Напоминание о целях |
| Трекер привычек (галочки) | Быстрый дофамин |
| Доска желаний / мудборд | Визуализация результата |
| Напоминалки в телефоне | Контроль воды, приёма БАД |
| Браслеты / трекеры (часы) | Шаги, пульс, сон |

#### 3.3.4. Поддерживающее окружение

- **Вдвоём** → риск схода одного > **90%**
- **Группа 5+ человек** → шанс, что все сойдутся одновременно, минимален. Остальные участники поддерживают временно выпавшего
- **Коучинговые программы** (не разовые консультации)
- **Клубы по интересам** (чат, марафоны, челленджи)

### 3.4. Меняем динамические стереотипы

**Динамический стереотип** — привычное действие, которое запускается другим действием на фоне постоянного подкрепления.

**Примеры:**
```
Работа → усталость → коллеги → чай с печенькой
Стресс → вечер перед телевизором → поход к холодильнику → торт на ночь
```

**Принцип замены:** нельзя убрать всю цепочку целиком. Можно **подменить одно звено** другим.

| Исходная цепочка | Новая цепочка (замена 1 звена) |
|---|---|
| Работа → усталость → коллеги → **чай с печенькой** | Работа → усталость → коллеги → **травяной чай** |
| Стресс → вечер → телевизор → **поход к холодильнику** | Стресс → вечер → телевизор → **дыхательная практика** |
| Утро → **кофе с сахаром** → скачок глюкозы → спад | Утро → **кофе без сахара + вода** → ровный тонус |

> Заменяем пошагово, по одному звену. Через 3–4 шага стереотип меняется полностью.

## 4. Работа с тягой

**Тяга** — сильное пищевое пристрастие к определённому продукту.

### 4.1. Механизм тяги

Тяга накатывает **волнами**. Если не поддаться — волна спадает. При следующем накате интенсивность ниже.

```
Интенсивность тяги:
  ██████████  волна 1 (пик — 8 сек)
  ██████      волна 2 (слабее)
  ███         волна 3 (ещё слабее)
  █           угасание
```

### 4.2. 5 шагов работы с тягой

| Шаг | Действие |
|---|---|
| **1. Переждать 8 секунд** | Мозг принимает решение за 7 сек до осознания. Выдержите 8 сек — включится сознание |
| **2. Заменить действие** | Позвонить коллеге, выйти пройтись, выпить воды, 5 приседаний, дыхательная практика |
| **3. Отличить эмоциональный голод от физического** | Эмоциональная тяга уходит от **маленького** количества продукта. Физический голод — нет |
| **4. Список полноценных радостей** | Иметь под рукой перечень того, что доставляет удовольствие (не еда) |
| **5. Искать альтернативы эмоциональному голоду** | Чтение, общение, рукоделие, прогулка, спорт |

### 4.3. Примеры замены тяги

| Тяга (чего хочется) | Потребность (чего на самом деле) | Альтернатива |
|---|---|---|
| Сладкое после обеда | Взбодриться, дофамин | Прогулка 5 мин, зелёный чай |
| Печенье под стресс | Расслабиться, снизить тревогу | Дыхательная практика 4-7-8 |
| Чипсы вечером | Снять усталость, переключиться | Тёплая ванна, тёплый травяной чай |
| Торт на празднике | Социальное одобрение, «быть как все» | Маленький кусочек осознанно, без вины |

## 5. Планируем форс-мажоры

### 5.1. В гостях

| Что делать | Почему |
|---|---|
| Принести своё блюдо | Контролируете состав |
| Оценить меню заранее | Знаете, что будет на столе |
| Поесть перед выходом | Не идти голодным |
| Выбирать овощи, фрукты, орехи, сыр | Минимальный риск |

**Главное:** если нарушили правила — это не провал. Воспринимайте как читмил и возвращайтесь к плану.

### 5.2. Праздники и свадьбы

- Поешьте заранее
- Выбирайте ресторан с подходящим меню
- На мероприятиях — овощные и фруктовые закуски
- Избегайте салатов с майонезом
- Берите понятные блюда (цельное мясо > котлета, нарезанные овощи > сложный салат)

### 5.3. Дорога (поезд, самолёт)

**Топ-6 поездных снеков:**

| Продукт | Как приготовить |
|---|---|
| **Льняная каша** | 1 ст.л. перемолотого льна + горячая вода, 10 мин. Добавить ягоды/фрукты |
| **Протеин** | Конопляный или гороховый — не портится, безопасен в дороге |
| **Фруктовые чипсы / флаксы** | Хлебцы из пророщенного зерна |
| **Орехово-ягодные батончики** | Домашние или магазинные без сахара |
| **Активированные и высушенные орехи** | Замочить, высушить в духовке |
| **Овощи и фрукты** | Мытые, нарезанные, в контейнере |

### 5.4. All Inclusive (памятка в путешествие)

| Правило | Пояснение |
|---|---|
| **Маленькая тарелка** | На полной тарелке мозгу кажется, что еды много |
| **Группы продуктов** | ¼ белок + ¼ сложные углеводы + ½ овощи |
| **Обойти все блюда** | Выбрать то, что хочется больше всего, начать с этого |
| **Начать с овощей** | Клетчатка наполняет желудок, снижает риск переедания |
| **Понятные блюда** | Цельное мясо, а не котлета. Овощи нарезанные, а не сложный салат |
| **Заправка — масло** | Взять с собой пакетики оливкового масла |
| **Дегустировать аккуратно** | Небольшие порции новых блюд (правило прикорма) |

## 6. Коучинговый подход в дизайне поведения

### Главные принципы

1. **Дизайн поведения снимает трудности именно вашего клиента**
2. **Никто не знает образ жизни клиента лучше него самого**
3. **Не предлагать готовые решения сразу** — исследовать обстоятельства, подталкивать к поиску своего решения

### Структура коучинговой сессии (применительно к дизайну поведения)

```
1. Выявить сложности клиента (наводящие вопросы)
2. Понять причины этих сложностей
3. Определить препятствия к цели
4. Вместе найти решение, которое подходит именно этому клиенту
5. Закрепить через маленькое действие
```

> Лучшее решение — то, которое клиент придумал сам, исходя из своего образа жизни. Это повышает самоценность, эмоциональный фон и дофамин.

## 7. Сводная таблица инструментов

| Инструмент | Суть | Пример |
|---|---|---|
| **Трудность — лёгкость** | Полезное сделать доступным, вредное — труднодоступным | Список покупок, заготовки, органайзер |
| **Самоидентичность** | Сменить «я делаю» на «я — это» | «Я — спортивный», «Я забочусь о себе» |
| **Привычки группами** | Объединить в цепочку | Утро: коврик → вода → зарядка → завтрак |
| **Критическая масса информации** | Объяснить «зачем» | Механизмы, гайды, книги |
| **Фокус внимания** | Постоянные напоминания | Трекер, доска желаний, пуши |
| **Поддерживающее окружение** | Группа, а не одиночка | Чат, клуб, коучинг-программа |
| **Динамические стереотипы** | Заменить 1 звено цепочки | Вместо печенья — травяной чай |
| **8 секунд до сознания** | Переждать волну тяги | Дыхание, вода, звонок |
| **Список радостей** | Альтернативы еде | Прогулка, хобби, общение |
| **Форс-мажоры заранее** | Сценарии для гостей, дороги, отпуска | Снеки, маленькая тарелка, заготовки |
