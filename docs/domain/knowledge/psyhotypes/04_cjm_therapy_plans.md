## 1. Общий подход к планированию терапии

Независимо от психотипа, план терапии проходит 4 этапа:

```
Диагностика психотипа и осознанности
  ↓
Выбор формата и периодичности плана
  ↓
Сопровождение и сбор обратной связи
  ↓
Корректировка на основе результатов
```

### Ключевые параметры плана, зависящие от психотипа:

| Параметр | Варианты |
|---|---|
| **Горизонт планирования** | 1 неделя / 1 месяц / 3 месяца / 6 месяцев |
| **Глубина рекомендаций** | Чёткий план / Альтернативы / Свободный выбор |
| **Частота контакта** | Ежедневно / Раз в неделю / Раз в месяц |
| **Формат обратной связи** | Чат / Звонок / Видео-сессия / Письменный отчёт |
| **Тип подкрепления** | Награда (бонус) / Избегание угрозы / Социальное одобрение |
| **Уровень контроля** | Самоконтроль / Сверка с коучем / Автоматический трекинг |

## 2. Сравнение планов терапии по психотипам

### Сангвиник — «Самодостаточный»

| Параметр | Значение |
|---|---|
| **Горизонт** | 3–6 месяцев. Хорошо удерживает долгосрочную цель |
| **Структура** | Чёткий пошаговый план с этапами и критериями |
| **Частота контакта** | Раз в 1–2 недели (сверка часов) |
| **Формат** | Письменные рекомендации + трекер привычек |
| **Контроль** | Самоконтроль. Самостоятельно регулирует темп |
| **Мотивация** | Процесс + результат. Внутренняя мотивация высокая |
| **Реакция на срыв** | Не теряет тонус, корректирует сам |
| **План Б** | Не обязателен |

**Типичный план:**
- Долгосрочная цель (3–6 мес) → декомпозиция по месяцам
- Еженедельные задачи с трекером
- Созвон раз в 2 недели для сверки
- Дополнительные гайды и материалы для углубления

---

### Холерик — «Целеустремлённый»

| Параметр | Значение |
|---|---|
| **Горизонт** | 1–3 месяца. Важна скорость, результат «здесь и сейчас» |
| **Структура** | Чёткий план с быстрыми победами. Здоровье = проект |
| **Частота контакта** | Раз в неделю + результаты измерений |
| **Формат** | Партнёрский договор. Эффективность, графики, метрики |
| **Контроль** | Внешний (авторитет коуча). Без давления |
| **Мотивация** | Награда, соревнование, статус. Карта желаний |
| **Реакция на срыв** | Раздражение, поиск виноватого. Нужна альтернатива |
| **План Б** | Обязателен. Несколько опций действий |

**Типичный план:**
- Цель, разбитая на короткие спринты (2–4 недели)
- Чёткие KPI: вес, объёмы, биомаркеры
- Еженедельная сверка с графиком эффективности
- Конкурентные элементы (рейтинг, уровни, челленджи)
- Бонусы за достижение этапов (гайды, лайфхаки)

---

### Флегматик — «Основательный»

| Параметр | Значение |
|---|---|
| **Горизонт** | 1–2 месяца. Дробные цели. Долгосрок — слишком абстрактно |
| **Структура** | Пошаговые микро-задачи. Максимум детализации |
| **Частота контакта** | Ежедневно в начале, затем 2–3 раза в неделю |
| **Формат** | Готовые решения: списки покупок, меню, чек-листы |
| **Контроль** | Внешний + сервисный. Сопровождение |
| **Мотивация** | Научность, доказательность. «Зачем я это делаю» |
| **Реакция на срыв** | Обесценивание, может бросить. Нужен план Б |
| **План Б** | Критически важен. «Что делать, если сорвался» |

**Типичный план:**
- Недельные блоки с готовыми материалами
- Чек-лист на каждый день (трекер привычек)
- Доступ к коучу в чате для оперативных вопросов
- Чёткое обоснование каждой рекомендации
- Лёгкие задания — минимум усилий для старта

---

### Меланхолик — «Чувствительный»

| Параметр | Значение |
|---|---|
| **Горизонт** | 2–4 недели. Максимум — 1 месяц |
| **Структура** | Мягкий план с акцентом на самочувствие, а не метрики |
| **Частота контакта** | Ежедневно (чат, поддержка). Регулярная похвала |
| **Формат** | Тёплое общение, дневник в фото, мудборд |
| **Контроль** | Мягкий. Коуч = старший товарищ |
| **Мотивация** | Вдохновение, эмоциональная поддержка, визуализация |
| **Реакция на срыв** | Сильное самобичевание. Нужна поддержка без осуждения |
| **План Б** | Обязателен. Запасной путь без чувства вины |

**Типичный план:**
- Месячный цикл с еженедельными итогами
- Дневник питания в фото («до/после»)
- Ежедневная поддержка в чате с акцентом на успехи
- Мудборд / карта желаний для визуализации
- Награда раз в неделю (не еда)

## 3. Матрица выбора плана по осознанности

Психотип определяет **форму**, уровень осознанности — **готовность** к плану:

| Уровень осознанности | Стратегия | Формат плана |
|---|---|---|
| **Неосознанная некомпетентность** | Вывести на осознание проблемы через дневники | Диагностический период (1–2 недели дневника) без назначений |
| **Осознанная некомпетентность** | Обучить, дать знания, связать с практикой | Образовательный план + простые первые шаги |
| **Осознанная компетентность** | Тренировать навыки, закрепить привычки | Структурированный план с трекером |
| **Неосознанная компетентность** | Повысить сложность, поставить новую цель | План следующего уровня (анти-эйдж, спорт, профилактика) |
| **Гиперконтроль** | Снизить тревожность, убрать избыточную фиксацию | План с послаблениями, без строгих запретов |

## 4. Форматы планов по каналам коммуникации

| Психотип | Основной канал | Дополнительно | НЕ использовать |
|---|---|---|---|
| **Сангвиник** | Письменные рекомендации, дашборд | Созвон раз в 2 недели | Постоянные пуши |
| **Холерик** | Голосовые / видео, графики | Партнёрский договор, рейтинг | Длинные письменные инструкции |
| **Флегматик** | Чат, готовые списки, чек-листы | Телефонное сопровождение | Много текста без структуры |
| **Меланхолик** | Личное общение, чат с поддержкой | Мудборд, визуальные дневники | Сухие отчёты, критика |

---

## 5. Адаптация плана при дисбалансе

Когда клиент в дисбалансе (стресс, тревога, апатия), план временно меняется:

| Психотип | Дисбаланс | Адаптация плана |
|---|---|---|
| **Сангвиник** | *(редко)* — временное снижение энергии | Упростить до 1–2 задач, остальное — поддержка |
| **Холерик** | Раздражительность, резкость | Убрать критику. Предложить альтернативы без давления |
| **Флегматик** | Полная апатия, бездействие | Микро-задачи (1 действие в день). Усилить контроль |
| **Меланхолик** | Тревога, самобичевание | Только положительное подкрепление. Убрать метрики. Подключить психолога при необходимости |

## 6. Способы выявления РПП и гиперконтрольных подходов

### 6.1. Скрининг расстройств пищевого поведения (РПП)

Красные флаги для направления к психологу / психотерапевту:

| РПП | Признаки | Скрининговые вопросы |
|---|---|---|
| **Орторексия** | Избыточная фиксация на «полезности» продуктов. Жёсткое деление еды на «чистую/грязную». Страх «неполезной» еды. Социальная изоляция из-за питания | «Есть ли продукты, которые вы категорически исключаете? Что вы чувствуете, если приходится съесть «запрещёнку»?» |
| **Анорексия** | Ограничение калорий, страх набора веса, искажённое восприятие тела. Низкий ИМТ, аменорея | «Как часто вы считаете калории? Ваш вес влияет на самооценку?» |
| **Булимия** | Циклы ограничения → переедание → очищение (рвота, слабительные). Чувство потери контроля | «Бывает ли, что вы съедаете большое количество еды и не можете остановиться? Что вы делаете после?» |
| **Компульсивное переедание** | Эпизоды неконтролируемого переедания без очищения. Еда как реакция на стресс. Стыд после эпизодов | «Случается ли, что вы едите, когда не голодны, до дискомфорта?» |
| **Ортосомния** | Избыточная фиксация на сне. Тревога из-за данных гаджетов (глубокие фазы, время засыпания). Жёсткие ритуалы отхода ко сну | «Вы отслеживаете сон гаджетами? Как реагируете на отклонения от нормы?» |

### 6.2. Гиперконтроль: признаки и уровни

**Характерные проявления гиперконтроля:**

- Постоянное сидение на ограничительных диетах / строгих протоколах
- Множество пройденных обучений по нутрициологии без системного образования
- Попытка внедрить все рекомендации одновременно
- Приём всех БАД сразу («чтобы наверняка»)
- Вся жизнь подчинена идее питания / спорта / сна
- Тревога при отклонении от режима

**Уровни гиперконтроля:**

| Уровень | Поведение | Тактика нутрициолога |
|---|---|---|
| **Лёгкий** | Клиент интересуется составами, выбирает «чистые» продукты, но гибок | Десенсибилизация: постепенно вводить «гибкие» дни, читмилы |
| **Умеренный** | Жёсткие правила, тревога при нарушениях, социальные ограничения | Психообразование + работа с тревогой. Подключать психолога |
| **Тяжёлый** | Орторексия, анорексия, булимия. Жизнь подчинена контролю | Не брать в работу без сопровождения психиатра / психотерапевта. Направить к специалисту |

### 6.3. Дифференциальная диагностика: гиперконтроль vs осознанность

| Критерий | Осознанный клиент | Гиперконтроль |
|---|---|---|
| **Цель** | Улучшение здоровья, энергии | Страх «нездоровья», избегание |
| **Гибкость** | Есть читмилы, послабления | Жёсткие запреты |
| **Реакция на срыв** | Анализ, корректировка | Тревога, самобичевание, ужесточение |
| **Знания** | Системные, проверенные | Поверхностные, из разных источников |
| **БАДы** | Точечно, по анализам | Всё подряд «для профилактики» |

## 7. Матрица осознанности и компетентности

### 7.1. Цикл развития компетентности по психотипам

```
                         ОСОЗНАННОСТЬ →
                         низкая              высокая
            ┌─────────────────────────────────────┐
     низкая │  I. Неосознанная       II. Осознанная │
            │     некомпетентность      некомпетентность │
            │                                         │
КОМПЕТЕНТ-  │  «Я не знаю, что я      «Я знаю, что    │
НОСТЬ       │   не знаю»              я не знаю»      │
            │                                         │
            ├─────────────────────────────────────────┤
     высокая│ III. Неосознанная      IV. Осознанная   │
            │      компетентность       компетентность  │
            │                                         │
            │  «Я делаю, не думая»   «Я знаю, что     │
            │                         я делаю»        │
            └─────────────────────────────────────────┘
```

### 7.2. Характеристика уровней

| Уровень | Состояние клиента | Речь клиента | Стратегия нутрициолога |
|---|---|---|---|
| **I. Неосознанная некомпетентность** | Не знает о проблеме. Не видит связи образа жизни и здоровья. Пришёл по настоянию других | «Я ничего не ем, а толстею». «Посоветуйте таблетку». «Гены такие» | Вывести на осознание через дневники питания / образа жизни. Запись дня по 15 мин. Фиксация реальных объёмов |
| **II. Осознанная некомпетентность** | Понял проблему, но не знает, как её решить. Мотивирован учиться | «Оказывается, я перекусываю каждые 30 минут». «Что мне теперь делать?» | Обучение: гайды, коучинг-сессии, объяснение причин. Связь знаний с самочувствием |
| **III. Неосознанная компетентность** | Навык автоматизирован. Делает, не задумываясь (как чистить зубы) | «Я уже не думаю о воде — просто пью». «Привычка вошла в жизнь» | Поставить новую, более высокую цель. Повысить сложность. Перевести на следующий уровень |
| **IV. Осознанная компетентность** | Сознательно применяет знания. Тренирует новые навыки | «Я отслеживаю белок, планирую меню, вижу прогресс» | Трекеры привычек, дизайн поведения, закрепление. Контроль и поддержка |

### 7.3. Матрица «Психотип × Осознанность» — подбор стратегии

| Психотип | I. Неосознанная некомпетентность | II. Осознанная некомпетентность | III. Осознанная компетентность | IV. Неосознанная компетентность |
|---|---|---|---|---|
| **Сангвиник** | Дать дневник + факты. Быстро переходит на след. уровень | Образовательные материалы. Минимум поддержки | Трекер привычек. 1 созвон в 2 недели | Новая сложная цель (анти-эйдж, спортпит) |
| **Холерик** | Показать быстрые метрики, графики. Вызвать азарт | Чёткий план + KPI. Без давления, но с авторитетом | Партнёрский договор. Соревновательные элементы | Ультра-цели (соревнования, челленджи) |
| **Флегматик** | Дневник без давления. Постепенное осознание | Научные объяснения, готовые решения. Микро-шаги | Чек-листы, автоматизация. Внешний контроль | Новые протоколы с обоснованием. Сервисность |
| **Меланхолик** | Очень осторожно. Не травмировать. Только поддержка | Эмоциональное обучение. Вдохновляющие примеры | Мягкий трекинг. Ежедневная похвала | Цели, заряжающие эмоционально. Мудборд |

### 7.4. Границы ответственности нутрициолога

```
ЗОНА ОТВЕТСТВЕННОСТИ НУТРИЦИОЛОГА:
  - Психотип: сангвиник, холерик, флегматик, меланхолик (в балансе)
  - Уровень осознанности: I → II → III → IV (развитие по циклу)
  - Гиперконтроль: лёгкий уровень (десенсибилизация)

ГРАНИЦА (направить к психологу):
  - Гиперконтроль: умеренный и тяжёлый
  - РПП: орторексия, анорексия, булимия, компульсивное переедание
  - Депрессивные состояния, тревожные расстройства
  - Расстройства личности (пограничные, нарциссические)

ЗОНА ПСИХИАТРА / ПСИХОТЕРАПЕВТА:
  - Тяжёлые формы РПП с медицинскими последствиями
  - Суицидальные мысли
  - Психотические расстройства
```

> Нутрициолог может сопровождать клиента на фоне терапии у психолога, но не заменяет её.

## 8. Дизайн поведения и формирование привычек

### 8.1. Принципы дизайна поведения

Дизайн поведения — проектирование среды и процессов, которые делают здоровое поведение лёгким, очевидным и неизбежным.

**Формула привычки (по Чарльзу Дахиггу):**
```
Стимул (триггер) → Действие (рутина) → Награда (вознаграждение)
```

**Дополнение для нутрициологии (Fogg Behavior Model):**
```
Поведение = Мотивация × Способность × Триггер
         (B = MAP)
```

Если поведение не происходит — не хватает хотя бы одного элемента.

### 8.2. Архитектура триггеров

| Тип триггера | Пример для питания | Пример для воды |
|---|---|---|
| **Внешний (среда)** | Тарелка зелёного цвета на видном месте | Бутылка воды на рабочем столе |
| **Внутренний (эмоция)** | «Чувствую голод → сначала выпить воды» | Ощущение сухости во рту |
| **Контекстный (привязка)** | После чистки зубов — выпить стакан воды | Перед каждым приёмом пищи — 200 мл |
| **Социальный** | Чат с клиентами, отчёт перед коучем | Публичный трекер воды |
| **Цифровой** | Пуш-уведомление «Время завтрака» | Напоминание на телефоне каждый час |

### 8.3. Конструкторы боулов

Боул — универсальная формула сбалансированного приёма пищи, снижающая когнитивную нагрузку выбора.

**Структура боула (визуальная формула):**

```

│   тофу, бобовые)        │
├────────────┬────────────┤
│  1/2 ОВОЩИ │  1/4       │
│  (листовые,│  УГЛЕВОДЫ  │
│   цветные, │  (крупы,   │
│   сезонные)│   киноа,   │
│            │   батат)   │

```

**Готовые шаблоны боулов для клиентов:**

| Тип | Белок | Овощи | Углеводы | Заправка |

> **Для меланхолика:** фото каждого боула + чек-лист ингредиентов
> **Для холерика:** калорийность и КБЖУ каждого варианта в таблице
> **Для сангвиника:** дать свободу комбинаций по формуле

### 8.4. Организация кухонного пространства

**Зелёная зона (видная, доступная):**
- Тарелка-боул на видном месте
- Миска с фруктами на столе
- Бутылка воды на рабочем месте
- Полезные снеки (орехи, ягоды) на уровне глаз

**Красная зона (скрытая, труднодоступная):**
- Сладости, печенье — в верхний ящик или убрать из дома
- Чипсы, снеки — не покупать «про запас»
- Тарелки маленького размера для контроля порций

**Правило 3 шагов:**
> Полезное — в 1 шаге (мойка, нарезка, тарелка)
> Вредное — в 3+ шагах (верхняя полка, магазин, упаковка)

### 8.5. Формирование привычек по фазам (для клиента)

```
ФАЗА 1. СТАРТ (дни 1–7)
  - 1 микро-привычка (например, стакан воды утром)
  - Не больше 1–2 изменений одновременно
  - Трекер галочек (каждый день — ✅)

ФАЗА 2. ЗАКРЕПЛЕНИЕ (недели 2–4)
  - Добавить 1–2 новые привычки
  - Привязать к существующим ритуалам
  - Трекер + еженедельная сверка

ФАЗА 3. АВТОМАТИЗМ (1–3 месяца)
  - Привычка выполняется без напоминаний
  - Добавить новую сложность
  - Перевести на следующий уровень
```

**Пример внедрения привычки «Водный режим»:**

| День | Действие | Триггер | Инструмент |
|---|---|---|---|
| 1–3 | 1 стакан воды утром | После пробуждения | ✅ трекер |
| 4–7 | +1 стакан перед обедом | Перед едой | ✅ + напоминание |
| 2 нед | +1 перед ужином | Перед едой | ✅ автомат |
| 3 нед | Итого 6 стаканов в день | По триггерам | Без напоминаний |

### 8.6. Адаптация дизайна поведения по психотипам

| Психотип | Триггеры | Конструктор боулов | Организация среды |
|---|---|---|---|
| **Сангвиник** | Внутренняя мотивация. Внешние триггеры — мин. | 3–4 шаблона, далее сам комбинирует | Достаточно базовой организации |
| **Холерик** | Цифровые триггеры (пуши, графики). Вызов | Таблица КБЖУ к каждому боулу. Соревнование | Чёткое зонирование + KPI |
| **Флегматик** | Внешние (напоминания, чек-листы). Социальные (коуч) | 7 готовых шаблонов на неделю + список покупок | Полная сервисная организация — доставка продуктов |
| **Меланхолик** | Эмоциональные (фото результата, мудборд). Поддержка | Фото каждого боула, чек-лист ингредиентов | Мягкая организация без чувства вины |

### 8.7. Типичные ошибки при внедрении привычек

| Ошибка | Почему не работает | Что делать вместо |
|---|---|---|
| Слишком много изменений сразу | Перегрузка → срыв → отказ | 1 микро-привычка в неделю |
| Нет триггера | Привычка забывается | Привязать к существующему ритуалу |
| Нет награды | Нет дофаминового подкрепления | Трекер галочек + маленькая награда |
| Слишком сложное действие | Нет способности (Fogg) | Разбить на микро-шаги |
| Нет плана Б | Срыв → всё брошено | Заранее обговорить «что делать, если...» |
| Игнорирование психотипа | Неподходящий формат мотивации | Учитывать тип ВНД клиента |

## 9. От привычки к навыку: автоматизация поведения

### 9.1. Привычка vs Навык

| Критерий | Привычка | Навык |
|---|---|---|
| **Осознанность** | Требует внимания и трекера | Выполняется автоматически |
| **Усилие** | Требует волевого усилия | Не требует усилия |
| **Контроль** | Внешний (чек-лист, коуч) | Внутренний (интегрирован) |
| **Сбой** | При стрессе — легко срывается | Устойчив к стрессу |
| **Пример** | «Я напоминаю себе пить воду» | «Я пью воду, не задумываясь» |
| **Мозг** | Активна префронтальная кора | Базальные ганглии (автоматизм) |

### 9.2. 4-ступенчатый переход Habit → Skill

```
ПРИВЫЧКА (сознательное усилие)
  ↓  1. Повторение в стабильной среде
РИТУАЛ (действие по триггеру без размышлений)
  ↓  2. Добавление вариативности
ГИБКИЙ НАВЫК (адаптация к контексту)
  ↓  3. Интеграция в идентичность
МАСТЕРСТВО (навык стал частью личности)
```

**Ступень 1. Повторение (21–66 дней)**

Цель: довести действие до автоматизма в фиксированных условиях.
- Фиксированный триггер (место, время, контекст)
- Трекер галочек — источник быстрого дофамина
- Минимальная вариативность (одно и то же действие)

```
Пример: «Каждое утро после пробуждения — 500 мл воды»
→ повторять ежедневно 21 день
→ после 21 дня — пропустить 1 день, проверить:
  «Я почувствовал дискомфорт?» → если да — привычка сформирована
```

**Ступень 2. Вариативность (1–3 месяца)**

Цель: навык работает в разных контекстах, а не только в исходном.
- Добавить другие триггеры (время, место, состояние)
- Усложнить действие (увеличить объём, скорость)
- Проверить устойчивость к стрессу (выходной, болезнь, командировка)

```
Пример: «500 мл воды в любых условиях: дома, в офисе, в поездке»
→ если пропустил — восстановить без самобичевания
→ если выполнил в сложных условиях — бонусная галочка
```

**Ступень 3. Интеграция в идентичность (3–6 месяцев)**

Цель: навык становится частью личности — «я человек, который...».
- Сдвиг мотивации с внешней («надо») на внутреннюю («я такой»)
- Навык встраивается в систему более высокого порядка
- Автоматически тянет за собой смежные привычки

```
Пример:
  «Я пью воду» → «Я забочусь о гидратации» → «Я человек,
  который внимателен к своему организму»
→ водный режим перестаёт быть отдельной задачей
→ становится фоном
```

### 9.3. Инструменты автоматизации (убираем трекер)

Когда привычка сформирована, трекер становится лишним и даже вредным (создаёт лишний шум).

| Время | Действие | Трекер |
|---|---|---|
| 0–3 нед | Сознательное выполнение | ✅ ежедневно |
| 3–8 нед | Уверенное выполнение | ✅ раз в 3 дня |
| 2–3 мес | Пропуски без чувства вины | ❌ убрать трекер |
| 3+ мес | Автоматизм | ❌ навык |

**Когда можно убрать трекер:**
- Действие выполняется без напоминания 7 дней подряд
- Пропуск 1 дня не вызывает срыва остальных привычек
- Восстановление после пропуска происходит без усилия

### 9.4. Связка привычек (habit stacking) — переход к системе

Формула: **После [текущая привычка] я [новая привычка]**

```
Уже есть:      «Чищу зубы утром»
Добавляю:      «После чистки зубов — выпиваю стакан воды»
→ Водный режим привязан к чистке зубов

Уже есть:      «Варю кофе утром»
Добавляю:      «Пока варится кофе — делаю 10 приседаний»
→ Микро-активность привязана к кофе

Цепочка: Чищу зубы → Вода → Кофе → Зарядка → Завтрак (боул)
→ Целое утро становится системой здоровых ритуалов
```

### 9.5. Матрица зрелости навыка по психотипам

| Психотип | Скорость автоматизации | Риск срыва | Стратегия перехода | Когда убирать трекер |
|---|---|---|---|---|
| **Сангвиник** | Быстрая (3–4 нед) | Низкий | Дать свободу, трекер — опционально | Через 3–4 недели |
| **Холерик** | Средняя (4–6 нед) | Средний (бросает, если нет быстрого результата) | Добавить KPI, челленджи на скорость | Через 6–8 недель, когда метрики стабильны |
| **Флегматик** | Медленная (6–12 нед) | Высокий (при малейшем сбое) | Оставлять трекер дольше. Подстраховка. План Б | Через 10–12 недель, только при стабильности |
| **Меланхолик** | Средняя (4–8 нед) | Высокий (самобичевание при пропуске) | Трекер убрать раньше (снизить тревогу). Заменить на дневник самочувствия | Через 4–6 недель, заменить на «дневник ощущений» |

### 9.6. Переход на следующий уровень (цикл развития)

Когда навык автоматизирован — клиент входит в состояние **неосознанной компетентности**. Задача нутрициолога — поставить новую цель и запустить цикл заново.

```
УРОВЕНЬ 1. Базовые привычки (1–3 мес)
  Вода → Завтрак → Сон → Активность
  ↓ автоматизация

УРОВЕНЬ 2. Качество (3–6 мес)
  КБЖУ → Баланс макронутриентов → Качество продуктов
  ↓ автоматизация

УРОВЕНЬ 3. Оптимизация (6–12 мес)
  Анти-эйдж → Биохакинг → Спортивные цели
  ↓ автоматизация

УРОВЕНЬ 4. Мастерство (1+ год)
  Индивидуальные протоколы → Генетика → Долгосрочная профилактика
```

**Ключевой принцип:** новый уровень начинается с новой **неосознанной некомпетентности** — клиент снова не знает, что он не знает. Цикл повторяется.

### 9.7. Признаки готовности к переходу на новый уровень

| Область | Признак |
|---|---|
| **Питание** | Боул собирается без рецепта за 10 мин. Нет сомнений в выборе продуктов |
| **Вода** | Не замечает, что пьёт воду — просто пьёт |
| **Сон** | Засыпает в одно время без усилия, встаёт без будильника |
| **Активность** | Зарядка / прогулка — естественная часть дня, а не задача |
| **Осознанность** | Отслеживает состояние без записи. Чувствует, что работает / не работает |
| **Эмоции** | Срыв не выбивает. Восстановление — автоматическое |
