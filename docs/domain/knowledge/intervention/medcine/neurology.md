# Неврологические интервенции

## Показания (красные флаги)
- Головные боли с аурой (впервые возникшие)
- Прогрессирующее ухудшение памяти
- Парестезии и онемение конечностей
- Нарушение координации, атаксия
- Мигрень >4 эпизодов в месяц

## Интервенции

### MED_NEURO_01 — Консультация невролога
- **Тип:** diagnostic
- **Воздействие:** 9.0
- **Доказательность:** A
- **Периодичность:** По показаниям
- **Описание:** Направление к неврологу при хронических головных болях, когнитивных нарушениях, парестезиях, нарушениях сна неясной этиологии

### MED_NEURO_02 — МРТ головного мозга
- **Тип:** diagnostic
- **Воздействие:** 8.5
- **Доказательность:** B
- **Периодичность:** По показаниям
- **Описание:** МРТ при подозрении на объёмное образование, демиелинизацию, сосудистую патологию (после консультации невролога)

### MED_NEURO_03 — Профилактическая терапия мигрени
- **Тип:** medication
- **Воздействие:** 8.0
- **Доказательность:** A
- **Периодичность:** Ежедневно / ежемесячно
- **Описание:** Бета-блокаторы (пропранолол), антидепрессанты (амитриптилин), CGRP-моноклональные антитела при частых атаках мигрени

### MED_NEURO_04 — Когнитивное тестирование
- **Тип:** diagnostic
- **Воздействие:** 7.0
- **Доказательность:** B
- **Периодичность:** 1 раз в год
- **Описание:** MMSE, MoCA тесты для скрининга когнитивных нарушений (при жалобах на память, возраст >65 лет)
