# Варианты реакций пользователя на интервенции

При симуляции чата каждая интервенция (сообщение бота) вызывает случайную реакцию пользователя. Реакции имитируют ответ живого человека, который получает напоминание или задание.

| Категория интервенции | Варианты реакций |
|---|---|
| **Сон** (sleep) | ✅ Готово, выключил свет<br>📤 Отправил дневник сна<br>⏳ В следующий раз лягу раньше<br>👍 Лёг в 22:30<br>📝 Записал в трекер<br>😴 Уже сплю 🙂<br>🛌 Выполнено |
| **Физический** (physical) | ✅ Сделал зарядку<br>📤 Отправил шаги<br>👍 Пробежка выполнена<br>💪 Тренировка прошла отлично<br>🚶 Прошёл 8000 шагов<br>⏳ Пойду вечером<br>🏋️ Готово |
| **Ментальный** (mental) | ✅ Помедитировал<br>📤 Отправил чек-ин<br>🧘 Сделал дыхание<br>📝 Записал мысли<br>👍 Стало спокойнее<br>🎯 Выполнено |
| **Питание** (food) | ✅ Записал приём пищи<br>📤 Отправил фото еды<br>🥗 Съел салат вместо булки<br>💧 Выпил воду<br>⏳ Позже запишу<br>🍎 Выполнено |
| **Медицинский** (medical) | ✅ Принял препарат<br>📤 Отправил показания<br>🏥 Записался к врачу<br>💊 Выпил витамины<br>📊 Измерил давление<br>✅ Всё сделал |
| **Добавки** (supplement) | ✅ Принял<br>📤 Отправил фото упаковки<br>⏳ Забуду — напомни позже<br>💊 Выпил с завтраком<br>✅ Готово |

## Универсальные реакции (подходят для любых категорий)

- ✅ Готово!
- 📤 Отправил!
- ⏳ В следующий раз
- 👍 Сделано
- 📝 Записал
- ✅ Выполнено
- 📊 Готово, данные отправил

## Формат в коде

```js
const userResponses = [
  '✅ Готово!',
  '📤 Отправил!',
  '⏳ В следующий раз',
  '👍 Сделано',
  '📝 Записал',
  '✅ Выполнено',
  '📊 Готово, данные отправил',
];
```

Каждая реакция появляется через 1–2 секунды после сообщения бота, имитируя время на ответ.
