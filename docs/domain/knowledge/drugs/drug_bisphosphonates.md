# Бисфосфонаты

**Код:** DRG_15  
**Категория:** Костные  
**МНН:** Алендронат, ибандронат, золендронат, ризедронат  
**Торговые названия:** Фосамакс, Бонвива, Акласта, Актонель

---

## Механизм действия

Аналоги пирофосфата: связывание с гидроксиапатитом кости → ингибиция остеокластной резорбции → снижение костного оборота. Золендронат — азотсодержащий, наиболее мощный.

## Показания в нутрициологии

- Остеопороз (постменопаузальный, стероидный, возрастной)
- Остеопения (T-score <-2.5)
- Болезнь Педжета
- Профилактика переломов (позвонки, бедро)

## Влияние на нутритивный статус

| Нутриент | Эффект | Механизм | Риск |
|----------|--------|----------|------|
| **Ca (кальций)** | **Снижение** | Связывание с бисфосфонатами (пероральные) | **Средний** |
| **Mg (магний)** | **Снижение** | Связывание | Средний |
| **Fe (железо)** | **Снижение** | Связывание | Средний |
| **Витамин D** | **Критичен для эффективности** | D необходим для всасывания Ca | **Обязателен** |

## Критические правила приёма

| Правило | Описание |
|---------|----------|
| Время | Строго утром натощак, за 30-60 мин до еды |
| Вода | 200-250 мл простой воды (не минеральной!) |
| Положение | Вертикально (стоя/сидя), не ложиться 30-60 мин |
| Добавки | Ca, Mg, Fe — минимум за 2 ч ДО или через 4 ч ПОСЛЕ |
| Пропуск | Не удваивать дозу, пропустить до следующего приёма |

## Красные флаги

| Показатель | Порог | Действие |
|------------|-------|----------|
| Остеонекроз челюсти | Боль в челюсти, незаживающая лунка | Отмена БФ, консультация стоматолога (ONJ) |
| Атипичный перелом бедра | Боль в бедре (продром) | Рентген, отмена БФ |
| Гипокальциемия | Ca <2.1 ммоль/л | Ca в/в, отмена БФ до коррекции |
| Эзофагит | Загрудинная боль, дисфагия | Отмена per os, переход на в/в |
| eGFR | <35 мл/мин | Противопоказание (в/в БФ) |

## Доказательная база

- Уровень A: Алендронат снижает риск переломов позвонков на 47%, бедра на 40% (FIT, PMID: 8961486)
- Уровень A: Золендронат — снижение переломов позвонков на 70% (HORIZON, PMID: 17446522)
- Уровень A: Ca + D обязательны для эффективности БФ (PMID: 18240728)
- Уровень B: ONJ — 0.1-1% при БФ (более риск при в/в, онкология)
