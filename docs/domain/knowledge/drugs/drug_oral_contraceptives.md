# Оральные контрацептивы (КОК)

**Код:** DRG_05  
**Категория:** Гормональные  
**МНН:** Этинилэстрадиол + дроспиренон / левоноргестрел / дезогестрел  
**Торговые названия:** Ярина, Джес, Линдинет, Регулон, Новинет

---

## Механизм действия

Комбинированные оральные контрацептивы (КОК): подавление овуляции через эстроген + прогестин. Эстроген ускоряет метаболизм ряда нутриентов через индукцию ферментов печени (цитохром P450, глюкуронилтрансферазы).

## Показания в нутрициологии

- Контрацепция (основное)
- Гормональная коррекция (СПКЯ, эндометриоз, акне)
- Регуляция цикла

---

## Влияние на нутритивный статус

| Нутриент | Эффект | Механизм | Риск |
|----------|--------|----------|------|
| **Фолат (B9)** | **Дефицит** | Ускорение метаболизма, снижение абсорбции | **Высокий** |
| **B6 (пиридоксин)** | **Дефицит** | Индукция триптофан-пирролазы | **Высокий** |
| B2 (рибофлавин) | Дефицит | Ускорение выведения | Средний |
| **Mg (магний)** | **Снижение** | Перераспределение, усиление экскреции | **Высокий** |
| **Zn (цинк)** | **Снижение** | Связывание с глобулинами, снижение абсорбции | **Высокий** |
| Витамин C | Снижение | Ускорение окисления | Средний |
| CoQ10 | Снижение | Непрямое | Низкий |
| Селен (Se) | Снижение | Связывание | Низкий |

## Психонутритивные эффекты

- Дефицит B6 → снижение синтеза серотонина (ПМС, депрессия)
- Дефицит Mg → тревожность, нарушения сна, судороги
- Дефицит Zn → акне, ломкость волос, снижение иммунитета
- Дефицит фолата → гипергомоцистеинемия (тромбозы)

## Протокол сопровождения

| Интервенция | Дозировка | Периодичность |
|-------------|-----------|---------------|
| Фолат (метилфолат) | 400-800 мкг | Ежедневно |
| B6 (P-5-P) | 25-50 мг | Ежедневно |
| Mg (глицинат/цитрат) | 200-400 мг | Ежедневно |
| Zn (пиколинат) | 15-30 мг | Ежедневно |
| Комплекс B | B2, B12 | Ежедневно |
| Гомоцистеин | Контроль <8 мкмоль/л | 1 раз в 6-12 мес |

## Особые группы

| Группа | Доп. риски | Рекомендация |
|--------|-------------|--------------|
| Курящие (≥35 лет) | Тромбоз, дефицит C, E | Витамин C 500 мг, E 400 МЕ |
| СПКЯ | Инсулинорезистентность, дефицит Mg | Mg 400 мг, инозитол 2-4 г |
| Длительный приём (>5 лет) | Кумулятивный дефицит B6, фолата | Мониторинг гомоцистеина + MMA |
