# Левотироксин (L-T4)

**Код:** DRG_04  
**Категория:** Гормональные  
**МНН:** Левотироксин натрия  
**Торговые названия:** Эутирокс, L-тироксин

---

## Механизм действия

Синтетический T4 (тироксин) — заместительная терапия при гипотиреозе. В тканях превращается в активный T3 (трийодтиронин) через дейодиназы (Se-зависимые ферменты).

## Показания в нутрициологии

- Гипотиреоз (первичный, вторичный)
- Субклинический гипотиреоз
- Зоб (диффузный, узловой)
- Аутоиммунный тиреоидит (Хашимото)

---

## Влияние на всасывание

Критически важны интервалы приёма из-за связывания с пищей и добавками:

| Вещество | Эффект | Интервал | Механизм |
|----------|--------|----------|----------|
| Кальций (Ca) | Снижение всасывания | ≥4 часа | Хелатирование |
| Железо (Fe) | Снижение всасывания | ≥4 часа | Комплексообразование |
| Магний (Mg) | Снижение всасывания | ≥4 часа | Комплексообразование |
| Цинк (Zn) | Снижение всасывания | ≥4 часа | Комплексообразование |
| Соевый белок | Снижение всасывания | ≥4 часа | Связывание |
| Клетчатка (высокие дозы) | Снижение всасывания | ≥1 час | Адсорбция |
| Селен (Se) | Синергия (конверсия T4→T3) | Совместно | Дейодиназы |
| Жирные кислоты | Не влияет | Любой | — |

## Нутриенты для конверсии T4→T3

| Нутриент | Роль | Дозировка |
|----------|------|-----------|
| Селен (Se) | Дейодиназы (тип 1, 2) | 55-100 мкг/сут |
| Цинк (Zn) | Кофактор дейодиназ | 15-30 мг/сут |
| Железо (Fe) | Тиреоидная пероксидаза | По дефициту |
| Йод (I) | Синтез T4/T3 | 150 мкг/сут |

## Протокол сопровождения

| Интервенция | Дозировка | Периодичность |
|-------------|-----------|---------------|
| Селен (селенометионин) | 55-100 мкг | Ежедневно |
| Цинк (Zn) | 15-30 мг | Ежедневно |
| Mg (глицинат) | 200-400 мг | Через ≥4 ч после L-T4 |
| Контроль ТТГ | Цель 0.5-2.5 мМЕ/л | 1 раз в 3-6 мес |
| Контроль T4 св., T3 св. | В норме | 1 раз в 3-6 мес |

## Красные флаги

| Показатель | Порог | Действие |
|------------|-------|----------|
| ТТГ >10 мМЕ/л | Нескорректированный гипотиреоз | Увеличить дозу под контролем врача |
| ТТГ <0.1 мМЕ/л | Гипертиреоз (передозировка) | Снизить дозу под контролем врача |
| Анти-TPO >100 | Аутоиммунный процесс | Добавить Se 100 мкг + контроль |
| Пропуск приёма | Более 3 дней | Удвоить дозу на 1 день (по назначению врача) |
