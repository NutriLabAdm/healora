# ГКС (глюкокортикостероиды)

**Код:** DRG_12  
**Категория:** Противовоспалительные  
**МНН:** Преднизолон, метилпреднизолон, дексаметазон, гидрокортизон  
**Торговые названия:** Преднизолон, Метипред, Дексаметазон, Солу-Медрол

---

## Механизм действия

Синтетические аналоги кортизола: связывание с цитоплазматическими рецепторами ГК → транскрипция противовоспалительных белков (липокортин, IκB) и подавление провоспалительных цитокинов (NF-κB, AP-1). Системное противовоспалительное и иммуносупрессивное действие.

## Показания в нутрициологии

- Аутоиммунные заболевания (РА, СКВ, васкулиты)
- Бронхиальная астма, ХОБЛ (системные формы)
- Воспалительные заболевания кишечника (ВЗК)
- Аллергические реакции (тяжёлые)
- Постковидный синдром (воспалительный фенотип)

## Влияние на нутритивный статус

| Нутриент | Эффект | Механизм | Риск |
|----------|--------|----------|------|
| **Кальций (Ca)** | **Деминерализация костей** | Подавление остеобластов, стимуляция остеокластов, снижение всасывания Ca | **Высокий** (остеопороз) |
| **Витамин D** | **Снижение** | Подавление синтеза D-связывающего белка, ускорение метаболизма | **Высокий** |
| **Белок (мышечный)** | **Катаболизм** | Усиление глюконеогенеза, распад мышечного белка | **Высокий** |
| **K (калий)** | **Дефицит** | Минералокортикоидная активность → калийурез | Средний |
| **Mg (магний)** | **Снижение** | Усиление экскреции | Средний |
| Fe | Перераспределение | Секвестрация (анемия хронического воспаления) | Средний |
| **Глюкоза** | **Гипергликемия** | Инсулинорезистентность, стимуляция глюконеогенеза | **Высокий** (стероидный диабет) |
| Na | Задержка | Минералокортикоидная активность | Средний (отёки) |

## Длительность и риск

| Длительность | Риск остеопороза | Риск миопатии | Риск гипергликемии |
|--------------|-------------------|---------------|-------------------|
| <3 мес | Низкий | Низкий | Средний |
| 3-6 мес | Средний | Средний | Высокий |
| >6 мес | **Высокий** | **Высокий** | **Очень высокий** |
| >1 года | Остеопороз у >50% | Саркопения | СД 2 типа у 20-30% |

## Протокол сопровождения

| Интервенция | Дозировка | Периодичность |
|-------------|-----------|---------------|
| Ca цитрат + D3 | Ca 1000-1200 мг + D 2000-4000 МЕ | Ежедневно |
| Mg (глицинат) | 200-400 мг | Ежедневно |
| Белок (протеин) | 1.5-2.0 г/кг | Ежедневно |
| K (при дефиците) | 20-40 мэкв | По контролю |
| Контроль глюкозы | Гликемический профиль | Ежедневно |
| Денситометрия (DEXA) | Z-score | 1 раз в год |
| Витамин K2 (MK-7) | 45-90 мкг | Ежедневно (остеопротекция) |

## Красные флаги

| Показатель | Порог | Действие |
|------------|-------|----------|
| Гипергликемия | Глюкоза >11 ммоль/л | Коррекция диеты, инсулин по назначению |
| Остеопороз | T-score <-2.5 | Бисфосфонаты, Ca + D + K2 |
| Миопатия | Слабость проксимальных мышц | Белок 2.0 г/кг, L-карнитин, Mg |
| Психоз | Ажитация, бессонница | Снижение дозы, консультация психиатра |
| Надпочечниковая недостаточность | Отмена >3 нед приёма | Постепенное снижение, тест стимуляции АКТГ |
| Гипокалиемия | K <3.5 ммоль/л | K+ добавки, контроль |
