# Инструменты дизайна поведения

> «Мы не можем изменить человека. Мы можем только изменить среду вокруг него так, чтобы правильное поведение стало путём наименьшего сопротивления.»

## Инструмент 1. Сложность-лёгкость (Difficulty-Ease)

### Принцип
Сделать желаемое поведение **максимально лёгким**, а нежелаемое — **максимально сложным**.

### Приёмы

| Приём | Пример |
|-------|--------|
| Список покупок | Составлять заранее, не ходить в магазин голодным |
| Простые рецепты | 10–15 минут на приготовление, минимальное количество ингредиентов |
| Организация кухни | Полезные продукты на видном месте, вредные — в дальний ящик или не покупать |
| Органайзер для БАД | На столе рядом с чашкой кофе (визуальный якорь) |
| Вечерняя подготовка | Спортивная форма надета с вечера, овощи помыты и нарезаны |

### Нейрофизиологическое обоснование
Утром уровень кортизола высок, а префронтальная кора (отвечающая за сознательные решения) ещё не активна. Поэтому утром человек действует «на автопилоте». Подготовка **с вечера** делает правильное поведение утром автоматическим.

## Инструмент 2. Смена идентичности (Self-Identity)

### Принцип
Изменить самовосприятие: **«Я — человек, который заботится о себе»**, а не «Я должен ходить в зал 3 раза в неделю».

### Цикл идентичности
```
Аффирмация («Я — спортивный человек»)
      ↓
   Действие (надевает кроссовки, выходит на пробежку)
      ↓
   Убеждение (после 3-х пробежек: «Я действительно могу»)
      ↓
   Идентичность (формируется новая идентичность)
      ↓
   (цикл повторяется)
```

### Примеры фраз для смены идентичности
- «Я — человек, который выбирает здоровую еду»
- «Я — тот, кто заботится о своём теле»
- «Для меня нормально — двигаться каждый день»

**Источник:** Clear, J. (2018). *Atomic Habits*. Avery. — Глава об identity-based habits.

## Инструмент 3. Выращивание доминанты (Growing the Dominant)

### Принцип
Сделать полезную привычку настолько сильной, чтобы она «перевешивала» все альтернативы.

### Методы

| Метод | Описание | Пример |
|-------|----------|--------|
| **Группировка привычек** | Связать несколько привычек в цепочку | Утренний ритуал: коврик-акупунктура → умывание → пробежка |
| **Критическая масса информации** | Накопить достаточно знаний о пользе привычки | Изучить, почему стояние на аппликаторе Кузнецова улучшает кровообращение и снимает боль в спине |
| **Фокус внимания** | Постоянные напоминания | Трекеры привычек, визуальные доски, пуш-уведомления приложений, умные браслеты |
| **Поддерживающая среда** | Социальное окружение | Группы от 5 человек (в парах 90% — один отсеивается, в группах — выше удержание) |

**Источник:** Ильичёва ссылается на концепцию доминанты Ухтомского (А. А. Ухтомский, «Доминанта», 1923) — физиологический принцип, согласно которому один очаг возбуждения в ЦНС подавляет другие.

## Инструмент 4. Смена динамических стереотипов (Changing Dynamic Stereotypes)

### Принцип
**Динамический стереотип** — это цепочка привычных действий, запускаемая определённым триггером и подкреплённая постоянным повторением.

Пример: Работа → усталость → коллеги → чай с печеньем (ритуал).

**Правило:** Нельзя разорвать всю цепочку сразу, но можно **заменить одно звено**.

### Алгоритм замены

1. **Определить цепочку**: выявить все звенья от триггера до конечного действия
2. **Выбрать одно звено** для замены (самое уязвимое или самое простое)
3. **Предложить альтернативу** с той же функцией (не печенье к чаю, а травяной чай или тёплая вода)
4. **Закрепить новое звено** через повторение (21–66 дней для автоматизма)

### Примеры замены звеньев

| Исходная цепочка | Заменяемое звено | Новая цепочка |
|-----------------|-----------------|---------------|
| Стресс → диван → телевизор → вечернее переедание | Телевизор → прогулка | Стресс → диван → прогулка → ужин (меньше) |
| Совещание → пирожные → упадок энергии | Пирожные → орехи + вода | Совещание → орехи → стабильная энергия |

**Источник:** Павлов, И. П. (1927). *Conditioned Reflexes*. Oxford University Press. — Термин «динамический стереотип» введён Павловым для описания закреплённых последовательностей условных рефлексов.
