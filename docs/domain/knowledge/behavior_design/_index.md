# Дизайн поведения

**Дизайн поведения** — это системный подход к изменению поведения человека через перепроектирование среды, управление триггерами и эмоциональную окраску действий. В отличие от волевых методов («взять себя в руки»), дизайн поведения **не требует силы воли** — он делает правильное поведение путём наименьшего сопротивления.

## Ключевое соотношение: поведение vs намерения

| Фактор | Доля влияния | Источник |
|--------|-------------|----------|
| **Среда** (доступность, организация) | ~30% | Ильичёва (МИИН), Clear (Atomic Habits) |
| **Эмоции** (окраска переживания) | ~30% | Ильичёва, Fogg (B=MAP) |
| **Триггеры** (контекстные якоря) | ~25% | Ильичёва, Duhigg (Habit Loop) |
| **Опыт** (повторение, привычка) | ~10% | Ильичёва |
| **Рациональные убеждения** (ценности, знания) | ~5% | Ильичёва |

> **Вывод:** 95% поведения определяется бессознательными факторами. Работа только с убеждениями (знаниями) — самый слабый рычаг изменений.

## Связь с теориями поведения

```
АТОMIC HABITS (Clear)       FOGG (B=MAP)            HABIT LOOP (Duhigg)
─────────────────────       ────────────────────     ─────────────────────
1. Make it obvious           Trigger (Триггер)        Cue (Стимул)
2. Make it attractive        Motivation (Мотивация)   Craving (Желание)
3. Make it easy              Ability (Способность)    Response (Реакция)
4. Make it satisfying        Reward (Награда)         Reward (Награда)
```
