# Шаблон: Сердечно-сосудистое здоровье

- **Протокол**: 3.1
- **Категория**: wellness
- **Назначение**: Профилактика сердечно-сосудистых заболеваний

## Заключение

Пациенту рекомендован комплекс мероприятий для поддержания
сердечно-сосудистого здоровья. Основные направления: регулярный
мониторинг АД и ЧСС, аэробные нагрузки, нутрицевтическая поддержка.

## Интервенции

| Код | Название | Периодичность |
|-----|----------|---------------|
| BIO_BP | Измерение АД | D |
| BIO_HRV | Вариабельность ритма | D |
| SL_BED | Сон: время отхода | D |
| FD_WTR | Водный режим | D |
| SP_02 | Витамин D3 | D |

## Ключевые назначения

- **АД**: контроль 2 раза/день, цель <130/85
- **ЧСС**: утренний пульс, HRV ежедневно
- **Активность**: 150 мин/нед аэробных нагрузок
- **Омега-3**: 1-2 г/день
