# Шаблон: Долголетие

- **Протокол**: 3.3
- **Категория**: wellness
- **Назначение**: Комплексная программа долголетия

## Заключение

Пациенту рекомендована комплексная программа активного долголетия.
Основные направления: метаболическое здоровье, физическая активность,
нутрицевтическая поддержка, управление стрессом.

## Интервенции

| Код | Название | Периодичность |
|-----|----------|---------------|
| FD_WTR | Водный режим | D |
| FD_TIM | Тайминг питания | D |
| BIO_BP | Измерение АД | D |
| SP_02 | Витамин D3 | D |
| SP_03 | Витамин C | D |
| SP_39 | Ресвератрол | D |
| SP_27 | Кофермент Q10 | D |

## Ключевые назначения

- **Питание**: средиземноморская диета + интервальное голодание (16:8)
- **Активность**: комбинация аэробных + силовых тренировок
- **Стресс**: медитация, дыхательные практики
- **Добавки**: D3 2000-5000 МЕ, Mg 300-400 мг, Q10 100-200 мг
- **Мониторинг**: ежегодный скрининг + биоимпеданс
