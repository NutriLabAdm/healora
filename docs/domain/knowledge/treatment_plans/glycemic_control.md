# Шаблон: Гликемический контроль

- **Протокол**: 1.1
- **Категория**: nutritional
- **Назначение**: Контроль уровня глюкозы через питание и физическую активность

## Заключение

Пациенту рекомендован курс нутрициологической коррекции гликемического профиля.
Основные направления: контроль углеводной нагрузки, регулярная физическая активность,
мониторинг гликемии в динамике.

## Интервенции

| Код | Название | Периодичность |
|-----|----------|---------------|
| FD_WTR | Водный режим | D |
| FD_TIM | Тайминг питания | D |
| FD_PRO | Белковая норма | D |
| FD_FAT | Жировой профиль | D |
| PH_Z2 | Тренировки: зона 2 | W |
| MN_MDT | Медитация | D |

## Ключевые назначения

- **Питание**: дробный приём пищи 4-5 раз/день, исключить быстрые углеводы
- **Активность**: аэробные нагрузки (зона 2) 3-4 раза/нед
- **Мониторинг**: глюкоза натощак + постпрандиальная через 2ч
