## О формате

Треклист — это Google-таблица, которую клиент ведёт ежедневно. Специалист оставляет комментарии. Преимущества:
- На одной странице видна динамика: точка А → неделя 1 → неделя 2 → точка Б
- Клиент видит свой прогресс и чувствует ответственность
- Специалист может точечно корректировать поведение

**Структура каждой таблицы:**
- Строки: дни недели (ПН–ВС) + дата
- Столбцы: привычки / показатели / комментарий специалиста
- Цветовая маркировка: 🟢 выполнено / 🟡 частично / 🔴 не выполнено

## Кейс 1. Коррекция веса

**Клиент:** Женщина, 34 года, ИМТ 29, запрос — минус 8 кг за 3 месяца

**Привычки на неделю:**
1. Вода — 1,5–2 л в день (норма)

3. Прогулка — 6000+ шагов
4. Ужин — без углеводов после 19:00

## Кейс 5. Отношения с едой / нормализация пищевого поведения

## Правила работы с треклистом

1. **Не больше 4 привычек одновременно** — иначе клиент перегружен
2. **Одна неделя = один шаг** — новая привычка добавляется только после закрепления предыдущей
3. **Цветовая маркировка** — 🟢/🟡/🔴 для быстрой визуальной оценки
4. **Еженедельный комментарий специалиста** — не только по факту, но и с прогнозом на следующую неделю
5. **Отмечать инсайты клиента** — когда клиент сам заметил связь (сон→энергия, пропуск еды→переедание), это сильнее любой рекомендации

## Промпты для презентации

### Для клиента
> **Заголовок:** «Без названия: план действий для улучшения здоровья»
>
> **Теория:**
> - Цель протокола: улучшение показателей здоровья
> - Кому подходит и когда начинать
> - Основные шаги и их последовательность
> - Ожидаемые результаты и сроки
>
> **Практика:**
> - Пошаговый план на неделю/месяц
> - Чек-лист ежедневных действий
> - Трекер прогресса (что записывать)
> - Когда ждать первых результатов
>
> **Дисклеймер:** Протокол носит ознакомительный характер. Индивидуальная программа составляется специалистом.

### Для специалиста
> **Заголовок:** «Без названия: клинический протокол ведения»
>
> **Теория:**
> - Обоснование протокола (биологические механизмы)
> - Показания и противопоказания (стратификация пациентов)
> - Доказательная база и уровни рекомендаций
>
> **Практика:**
> - Алгоритм: скрининг → стратификация → интервенция → мониторинг
> - 
> - Критерии эффективности и точки коррекции
> - Протокол безопасности (побочные эффекты, лабораторный контроль)
>
> **Дисклеймер:** Референсные протоколы. Адаптируются под конкретного пациента.
