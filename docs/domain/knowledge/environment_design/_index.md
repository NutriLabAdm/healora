## Связь с теориями поведения

```
Atomic Habits (Clear)        Fogg Behavior Model          Duhigg's Habit Loop
─────────────────────        ────────────────────         ─────────────────────
1. Make it obvious    ────►  Trigger (Триггер)       ────► Cue (Стимул)
2. Make it attractive ────►  Motivation (Мотивация)   ────► Craving (Желание)
3. Make it easy       ────►  Ability (Способность)   ────► Response (Реакция)
4. Make it satisfying ────►  Reward (Награда)         ────► Reward (Награда)
```

## Ключевое соотношение: среда vs сознание

| Фактор | Доля влияния на поведение | Что делаем |
|---|---|---|
| **Среда** (доступность, организация пространства) | 30% | Перепроектируем физическое окружение |
| **Эмоции** (окраска поступка) | 30% | Делаем полезное приятным |
| **Триггеры** (предметы, контекст, напоминания) | 25% | Добавляем визуальные и контекстные якоря |
| **Опыт** (повторение, привычка) | 10% | Тренируем микро-привычки |
| **Рациональное** (потребности, ценности, убеждения) | 5% | Работаем с идентичностью и ценностями |

> **Вывод:** 95% поведения определяется бессознательными факторами. Дизайн среды — самый эффективный рычаг изменения поведения.
