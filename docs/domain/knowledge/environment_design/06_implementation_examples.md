## Сквозные инструменты всех кейсов

### 1. Дневник питания с фото

**Формат:** клиент фотографирует каждую тарелку / перекус до еды + отмечает время.

| Параметр | Рекомендация |
|---|---|
| **Когда фоткать** | Перед каждым приёмом пищи и перекусом |
| **Что должно быть в кадре** | Тарелка + рука / предмет для масштаба + стол/фон |
| **Куда отправлять** | Чат с диетологом / Telegram-бот / приложение |
| **Частота ревью** | Диетолог просматривает раз в 3 дня на старте, затем раз в неделю |
| **Психотип-адаптация** | Сангвиник — свободный график, холерик — чек-лист с временем, флегматик — шаблон, меланхолик — без критики |

> **Почему это работает:** фото фиксирует реальный объём порции (а не «на глаз»). Исследования показывают: visual food diary снижает переедание на 30% за счёт осознанности (Zepeda & Deal, 2008).

### 2. Планирование закупок

**Принцип:** список покупок составляется под план питания на неделю. Клиент идёт в магазин сытым, с листом.

| Элемент | Детали |
|---|---|
| **Меню на неделю** | 7 завтраков, 7 обедов, 7 ужинов, 2-3 перекуса |
| **Список покупок** | Сгруппирован по отделам магазина (овощи, мясо, бакалея, молочка) |
| **Правило «зелёной корзины»** | 60% корзины — овощи/фрукты/зелень, 20% — белок, 15% — сложные углеводы, 5% — полезные снеки |
| **Бюджетирование** | Ориентировочная стоимость корзины на неделю |
| **Тайминг** | Закупка 1 раз в неделю (выходной) + 1 дозакупка свежих овощей (среда) |

### 3. Сопровождение врача-диетолога

**Роль в дизайне среды:** диетолог — внешняя система контроля, обратной связи и коррекции.

| Функция диетолога | Частота | Формат |
|---|---|---|
| Ревью дневника питания с фото | 1 раз в 3 дня (первые 2 нед), затем 1 раз в нед | Чат / асинхронно |
| Корректировка плана питания | 1 раз в 2 нед | Видео-созвон 30 мин |
| Анализ динамики веса и замеров | Еженедельно | Таблица / график |
| Анализ списка покупок и корзины | 1 раз в нед | Чат |
| Психотип-адаптация (см. `04_cjm_therapy_plans.md`) | Подбор частоты и формата | Индивидуально |

## Кейс 1. «Офисный сотрудник, 34 года, холерик» — переедание на работе

### Профиль
- **Психотип:** Холерик
- **Запрос:** Сбросить 12 кг, убрать вечерние перекусы, снизить тягу к сладкому
- **Диетолог:** Да, наблюдается у диетолога, назначен план 1800 ккал с дефицитом 20%

### Аудит среды

| Проблема | Фактор | Влияние на контроль веса |
|---|---|---|
| Нет дисциплины фото-дневника | Отсутствие инструмента | Не видит реальный объём еды -> переедает |
 | Печеньки на офисной кухне | Среда: вредное в 1 шаге | Каждый день 300-500 ккал сверх плана |
| Доставка фастфуда в 2 клика | Среда: слишком легко | 3 раза в неделю спонтанные заказы |
| Закупки без списка | Нет планирования | Импульсивные покупки в магазине |

### Дизайн среды — контроль веса

| Область | Что сделали | Связь с инструментом |
|---|---|---|
| **Дневник питания (фото)** | Клиент сфоткал тарелку до еды -> отправил в чат диетологу. Диетолог раз в 2 дня комментирует объём/состав. Через 2 недели — раз в неделю | Фото-дневник + ревью диетолога |
| **Список покупок** | Диетолог составил шаблон на неделю под 1800 ккал. Клиент идёт с распечаткой. Сгруппировано по отделам | Планирование закупок |
| **Офисная среда** | Своя коробка с полезными снеками на столе. Удалил приложение доставки | Среда: зелёная зона |
| **Правило тарелки** | Каждый приём: 1/2 овощи, 1/4 белок, 1/4 углеводы. Фото-контроль | Дневник питания |
| **Социальное** | Публичное обещание в рабочем чате: «21 день без доставки, штраф — 500 руб в общий котёл» | Приверженность |

### Динамика

| Метрика | Неделя 0 | Неделя 3 | Неделя 8 | Источник |
|---|---|---|---|---|
| Вес | 94 кг | 90 кг | 85 кг | Еженедельное взвешивание |
| Дефицит ккал | -200 | -450 | -500 | Фото-дневник -> анализ диетолога |
| Спонтанные заказы | 3/нед | 1/нед | 0/нед | Лог чата |
| Выполнение списка покупок | нет | 60% | 90% | Чек фото чека |

### Роль диетолога

| Время | Действие |
|---|---|
| Старт | Составил план 1800 ккал, шаблон списка покупок, памятку по фото-дневнику |
| Неделя 1-2 | Ежедневный просмотр фото тарелок, корректировка порций |
| Неделя 3-8 | Просмотр 1 раз в 3 дня, созвон 1 раз в 2 недели для корректировки |
| Месяц 3 | Переход на самоконтроль с ежемесячной сверкой |

## Кейс 2. «Мама в декрете, 29 лет, меланхолик» — хаотичное питание и ночные перекусы

### Профиль
- **Психотип:** Меланхолик
- **Запрос:** Перестать есть после 22:00, сбросить 7 кг, вернуть энергию
- **Диетолог:** Да, но требуется мягкий формат без критики

### Аудит среды

| Проблема | Фактор | Влияние на контроль веса |
|---|---|---|
| Еда на бегу, нет тарелки | Нет фото-дневника | Не осознаёт объём -> 2500+ ккал в день |
| Холодильник — соблазн вечером | Среда: еда в 1 шаге | Перекусы после 22:00 каждый день |
| Нет плана закупок | Нет структуры | Покупает что под руку попадётся -> нет здоровых продуктов |
| Стыд за срывы | Эмоции: самобичевание | -> стресс -> ещё больше еды |

### Дизайн среды — контроль веса

| Область | Что сделали | Связь с инструментом |
|---|---|---|
| **Дневник питания (фото)** | Формат «мой лучший приём» — фоткает ТОЛЬКО красивую полезную тарелку. Без критики. Диетолог комментирует через «здорово выглядит!» — позитивное подкрепление | Фото-дневник (сниженный порог) |
| **Планирование закупок** | Диетолог составил меню на неделю с простыми рецептами (до 20 мин). Сформировал список покупок. Партнёр берёт закупку на себя | Планирование закупок + делегирование |
| **Зелёная зона** | Вечерняя тарелка приготовлена с утра. Сладости не покупать. Фрукты и овощи — на уровне глаз | Среда: зонирование |
| **Ритуал вместо перекуса** | 21:00 — чай + книга (5-10 мин). Телефон — в другую комнату | Среда: замена триггера |
| **Контроль диетолога** | Чат: каждый вечер клиент пишет одно слово («Норма» / «Срыв») -> диетолог отвечает тёплым сообщением | Сопровождение без давления |
| **Поддержка** | Чат с мамами: «Кто сегодня без перекуса после 22:00? ✅» | Социальная поддержка |

### Динамика

| Метрика | Неделя 0 | Недера 3 | Неделя 8 |
|---|---|---|---|
| Вес | 68 кг | 65 кг | 62 кг |
| Еда после 22:00 | Каждый день | 2 раза в нед | 0-1 раз в нед |
| Фото-дневник | 0 | 5/7 дней | 7/7 дней |
| Энергия (субъективно) | 2/10 | 6/10 | 8/10 |

### Роль диетолога

| Время | Действие |
|---|---|
| Старт | Мягкая беседа, без цифр. Составил простой план на 7 дней. Объяснил: «никаких запретов, только добавки» |
| Неделя 1-4 | Ежедневный отклик в чате. Только похвала. Никаких замечаний |
| Месяц 2 | Ввёл лёгкую обратную связь: «что можно добавить в тарелку для сытости?» |
| Месяц 3 | Снижение частоты: созвон 1 раз в 2 недели |

## Кейс 3. «Бизнесмен, 45 лет, флегматик» — метаболический синдром

### Профиль
- **Психотип:** Флегматик
- **Запрос:** Нормализовать вес и давление по назначению врача-диетолога (диагноз: ожирение I ст., АГ 1 ст.)
- **Диетолог:** Ведёт врач-диетолог (регулярное наблюдение, лабораторный контроль)

### Аудит среды

| Проблема | Фактор | Влияние на контроль веса |
|---|---|---|
| Нет структуры питания | Отсутствие планирования | Еда «как получится» -> перекос по КБЖУ |
| Командировки срывают режим | Форс-мажор | 3-5 дней без контроля |
| Нет дневника | Нет обратной связи | Клиент не видит, сколько ест |
| Гигантские порции вечером | Среда: большие тарелки | Переедание на 600-800 ккал |

### Дизайн среды — контроль веса

| Область | Что сделали | Связь с инструментом |
|---|---|---|
| **Дневник питания (фото)** | Клиент фоткает каждую тарелку в течение дня. Врач-диетолог проверяет раз в 3 дня. Обратная связь через табличку: «белок ✅ / овощи ✅ / порция ⚠️» | Фото-дневник + structured feedback |
| **Планирование закупок** | Диетолог сформировал меню на неделю + матрицу закупок с группировкой по отделам. Добавлена автоматическая подписка на доставку овощей/зелени | Планирование + автоматизация |
| **Холодильник** | Зелёная зона на уровне глаз (овощи, готовый белок). Красная зона — нижний ящик (редкое). Большие тарелки убраны -> на видном месте маленькие (20 см) | Среда: зонирование |
| **Командировки** | Чек-лист «еда в дорогу» в приложении диетолога: снеки, рестораны с КБЖУ, план Б на каждый день | Форс-мажор |
| **Вес и замеры** | Еженедельное взвешивание + фото в одном ракурсе. Врач отслеживает тренд | Медицинский контроль |
| **Врач-диетолог** | Приём 1 раз в 2 недели с анализом дневника, лабораторным контролем (глюкоза, HbA1c, липиды) | Медицинское сопровождение |

### Динамика

| Метрика | Неделя 0 | Неделя 4 | Неделя 12 | Норма (цель) |
|---|---|---|---|---|
| Вес | 102 кг | 96 кг | 88 кг | < 85 кг |
| ИМТ | 32.1 | 30.2 | 27.7 | < 25 |
| Глюкоза | 6.8 | 6.1 | 5.4 | < 5.6 |
| HbA1c | 6.5% | 6.0% | 5.6% | < 5.7% |
| Давление | 150/95 | 135/85 | 125/80 | < 130/80 |
| Выполнение списка покупок | — | 70% | 95% | > 90% |

### Роль диетолога

| Время | Действие |
|---|---|
| До старта | Лабораторная диагностика, план питания с дефицитом 500 ккал, список покупок, шаблон фото-дневника |
| Каждые 2 недели | Приём: анализ дневника, замеры, коррекция, контроль лаборатории |
| Еженедельно | Проверка фото-дневника (асинхронно), обратная связь по списку покупок |
| Месяц 3 | Контрольная лаборатория, переход на поддерживающий режим |

## Кейс 4. «Студентка, 22 года, сангвиник» — набор веса на фоне гиперконтроля

### Профиль
- **Психотип:** Сангвиник (с элементами лёгкого гиперконтроля)
- **Запрос:** Сбросить 5 кг, перестать зацикливаться на КБЖУ, нормализовать отношения с едой
- **Диетолог:** Да, с фокусом на психообразование и снятие тревоги о еде

### Аудит среды

| Проблема | Фактор | Влияние на контроль веса |
|---|---|---|
| Счётчик калорий 24/7 | Цифровая среда | Гиперконтроль -> тревога -> срывы |
| Дневник питания = подсчёт цифр | Ошибка формата | Фокус на запретах, а не на добавках |
| Нет списка покупок | Нет планирования | Импульсивные покупки в общежитии |
| Общага: общая кухня, бюджет | Среда: ограничения | Сложно готовить -> дешёвые полуфабрикаты |

### Дизайн среды — контроль веса

| Область | Что сделали | Связь с инструментом |
|---|---|---|
| **Дневник питания (фото)** | Убрала все трекеры калорий. Формат: фото тарелки + одно слово о самочувствии. Диетолог комментирует: «отличный баланс!» | Фото-дневник (без цифр) |
| **Планирование закупок** | Диетолог составил список дешёвых полезных продуктов (яйца, крупы, сезонные овощи, заморозка). Разбил на 2 закупки в неделю (маленький бюджет) | Бюджетное планирование |
| **Мини-кухня в комнате** | Зелёная зона: фрукты, вода, боул, контейнер с нарезанными овощами. Однокомпонентные ужины (яичница + овощи) | Среда: доступность |
| **Социальные сети** | Отписка от фитоняш -> подписка на нутрициологов и интуитивное питание | Цифровая среда |
| **Диетолог** | Сессии 1 раз в 2 недели: без цифр, только «как самочувствие?». Психообразование о вреде гиперконтроля | Медицинское сопровождение |
| **Вес** | 1 раз в неделю, без паники | Контроль без гиперфиксации |

### Динамика

| Метрика | Неделя 0 | Неделя 4 | Неделя 12 |
|---|---|---|---|
| Вес | 63 кг | 61 кг | 58 кг |
| Подсчёт калорий | Каждый приём | 1 раз в день | 0 (отпустила) |
| Тревога о еде | 8/10 | 4/10 | 2/10 |
| Разнообразие рациона | 5 продуктов | 10 продуктов | 15+ продуктов |
| Выполнение списка покупок | 0% | 60% | 85% |

### Роль диетолога

| Время | Действие |
|---|---|
| Старт | Беседа: «никаких запретов, только добавляем полезное». Список продуктов |
| Каждые 2 недели | Созвон 30 мин: обсуждаем самочувствие, смотрим фото дневника, корректируем |
| По запросу | Чат поддержки: клиент пишет если тревога или вопрос |

## Кейс 5. «Семейная пара, 38 и 36 лет, разные психотипы» — общий контроль веса семьи

### Профиль
- **Муж (холерик):** +15 кг за 2 года, метаболический синдром, наблюдается у диетолога
- **Жена (флегматик):** +8 кг, хочет наладить питание всей семьи (двое детей 5 и 8 лет)
- **Диетолог:** Ведёт обоих, адаптирует план под разные психотипы

### Дизайн среды — контроль веса

| Область | Для мужа (холерик) | Для жены (флегматик) |
|---|---|---|
| **Дневник питания (фото)** | Фоткает тарелку -> Telegram-бот с KPI: белок/овощи/порция. Статус «зелёный/жёлтый/красный» | Фоткает тарелку -> чат с диетологом. Обратная связь: «добавь овощей, отличный белок» |
| **Планирование закупок** | Не участвует. Просто ест что готовит жена | Составляет общее меню на неделю (для всех 4). Список покупок — на магните на холодильнике. Доставка овощей — авто-подписка |
| **Среда кухни** | Челлендж «30 дней зелёной зоны» (KPI на доске) | Чёткие зоны в холодильнике под каждого. Детские перекусы — отдельная цветная полка |
| **Контроль диетолога** | Графики веса, давления, лаборатория раз в месяц. Требует чётких KPI | Еженедельный просмотр дневника. Созвон 1 раз в 2 недели |
| **Совместный ритуал** | Вечерняя прогулка (замена вечернему перекусу) | — |

### Меню на неделю (пример закупочного листа)

| День | Завтрак | Обед | Ужин | Перекус детям |
|---|---|---|---|---|
| ПН | Овсянка + ягоды | Курица + гречка + овощи | Рыба на пару + салат | Яблоко + йогурт |
| ВТ | Яйца пашот + тост | Борщ (без сметаны) + хлеб | Индейка + киноа + брокколи | Морковь + хумус |
| СР | Смули-боул | Лосось + батат + шпинат | Тофу-стир-фрай с овощами | Орехи + сухофрукты |
| ЧТ | Творог + фрукты | Говядина + гречка + салат | Овощной суп + яйцо | Фруктовое пюре |
| ПТ | Гранола + йогурт | Тунец + кускус + овощи | Куриная грудка + салат Цезарь (йогурт) | Сыр + виноград |
| СБ | Блины (ц/з мука) | Боул с нутом + авокадо | Стейк + запеч. овощи | Ягоды + сливки |
| ВС | Запеканка творожная | Уха + хлеб | Микс-салат + креветки | Банан + орех. паста |

### Динамика

| Метрика | Муж (12 нед) | Жена (12 нед) |
|---|---|---|
| Вес | -11 кг (94 -> 83 кг) | -6 кг (68 -> 62 кг) |
| Объём талии | -12 см | -7 см |
| Фото-дневник (выполнение) | 85% | 95% |
| Выполнение списка покупок | — | 100% (жена), 90% (муж не покупает лишнего) |
| Давление | 145/92 -> 125/80 | — |
| Конфликты о еде | Исчезли («было каждый день») | |

### Роль диетолога

| Время | Муж | Жена |
|---|---|---|
| Старт | План с дефицитом 600 ккал, лаборатория, KPI | План на семью (общий), список покупок |
| Еженедельно | Просмотр фото-дневника (бот), вес, давление | Просмотр фото-дневника (чат) |
| 1 раз в 2 нед | Созвон 15 мин (цифры, динамика) | Созвон 20 мин (самочувствие, дети) |

## Чек-лист «Дизайн среды для контроля веса»

### Дневник питания с фото
- [ ] Формат определён (приложение / чат / Telegram-бот)
- [ ] Клиент знает: фоткать ДО еды, а не после
- [ ] Диетолог установил частоту ревью (первые 2 нед — 1 раз в 3 дня, затем 1 раз в нед)
- [ ] Психотип-адаптация: формат обратной связи
- [ ] Нет подсчёта КБЖУ без назначения (только если врач прописал)

### Планирование закупок
- [ ] Меню на неделю составлено (фиксированный или ротационный шаблон)
- [ ] Список покупок сгруппирован по отделам магазина
- [ ] Правило «зелёной корзины»: 60% овощи/фрукты, 20% белок, 15% углеводы, 5% снеки
- [ ] Закупка 1 раз в неделю + 1 дозакупка
- [ ] Клиент идёт в магазин сытым

### Контроль веса
- [ ] Еженедельное взвешивание в одно и то же время (утро, натощак, после туалета)
- [ ] Фото в одном ракурсе 1 раз в 2 недели
- [ ] Объём талии 1 раз в месяц
- [ ] Диетолог отслеживает тренд (не день-к-дню, а скользящую среднюю)

### Сопровождение врача-диетолога
- [ ] Частота контакта согласована по психотипу
- [ ] Формат обратной связи: асинхронно / синхронно
- [ ] Лабораторный контроль по показаниям
- [ ] План Б на срыв (обсуждён заранее)
- [ ] Критерии перехода на самоконтроль определены
