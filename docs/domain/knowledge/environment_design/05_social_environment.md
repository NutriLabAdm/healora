## Роль социальной среды

Социальное окружение определяет 10% поведения (приверженность/убеждения) + 25% триггеров (социальные триггеры).

| Фактор | Доля | Социальный аспект |
|---|---|---|
| **Среда** | 30% | Доступность здоровой еды в окружении (офис, семья) |
| **Эмоции** | 30% | Социальное удовольствие от совместных действий |
| **Триггеры** | 25% | Социальные триггеры (друг, чат, группа, коуч) |
| **Приверженность** | 10% | Публичное обещание, социальный контракт |
| **Ценности** | 5% | Социальные нормы, «так принято» |

## Типы социальных сред

### 1. Ближнее окружение (семья, партнёр)

| Ситуация | Проблема | Решение |
|---|---|---|
| Супруг/а покупает вредные продукты | Соблазн каждый день | Договориться: полезная корзина для всех |
| Дети просят сладкое | Срыв из-за детских零食ов | Здоровые альтернативы (фрукты, йогурты) |
| Традиция семейных ужинов | Переедание по привычке | Маленькие тарелки, контроль порций |
| Партнёр не поддерживает диету | Отсутствие моральной поддержки | Вовлечь партнёра: совместная готовка |

### 2. Рабочее окружение (коллеги, офис)

| Ситуация | Проблема | Решение |
|---|---|---|
| Офисные печеньки / торты | Социальное давление: «бери, не стесняйся» | Свои полезные снеки, отказаться без объяснений |
| Совместные обеды в кафе | Сложно контролировать КБЖУ | Изучить меню заранее, выбрать ресторан заранее |
| Перекусы за столом | Неконтролируемое питание | Контейнер с овощами, вода на столе |

### 3. Цифровое социальное окружение

| Канал | Проблема | Решение |
|---|---|---|
| Чаты группы поддержки | Молчание -> снижение мотивации | Ежедневные отчёты, реакции, эмодзи |
| Социальные сети | Лента «идеальных» тел -> тревога | Подписка на нутрициологов, а не на фитоняшек |
| Челенджи | Социальное сравнение -> стресс | Индивидуальные цели в групповом формате |
| Публичные обещания | Страх осуждения при срыве | Закрытая группа поддержки |

### 4. Коуч / нутрициолог как социальная среда

| Функция | Как работает |
|---|---|
| Внешний триггер | Регулярные созвоны / чаты напоминают о целях |
| Социальное одобрение | Похвала за прогресс -> дофамин |
| Социальный контракт | Публичное обещание -> ответственность |
| Безопасное пространство | Без осуждения -> доверие -> честность |

## Инструменты дизайна социальной среды

### Социальные триггеры

| Триггер | Как создать | Пример |
|---|---|---|
| **Групповой отчёт** | «Сделал ✅» в чат каждый день | Каждый вечер: «Вода 2 л ✅, завтрак ✅, шаги 8000» |
| **Парное выполнение** | Напарник по привычке | Вместе пить воду, вместе выходить на прогулку |
| **Публичное обещание** | Объявить цель группе | «Я пью 2 л воды 30 дней. Спросите меня через месяц» |
| **Штраф за срыв** | Финансовая или социальная ответственность | 500 руб в общий котёл за пропуск |
| **Назначение «смотрящего»** | Кто-то проверяет выполнение | Муж/жена спрашивает: «Ты выпила воду сегодня?» |
| **Клуб по интересам** | Группа с общей целью | «Клуб воды», «Марафон 10 000 шагов» |

### Социальный контракт

```
Я, ____________________, обязуюсь:

1. Пить 2 литра воды каждый день в течение 30 дней
2. Отмечать выполнение в чате группы ежедневно до 23:00
3. Если пропускаю 2 дня подряд — плачу 500 руб в общий фонд группы

Дата: ___________  Подпись: ___________
Свидетель: _____________________________
```

### Поддерживающая среда: групповая динамика

| Размер группы | Характеристика | Удержание |
|---|---|---|
| **1 (один)** | Высокий риск схода | < 10% через 3 месяца |
| **2 (пара)** | Риск схода обоих > 90% | ~ 15% через 3 месяца |
| **3-5** | Стабильная динамика, взаимопомощь | ~ 40% через 3 месяца |
| **5-15** | Групповая поддержка + разнообразие опыта | ~ 50% через 3 месяца |
| **> 15** | Потеря индивидуального внимания | ~ 30% через 3 месяца |

## Адаптация социальной среды под психотип

| Психотип | Формат группы | Роль в группе | Частота контакта |
|---|---|---|---|
| **Сангвиник** | Свободный чат, нет обязательных отчётов | Лидер / эксперт | 1 раз в 2 недели |
| **Холерик** | Соревнование, рейтинг, челленджи | Лидер / мотиватор | 1 раз в неделю |
| **Флегматик** | Закрытая группа с чёткими правилами и чек-листами | Участник | Ежедневно чек-лист |
| **Меланхолик** | Тёплая группа поддержки, без критики | Участник / ищущий поддержку | Ежедневно похвала |

### Социальные роли в группе

| Роль | Что делает | Какой психотип лучше подходит |
|---|---|---|
| **Лидер** | Задаёт темп, публикует отчёты первым | Сангвиник, холерик |
| **Мотиватор** | Подбадривает, пишет поддерживающие комментарии | Сангвиник |
| **Эксперт** | Делится исследованиями, гайдами | Флегматик |
| **Хранитель** | Следит за выполнением контрактов, штрафами | Флегматик |
| **Вдохновитель** | Делится фото результата, историями успеха | Меланхолик |

## Социальный дизайн в Healora

### Предусмотренные сценарии

| Сценарий | Компоненты | Психотип |
|---|---|---|
| Индивидуальное сопровождение | 1:1 с нутрициологом, чат, отчёты | Все, базовый сценарий |
| Парный челлендж | 2 клиента + 1 коуч, соревнование | Холерик |
| Групповой марафон | 5-10 участников, общий чат, трекер | Все (кроме меланхолика) |
| Клуб выпускников | Закрытый чат для удержания после программы | Сангвиник |
| Семейный профиль | Вовлечение семьи в план клиента | Флегматик |

## Чек-лист «Социальный аудит» (для клиента)

- [ ] Определён круг поддерживающих людей (кто поддерживает, кто саботирует)
- [ ] Есть чат/группа с единомышленниками
- [ ] Установлен регулярный контакт с коучем (частота согласована)
- [ ] Составлен социальный контракт (письменно)
- [ ] Семья / партнёр ознакомлены с планом и поддерживают
- [ ] Офисная среда адаптирована (снеки, вода, обед)
- [ ] Социальные сети настроены на поддержку (подписки, фильтры)
