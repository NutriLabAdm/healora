## Цифровая среда: 30% фактора среды

Современный человек проводит 6-10 часов в день в цифровой среде. Смартфон, ноутбук, часы — это не нейтральные инструменты, а **спроектированная среда**, которая либо поддерживает здоровое поведение, либо разрушает его.

### Цифровые триггеры: вредные vs полезные

| Тип триггера | Вредный пример | Полезная замена |
|---|---|---|
| **Пуш-уведомления** | «Скидка 50% на пиццу!» | «Время выпить воды!» |
| **Социальные сети** | Лента с едой / ресторанами | Чат группы поддержки клиентов |
| **Приложения доставки** | 1 клик — и пицца в пути | Приложение для планирования меню |
| **Экран перед сном** | Синий свет -> мелатонин -> плохой сон | Ритуал: чай + книга, экран за 30 мин до сна |
| **Трекеры активности** | Навязчивые напоминания -> тревога | Игровые челленджи -> азарт |

## Дизайн цифровой среды: 5 принципов

### 1. Заменить вредные цифровые триггеры на полезные

| Вредный триггер | Полезный триггер |
|---|---|
| Уведомление Instagram | Напоминание выпить воду (1 раз/час) |
| Лента Facebook перед сном | Аудиокнига / подкаст перед сном |
| Приложение доставки еды на главном экране | Приложение трекера привычек на главном экране |
| Игры в телефоне | Приложение для медитации / дыхания |

### 2. Проектировать экран «первого взгляда»

Принцип: **то, что видит пользователь при разблокировке телефона, определяет его первое действие.**

| До | После |
|---|---|
| На главном экране: Instagram, YouTube, игры, доставка еды | На главном экране: трекер воды, трекер привычек, заметки питания |
| Виджеты: погода, новости | Виджеты: количество шагов, прогресс воды |
| Папки: соцсети, игры, развлечения | Папки: здоровье, активность, сон |

### 3. Использовать носимые устройства как среду

| Устройство | Триггер | Полезное действие |
|---|---|---|
| Apple Watch / Garmin | Напоминание встать | 1 мин ходьбы |
| Умные часы | Напоминание о дыхании | Дыхательная практика 1 мин |
| Фитнес-браслет | 10 000 шагов: прогресс | Дойти до цели |
| Умные весы | Еженедельное взвешивание | Фиксация тренда |

### 4. Цифровой детокс: убрать лишнее

| Действие | Эффект |
|---|---|
| Убрать приложения доставки еды | -90% спонтанных заказов |
| Отключить уведомления соцсетей на 2 часа утром | Продуктивное утро без отвлечений |
| Убрать телефон из спальни | +40 мин сна (исследование Harvard) |
| Использовать чёрно-белый экран в вечернее время | Снижение тяги к соцсетям на 30% |

### 5. Создать «цифровой ритуал» вместо «цифровой рутины»

| Время | Вредный паттерн | Полезный ритуал |
|---|---|---|
| Утро | Телефон в руки -> соцсети -> новости -> 30 мин потеряно | Вода -> зарядка -> завтрак -> телефон через 1 час |
| Работа | Фоновые пуши -> отвлечение каждые 5 мин | Режим фокуса: пуши отключены, только трекер воды |
| Вечер | Скролл ленты -> прокрастинация -> поздний сон | Экран за 30 мин до сна -> чай -> книга -> сон |

## Интеграция с Digital Twin Healora

### Атрибуты цифровой среды в Digital Twin

| Атрибут | Тип | Измерение |
|---|---|---|
| Время экрана (screen time) | часов/день | Apple Screen Time / Android Digital Wellbeing |
| Основные приложения | список топ-5 | Self-report / скриншот |
| Носимые устройства | да/нет + тип | Self-report |
| Цифровые триггеры на главном экране | категории | Фото экрана -> AI анализ |
| Время последнего использования экрана перед сном | минут | Self-report |
| Количество пущей в день | число | Self-report |

### Рекомендации по психотипу

| Психотип | Цифровой дизайн |
|---|---|
| **Сангвиник** | Минимум пущей, уведомления — раз в день. Свобода |
| **Холерик** | Чёткие цифровые KPI, графики, рейтинги. Геймификация |
| **Флегматик** | Максимум автоматизации: трекеры, напоминания, готовые шаблоны |
| **Меланхолик** | Мягкие пуши (эмодзи, тёплые слова). Без давления. Фото-дневник |

## Чек-лист «Цифровой аудит» (для клиента)

- [ ] Экран «первого взгляда»: здоровые приложения на главном
- [ ] Уведомления соцсетей выключены (или отложены)
- [ ] Приложения доставки еды удалены / в папке
- [ ] Трекер воды / привычек установлен и настроен
- [ ] Носимые устройства синхронизированы
- [ ] Режим «Не беспокоить» настроен на время сна
- [ ] Электронные книги / аудиокниги под рукой (альтернатива скроллу)
- [ ] За 30 мин до сна — экран выключен
