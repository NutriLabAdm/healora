# Конвенция именования файлов

- **Профиль персонажа** (без истории протокола): `{PERS_MD}.json`
  - Пример: `05_Дмитрий_57_notepad.json`

- **План терапии / история выполнения протокола**: `{PERS_MD}_{PROTOCOL_ID}.json`
  - Пример: `05_Дмитрий_57_notepad_PTL_RAPID_WL.json`

Где `{PERS_MD}` — идентификатор персоны из `notepad`-файла,
`{PROTOCOL_ID}` — мнемонический ID протокола (PTL_xxx).
