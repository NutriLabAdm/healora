# ИМТ (Индекс массы тела)

**Раздел:** 01_Demographics

## Описание

Индекс массы тела = вес(кг) / рост(м)². Косвенный показатель состава тела. ИМТ > 25 — избыточная масса тела, > 30 — ожирение. Не учитывает соотношение мышечной и жировой ткани.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | кг/м² |
| Целевое значение | 22 |
| Норма | 18.5–25 |
| Оптимальный диапазон | 20–23 |
| Допустимый диапазон | 14–45 |

## Причины отклонения от нормы

- Те же причины, что и для избыточного веса
- Низкая мышечная масса при нормальном весе может маскировать ожирение (саркопеническое ожирение)

## Способы коррекции

- Коррекция веса (см. атрибут Вес)
- Силовые тренировки для увеличения мышечной массы (PH_STR)
- Сбалансированное питание с достаточным потреблением белка (FD_CAL, FD_FIB)
