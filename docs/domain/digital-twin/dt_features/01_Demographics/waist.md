# Талия (окружность талии)

**Раздел:** 01_Demographics

## Описание

Окружность талии — маркер абдоминального (висцерального) ожирения. Избыток висцерального жира — независимый фактор риска метаболического синдрома, ССЗ, диабета 2 типа.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | см |
| Целевое значение | 80 |
| Норма | 60–100 |
| Оптимальный диапазон | 70–90 |
| Допустимый диапазон | 40–150 |

## Причины отклонения от нормы

- Избыточное потребление калорий при низкой физической активности
- Хронический стресс → повышение кортизола → абдоминальное ожирение
- Инсулинорезистентность и гиперинсулинемия
- Недостаток сна → нарушение лептин-грелиновой регуляции
- Злоупотребление алкоголем

## Способы коррекции

- Снижение калорийности рациона (FD_CAL)
- Аэробные нагрузки средней интенсивности (PH_AER, PH_Z2)
- Управление стрессом (MN_MDT, MN_BRTH)
- Нормализация сна (SL_BED, SL_DUR)
- Отказ от алкоголя (FD_SUG)
