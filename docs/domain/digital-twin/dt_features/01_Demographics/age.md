# Возраст

**Раздел:** 01_Demographics

## Описание

Хронологический возраст пациента — фундаментальный фактор, влияющий на все системы организма. Возраст определяет референсные интервалы для большинства биомаркеров и риски возраст-ассоциированных заболеваний.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | лет |
| Целевое значение | — |
| Норма | 18–65 |
| Оптимальный диапазон | 25–55 |
| Допустимый диапазон | 0–120 |

## Причины отклонения от нормы

- Естественный биологический процесс, не является патологией
- Ускоренное биологическое старение при хроническом стрессе, недосыпе, токсической нагрузке
- Преждевременное старение на фоне метаболического синдрома, инсулинорезистентности, хронического воспаления

## Способы коррекции

- Регулярная физическая активность (PH_HIIT, PH_AER, PH_Z2) замедляет биологическое старение
- Оптимизация сна (SL_BED, SL_DUR, SL_QLT, SL_HYG) — 7–9 часов
- Управление стрессом (MN_MDT, MN_BRTH, MN_STR)
- Противовоспалительное питание (FD_ANTI, FD_FIB, FD_OMEGA)
- Нутрицевтическая поддержка (SP_MG, SP_ADP, M_NEU01)
