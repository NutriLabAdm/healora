# Вес

**Раздел:** 01_Demographics

## Описание

Масса тела — интегральный показатель энергетического баланса. Избыток веса коррелирует с риском ССЗ, диабета 2 типа, гипертонии. Дефицит указывает на недостаточность питания или хронические заболевания.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | кг |
| Целевое значение | 70 |
| Норма | 50–120 |
| Оптимальный диапазон | 60–85 |
| Допустимый диапазон | 30–200 |

## Причины отклонения от нормы

- Избыток калорий (> потребления) — основная причина набора веса
- Малоподвижный образ жизни (sedentary)
- Гормональные нарушения: гипотиреоз, инсулинорезистентность, гиперкортицизм
- Лекарственные препараты (антидепрессанты, кортикостероиды)
- Нарушение пищевого поведения (РПП)
- Генетическая предрасположенность (FTO-ген)

## Способы коррекции

- Дефицит калорий 300–500 ккал/день (FD_CAL)
- Снижение потребления сахара и рафинированных углеводов (FD_CRB, FD_SUG)
- Интервальное голодание (FD_IF)
- Высокоинтенсивные тренировки (PH_HIIT)
- Силовые тренировки (PH_STR)
- Контроль водного баланса (SP_MG, FD_EAT)
