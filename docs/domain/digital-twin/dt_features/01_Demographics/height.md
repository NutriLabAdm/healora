# Рост

**Раздел:** 01_Demographics

## Описание

Длина тела стоя. Используется для расчёта ИМТ, оценки нутритивного статуса и дозировки препаратов. Генетически детерминирован, незначительно снижается с возрастом.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | см |
| Целевое значение | — |
| Норма | 150–200 |
| Оптимальный диапазон | 160–190 |
| Допустимый диапазон | 100–250 |

## Причины отклонения от нормы

- Генетическая предрасположенность
- Возрастная дегенерация межпозвонковых дисков (уменьшение на 1–3 см после 50 лет)
- Остеопороз, компрессионные переломы позвонков

## Способы коррекции

- Коррекция осанки (PH_FLEX, PH_STR)
- Профилактика остеопороза: витамин D + K2, кальций (SP_MG, M_END02)
- Гормональная коррекция при дефиците гормона роста (крайне редко)
