# Питание

**Раздел:** 04_Lifestyle

## Описание

Сбалансированное питание — основа профилактики хронических заболеваний и поддержания веса.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | — |
| Целевое значение | — |
| Норма | Сбалансир. |
| Оптимальный диапазон | — |
| Допустимый диапазон | — |

## Причины отклонения от нормы

- Нерегулярное питание
- Ультра-обработанные продукты
- Недостаток овощей
- Избыток сахара, соли, жиров

## Способы коррекции

- Сбалансированное питание
- Увеличение овощей (FD_FIB)
- Ограничение сахара (FD_SUG, FD_CRB)
- Режим питания (FD_EAT)
- Средиземноморская диета (FD_OMEGA)
