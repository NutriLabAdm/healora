# Вода (потребление)

**Раздел:** 04_Lifestyle

## Описание

Суточное потребление чистой воды. Необходима для когнитивных функций, обмена веществ, детоксикации.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | л/день |
| Целевое значение | 2.5 |
| Норма | 2–3 |
| Оптимальный диапазон | 2–2.5 |
| Допустимый диапазон | 0–5 |

## Причины отклонения от нормы

- Отсутствие привычки
- Предпочтение сладких напитков и кофе
- Занятость
- Страх отёков

## Способы коррекции

- Напоминания о питье
- Бутылка воды на столе
- Добавление лимона/огурца для вкуса
- Вода перед каждым приёмом пищи
