# Стресс (уровень 0–10)

**Раздел:** 04_Lifestyle

## Описание

Хронический стресс (≥7/10) повышает риск ССЗ, депрессии, нарушает сон, способствует абдоминальному ожирению.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | /10 |
| Целевое значение | 3 |
| Норма | <5 |
| Оптимальный диапазон | 0–3 |
| Допустимый диапазон | 0–10 |

## Причины отклонения от нормы

- Высокая рабочая нагрузка
- Финансовые трудности
- Семейные конфликты
- Хронические заболевания
- Недостаток сна
- Нездоровое питание (дисбиоз)
- Информационная перегрузка

## Способы коррекции

- Медитация осознанности (MN_MDT)
- Дыхательные практики (MN_BRTH)
- Физ. активность (PH_HIIT, PH_AER, PH_STR)
- Магний и адаптогены (SP_MG, SP_ADP)
- Нейротерапия (M_NEU01)
- Образовательные программы (ED_MA, ED_BODY, ED_EMO, ED_SUP)
