# РПП (Расстройство пищевого поведения)

**Раздел:** 04_Lifestyle

## Описание

Анорексия, булимия, компульсивное переедание, орторексия. Требует психотерапии.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | — |
| Целевое значение | Нет |
| Норма | Нет |
| Оптимальный диапазон | — |
| Допустимый диапазон | — |

## Причины отклонения от нормы

- Искажённое восприятие тела
- Перфекционизм
- Социальное давление
- Психотравма
- Генетика

## Способы коррекции

- Психотерапия (КПТ, DBT)
- Нутритивная поддержка
- Медикаментозная терапия
