# Сон (продолжительность)

**Раздел:** 04_Lifestyle

## Описание

Средняя продолжительность ночного сна. <7 ч — риск ожирения, диабета, ССЗ, депрессии.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | ч |
| Целевое значение | 7.5 |
| Норма | 7–9 |
| Оптимальный диапазон | 7.5–8.5 |
| Допустимый диапазон | 3–14 |

## Причины отклонения от нормы

- Высокая занятость
- Хронический стресс
- Нарушение гигиены сна (экранное время, кофеин)
- СОАС
- Болевой синдром
- Депрессия
- Гормональные нарушения

## Способы коррекции

- Фиксированное время отхода ко сну (SL_BED)
- Оптимальная продолжительность 7–9 ч (SL_DUR)
- Гигиена сна: темнота, тишина, прохлада (SL_HYG)
- Медитация (MN_MDT)
- Цифровой детокс (MN_DTOX)
- Ограничение кофеина (FD_CAF)
- Магний (SP_MG)
- Нейротерапия (M_NEU01)
