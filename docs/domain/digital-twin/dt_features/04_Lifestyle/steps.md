# Шаги (ежедневная активность)

**Раздел:** 04_Lifestyle

## Описание

<5000 — сидячий образ жизни. 8000–12000 — активный. Рекомендация ВОЗ: 8000–10000 шагов/день.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | /день |
| Целевое значение | 10000 |
| Норма | 8000–12000 |
| Оптимальный диапазон | 10000–15000 |
| Допустимый диапазон | 0–30000 |

## Причины отклонения от нормы

- Сидячая работа
- Отсутствие привычки к ходьбе
- Транспортная зависимость
- Хроническая усталость
- Физические ограничения

## Способы коррекции

- Постепенное увеличение на 1000 шагов/нед
- Использование трекера
- Прогулки в обед
- Замена транспорта на ходьбу
