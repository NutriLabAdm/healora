# Курение

**Раздел:** 04_Lifestyle

## Описание

Один из самых значимых модифицируемых факторов риска. Повышает риск рака лёгких, ССЗ, ХОБЛ.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | — |
| Целевое значение | — |
| Норма | Нет |
| Оптимальный диапазон | — |
| Допустимый диапазон | — |

## Причины отклонения от нормы

- Никотиновая зависимость
- Психологическая привычка
- Социальное давление, стресс

## Способы коррекции

- Никотинзаместительная терапия
- Поведенческая терапия
- Медикаментозная поддержка
