# Тип активности

**Раздел:** 04_Lifestyle

## Описание

Разнообразие типов нагрузки (аэробная, силовая, гибкость) обеспечивает комплексный эффект.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | — |
| Целевое значение | — |
| Норма | — |
| Оптимальный диапазон | — |
| Допустимый диапазон | — |

## Причины отклонения от нормы

- Ограниченный выбор
- Монотонный режим

## Способы коррекции

- Комбинация аэробных (PH_AER), силовых (PH_STR) и гибкостных (PH_FLEX) нагрузок
