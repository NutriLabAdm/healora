# Ферритин

**Раздел:** 03_Labs

## Описание

Белок, депонирующий железо. Низкий — железодефицит. Высокий — перегрузка железом или воспаление.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | нг/мл |
| Целевое значение | 80 |
| Норма | 20–200 |
| Оптимальный диапазон | 50–150 |
| Допустимый диапазон | 5–500 |

## Причины отклонения от нормы

- Дефицит: кровопотеря, недостаток железа в рационе, мальабсорбция
- Избыток: гемохроматоз, гемотрансфузии
- Повышение: острофазовый ответ (воспаление)

## Способы коррекции

- При дефиците: препараты железа (M_LAB01)
- Диагностика причины (DG_CHK)
- Гастроэнтерология (M_GAS02, M_GAS03)
- При избытке: донация крови, хелаторы
- Мониторинг (M_LAB01)
