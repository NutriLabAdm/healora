# ЛПВП (Липопротеины высокой плотности)

**Раздел:** 03_Labs

## Описание

«Хороший» холестерин. Высокий ЛПВП (>60 мг/дл) снижает риск ССЗ.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | мг/дл |
| Целевое значение | 60 |
| Норма | >40 |
| Оптимальный диапазон | 50–70 |
| Допустимый диапазон | 20–100 |

## Причины отклонения от нормы

- Малоподвижность
- Трансжиры, рафинированные углеводы
- Курение
- Ожирение
- Метаболический синдром

## Способы коррекции

- Аэробные нагрузки (PH_AER, PH_Z2)
- Мононенасыщенные жиры (оливковое масло, орехи)
- Омега-3 (FD_OMEGA)
- Отказ от курения
- Снижение веса
