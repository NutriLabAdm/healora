# Глюкоза крови натощак

**Раздел:** 03_Labs

## Описание

Концентрация глюкозы в плазме крови натощак. >100 мг/дл — преддиабет, >126 мг/дл — сахарный диабет.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | мг/дл |
| Целевое значение | 100 |
| Норма | 70–100 |
| Оптимальный диапазон | 75–95 |
| Допустимый диапазон | 40–300 |

## Причины отклонения от нормы

- Избыточное потребление простых углеводов (FD_SUG, FD_CRB)
- Инсулинорезистентность
- Ожирение
- Малоподвижный образ жизни
- Хронический стресс
- Генетическая предрасположенность

## Способы коррекции

- Ограничение сахара (FD_SUG, FD_CRB)
- Интервальное голодание (FD_IF)
- Клетчатка (FD_FIB)
- Физ. активность (PH_AER, PH_HIIT)
- Контроль калорий (FD_CAL)
- Эндокринология (M_END01, M_END02)
- Лабораторный мониторинг (M_LAB01, M_LAB02)
- Обучение (ED_SCH)
