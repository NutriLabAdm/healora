# Триглицериды

**Раздел:** 03_Labs

## Описание

Основной транспортный липид. Высокий уровень (>150) — риск панкреатита и ССЗ.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | мг/дл |
| Целевое значение | 150 |
| Норма | <150 |
| Оптимальный диапазон | 70–130 |
| Допустимый диапазон | 30–500 |

## Причины отклонения от нормы

- Избыток углеводов и сахара
- Алкоголь
- Ожирение
- Инсулинорезистентность
- Гипотиреоз
- ХБП

## Способы коррекции

- Ограничение сахара (FD_SUG, FD_CRB)
- Омега-3 (FD_OMEGA)
- Аэробные нагрузки (PH_AER)
- Снижение веса (FD_CAL)
- Ограничение алкоголя
