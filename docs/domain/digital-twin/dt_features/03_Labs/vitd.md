# Витамин D (25-гидроксикальциферол)

**Раздел:** 03_Labs

## Описание

Критически важен для иммунитета, костной ткани, нервной системы. Дефицит <30 нг/мл.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | нг/мл |
| Целевое значение | 40 |
| Норма | 30–100 |
| Оптимальный диапазон | 40–80 |
| Допустимый диапазон | 5–150 |

## Причины отклонения от нормы

- Недостаточная инсоляция (северные широты)
- Малоподвижность (редкое пребывание на солнце)
- Ожирение (секвестрация в жировой ткани)
- Мальабсорбция (целиакия, ВЗК)
- Возраст >50 лет
- Тёмная кожа

## Способы коррекции

- Приём витамина D3 2000–5000 МЕ/сут (SP_MG)
- Пребывание на солнце 15–20 мин/день
- Жирная рыба, яичный желток
- Контроль магния
