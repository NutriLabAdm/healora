# ТТГ (Тиреотропный гормон)

**Раздел:** 03_Labs

## Описание

Регулятор функции щитовидной железы. Повышен — гипотиреоз, снижен — гипертиреоз. Оптимум <2.5.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | мМЕ/л |
| Целевое значение | 2.5 |
| Норма | 0.4–4.0 |
| Оптимальный диапазон | 0.5–2.5 |
| Допустимый диапазон | 0.1–10.0 |

## Причины отклонения от нормы

- Гипотиреоз: тиреоидит Хашимото, йододефицит
- Гипертиреоз: болезнь Грейвса, узловой зоб
- Стресс, недостаток сна
- Лекарства: литий, амиодарон

## Способы коррекции

- Заместительная терапия левотироксином (M_END01, M_END03)
- Тиреостатики при гипертиреозе (M_END01)
- Нормализация селена, цинка, йода
- Управление стрессом (MN_MDT)
- Мониторинг (M_LAB02)
