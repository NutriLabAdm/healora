# ЛПНП (Липопротеины низкой плотности)

**Раздел:** 03_Labs

## Описание

«Плохой» холестерин — основная атерогенная фракция. Цель <100 мг/дл.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | мг/дл |
| Целевое значение | 100 |
| Норма | <100 |
| Оптимальный диапазон | 70–100 |
| Допустимый диапазон | 30–250 |

## Причины отклонения от нормы

- Насыщенные жиры и трансжиры
- Генетика (семейная гиперхолестеринемия)
- Гипотиреоз
- Заболевания печени

## Способы коррекции

- Диета с низким содержанием насыщенных жиров (FD_ANTI)
- Растворимая клетчатка (FD_FIB)
- Фитостерины (FD_OMEGA)
- Статины (M_CAR02)
- Мониторинг (M_LAB01)
