# Холестерин общий

**Раздел:** 03_Labs

## Описание

Суммарный холестерин всех фракций. Высокий уровень — фактор риска атеросклероза.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | мг/дл |
| Целевое значение | 200 |
| Норма | <200 |
| Оптимальный диапазон | 150–180 |
| Допустимый диапазон | 100–350 |

## Причины отклонения от нормы

- Избыток насыщенных жиров и трансжиров
- Ожирение
- Малоподвижность
- Генетика (семейная гиперхолестеринемия)
- Гипотиреоз

## Способы коррекции

- Диета с низким содержанием насыщенных жиров (FD_ANTI, FD_OMEGA)
- Аэробные нагрузки (PH_AER)
- Клетчатка (FD_FIB)
- Статинотерапия
