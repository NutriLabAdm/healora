# CRP (С-реактивный белок)

**Раздел:** 03_Labs

## Описание

Маркер системного воспаления. hs-CRP >2 мг/л — повышенный риск ССЗ.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | мг/л |
| Целевое значение | 1 |
| Норма | <3 |
| Оптимальный диапазон | 0–1 |
| Допустимый диапазон | 0–20 |

## Причины отклонения от нормы

- Хронические воспалительные заболевания
- Ожирение (жировая ткань → IL-6 → CRP)
- Курение
- Стресс
- Нарушение сна
- Инфекции
- Метаболический синдром

## Способы коррекции

- Противовоспалительная диета (FD_ANTI)
- Омега-3 (FD_OMEGA)
- Физ. активность (PH_AER, PH_HIIT)
- Снижение веса
- Отказ от курения
- Управление стрессом (MN_MDT, MN_BRTH)
- Гастроэнтерология (M_GAS01)
- Мониторинг (M_LAB01)
