# HbA1c (Гликированный гемоглобин)

**Раздел:** 03_Labs

## Описание

Средний уровень глюкозы за 2-3 месяца. >5.6% — преддиабет, >6.5% — диабет.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | % |
| Целевое значение | 5.5 |
| Норма | 4.0–5.6 |
| Оптимальный диапазон | 4.5–5.4 |
| Допустимый диапазон | 3.0–10.0 |

## Причины отклонения от нормы

- Хроническая гипергликемия
- Анемия (ложное повышение)
- ХБП
- Беременность

## Способы коррекции

- См. коррекцию глюкозы
- Эндокринологическая терапия (M_END02)
- Мониторинг (M_LAB01)
