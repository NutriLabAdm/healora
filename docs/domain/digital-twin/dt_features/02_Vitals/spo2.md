# SpO₂ (Сатурация кислорода)

**Раздел:** 02_Vitals

## Описание

Насыщение гемоглобина кислородом. Норма >95%. Снижение (<94%) — гипоксемия.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | % |
| Целевое значение | 98 |
| Норма | 95–100 |
| Оптимальный диапазон | 97–100 |
| Допустимый диапазон | 85–100 |

## Причины отклонения от нормы

- ХОБЛ, астма
- Анемия
- Сердечная недостаточность
- Ожирение (гиповентиляция)
- СОАС
- Курение

## Способы коррекции

- Дыхательная гимнастика (MN_BRTH)
- Аэробные тренировки (PH_AER)
- Коррекция анемии (M_LAB01)
- Терапия СОАС (CPAP)
- Отказ от курения
- Снижение веса
