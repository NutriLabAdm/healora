# ЧСС покоя

**Раздел:** 02_Vitals

## Описание

Пульс в состоянии покоя — маркер кардиореспираторной выносливости и вегетативной регуляции. Высокая ЧСС (>80) ассоциирована с повышенным риском ССЗ.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | уд/мин |
| Целевое значение | 72 |
| Норма | 60–80 |
| Оптимальный диапазон | 55–70 |
| Допустимый диапазон | 40–120 |

## Причины отклонения от нормы

- Детренированность, низкая аэробная выносливость
- Хронический стресс, тревожность
- Анемия, гипертиреоз, дегидратация
- Избыточное потребление кофеина (FD_CAF)
- Нарушение сна
- Избыточная масса тела

## Способы коррекции

- Аэробные тренировки 3–5 раз/нед (PH_AER, PH_Z2, PH_HIIT)
- Медитация и дыхательные практики (MN_MDT, MN_BRTH)
- Нормализация сна (SL_BED, SL_DUR, SL_QLT)
- Ограничение кофеина (FD_CAF)
- Тренировка HRV (PH_HRV)
- Кардиопротекторная терапия (M_CAR03)
