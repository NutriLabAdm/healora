# HRV (Вариабельность сердечного ритма)

**Раздел:** 02_Vitals

## Описание

Вариабельность сердечного ритма — показатель баланса симпатической и парасимпатической нервной системы. Высокая HRV = хорошая адаптивность.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | мс |
| Целевое значение | 60 |
| Норма | 40–100 |
| Оптимальный диапазон | 50–80 |
| Допустимый диапазон | 10–200 |

## Причины отклонения от нормы

- Хронический стресс, выгорание
- Нарушение сна
- Малоподвижный образ жизни
- Метаболический синдром
- Злоупотребление алкоголем, курение
- Хроническое воспаление

## Способы коррекции

- Медитация осознанности (MN_MDT)
- Дыхательные практики (MN_BRTH)
- Тренировка HRV (PH_HRV)
- Аэробные нагрузки (PH_AER)
- Кардиопротекторная терапия (M_CAR03)
- Нормализация сна
