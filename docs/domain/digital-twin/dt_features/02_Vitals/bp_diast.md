# АД диастолическое

**Раздел:** 02_Vitals

## Описание

Диастолическое артериальное давление — давление в артериях в момент расслабления сердца. Повышение ≥80 мм рт.ст. указывает на периферическую вазоконстрикцию.

## Целевые значения

| Параметр | Значение |
|----------|--------|
| Единица измерения | мм рт.ст. |
| Целевое значение | 80 |
| Норма | 60–85 |
| Оптимальный диапазон | 70–80 |
| Допустимый диапазон | 40–130 |

## Причины отклонения от нормы

- Те же причины, что и для систолического АД
- Эндотелиальная дисфункция
- Симпатикотония

## Способы коррекции

- См. коррекцию для систолического АД
- Газовые процедуры (M_GAS01)
- Лабораторный мониторинг (M_LAB01)
