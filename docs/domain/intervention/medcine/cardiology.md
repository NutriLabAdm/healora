# Кардиологические интервенции

## Показания (красные флаги)
- АД >140/90 — артериальная гипертензия
- Холестерин ЛПНП >3.0 ммоль/л — гиперлипидемия
- ЧСС покоя >100 уд/мин — тахикардия
- Вариабельность ЧСС <20 мс — вегетативная дисфункция

## Интервенции

### MED_CARD_01 — Назначение гипотензивной терапии
- **Тип:** medication
- **Воздействие:** 9.5
- **Доказательность:** A
- **Периодичность:** Ежедневно
- **Описание:** Старт антигипертензивной терапии (ингибиторы АПФ/сартаны, тиазидные диуретики, антагонисты Ca) при стойком повышении АД >140/90 мм рт.ст.

### MED_CARD_02 — Назначение статинов
- **Тип:** medication
- **Воздействие:** 9.0
- **Доказательность:** A
- **Периодичность:** Ежедневно
- **Описание:** Старт терапии статинами (аторвастатин/розувастатин) при ЛПНП >3.0 ммоль/л и/или высоком сердечно-сосудистом риске (SCORE >5%)

### MED_CARD_03 — Холтеровское мониторирование ЭКГ
- **Тип:** diagnostic
- **Воздействие:** 7.5
- **Доказательность:** B
- **Периодичность:** По показаниям
- **Описание:** Суточное мониторирование ЭКГ при тахикардии, аритмии, синкопальных состояниях

### MED_CARD_04 — Назначение бета-блокаторов
- **Тип:** medication
- **Воздействие:** 8.0
- **Доказательность:** A
- **Периодичность:** Ежедневно
- **Описание:** Бета-блокаторы (бисопролол/метопролол) при тахикардии (>100 уд/мин), ИБС, ХСН
