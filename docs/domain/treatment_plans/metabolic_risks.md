# Шаблон: Метаболические риски

- **Протокол**: 2.1
- **Категория**: medical
- **Назначение**: Коррекция метаболических нарушений

## Заключение

Пациенту рекомендовано медицинское наблюдение и коррекция метаболических
нарушений. Основные направления: контроль липидного профиля, мониторинг
артериального давления, коррекция углеводного обмена.

## Интервенции

| Код | Название | Периодичность |
|-----|----------|---------------|
| BIO_BP | Измерение АД | D |
| BIO_GLU | Глюкоза крови | D |
| FD_WTR | Водный режим | D |
| SP_02 | Витамин D3 | D |
| M_DD02 | Метформин (при ИР) | D |

## Ключевые назначения

- **АД**: контроль утром/вечером, цель <130/80
- **Глюкоза**: натощак <5.6 ммоль/л, через 2ч <7.8
- **Липиды**: ЛПНП <2.6 ммоль/л, триглицериды <1.7
- **Терапия**: по назначению лечащего врача
