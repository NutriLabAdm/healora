# Кверцетин

**Категория:** Биофлавоноид (Витаминоподобное вещество)  
**Тип:** Водорастворимое (агликон флавонолов)  
**Эндогенный синтез:** Нет (только из пищи / из рутина через гидролиз)  
**Ранее известен как:** Составная часть витамина P  

## Что это такое

Кверцетин (3,3′,4′,5,7-пентагидроксифлавон) — наиболее распространённый в природе флавоноид, агликон многих флавонол-гликозидов (рутин, изокверцитрин, кверцитрин). Относится к подклассу флавонолов. Один из наиболее изученных флавоноидов (>20,000 публикаций в PubMed). Активно исследуется как сенолитик (вещество, удаляющее стареющие клетки).

**Формы:**
- **Кверцетин (агликон)** — свободная форма (липофильнее, чем гликозиды)
- **Кверцетин-3-О-глюкозид (изокверцитрин)** — форма с лучшей биодоступностью
- **Рутин (кверцетин-3-О-рутинозид)** — гликозид, основной резерв кверцетина
- **Кверцетин-дигидрат** — стандартная форма в БАД
- **Кверцетин-липосомальный / фитосомный** — повышенная биодоступность (в 2–5 раз)

## Функции в организме

- **Антиоксидант:** нейтрализация АФК (супероксид, гидроксил-радикал, пероксинитрит)
- **Противовоспалительное:** ингибирование NF-κB, ЦОГ-2, 5-ЛОГ, iNOS, подавление выброса IL-6 и TNF-α
- **Сенолитическое (против Age-Related):** ингибиция BCL-2 и PI3K/Akt в сенесцентных клетках → апоптоз «старых» клеток
- **Антигистаминное:** стабилизация тучных клеток (ингибирование IgE-опосредованной дегрануляции) — «природный антигистамин»
- **Кардиопротекция:** снижение окисленного LDL, улучшение эндотелиальной функции (↑ NO)
- **Антипролиферативное:** ингибирование циклин-зависимых киназ (CDK), апоптоз раковых клеток (in vitro)
- **Иммуномодуляция:** активация Nrf2 и фагоцитоза макрофагов
- **Нейропротекция:** снижение β-амилоида, ингибирование ацетилхолинэстеразы

## Признаки дефицита

⚠️ Как и для других флавоноидов, термин «дефицит» не применяётся. Низкое потребление (<50 мг/сут, как в рационе без овощей/фруктов/чая) может снижать антиоксидантную защиту.

## Признаки избытка

⚠️ Относительно безопасен:
- Головная боль (редко, при >1000 мг/сут)
- Дискомфорт в ЖКТ, метеоризм
- Нефротоксичность (теоретически, при длительном >2 г/сут — оксалатные камни)
- Контактный дерматит (при местном применении, редко)
- Гипотиреоз (высокие дозы ингибируют тиреоидную пероксидазу — риск для лиц с субклиническим гипотиреозом)

## Применение (терапевтическое)

- **Сенная лихорадка (аллергический ринит):** кверцетин 400–500 мг/сут — снижение чихания, зуда, ринореи (Hirsch et al., 2020)
- **Воспалительные заболевания:** хронический простатит, артрит (кверцетин 500 мг × 2 р/сут, уровень B)
- **Атеросклероз (вторичная профилактика):** кардиопротекция (кверцетин 150–500 мг/сут)
- **Спортивная медицина:** снижение воспаления после интенсивных нагрузок (кверцетин 500–1000 мг/сут)
- **Сенолитическая терапия:** кверцетин + дазатиниб (D+Q протокол для идиопатического фиброза лёгких — фаза II РКИ)
- **Гиперурикемия (подагра):** ингибирование ксантиноксидазы — снижение мочевой кислоты (кверцетин 500 мг/сут, скромно)
- **Профилактика рака:** эпидемиологические данные — потребление >50 мг/сут снижает риск рака лёгкого на 35% (J Natl Cancer Inst)
- **Гипертония:** мягкое снижение АД (кверцетин 150–500 мг/сут, –5/–3 мм рт. ст. в мета-анализе)

## Биодоступность

**Усвоение:**
- Кверцетин (агликон): 5–15% (низкая, ограниченная растворимостью)
- Кверцетин-3-О-глюкозид: 15–25% (лучше, транспортер SGLT1)
- Рутин (рутинозид): 5–10% (требует гидролиза микрофлорой)
- Липосомальный / фитосомный кверцетин: 30–50% (в 2–5× выше агликона)
- Кверцетин-глюкуронид (после метаболизма): активен в тканях (деглюкуронизация in situ)

**Энхансеры:**
- Жиры (органические кислоты + жиры → эмульгация)
- Бромелайн (ананаса) — совместный приём повышает абсорбцию кверцетина на 30–50% (данные in vitro)
- Витамин C (рециклизация окисленного кверцетина)
- Пиперин (чёрный перец) — повышает биодоступность на 60% (ингибирование глюкуронизации)
- Ферментация (квашеные продукты, гидролиз гликозидов)

**Ингибиторы:**
- Клетчатка (связывание флавоноидов)
- Молоко (казеин связывает флавоноиды — чай с молоком теряет 30–80% усвояемости)
- Кипячение (разрушение гликозидов, потери 30–50%)

## Взаимодействие

### Протагонисты (синергисты)

| Нутриент | Эффект |
|----------|--------|
| Витамин C | Взаимная антиоксидантная защита; C регенерирует кверцетин |
| Бромелайн | Повышает абсорбцию кверцетина на 30–50% |
| Пиперин | Повышает биодоступность на 60% (ингибирование УДФ-глюкуронилтрансферазы) |
| Рутозид (рутин) | Предшественник кверцетина (гидролиз в кишечнике) |
| Ресвератрол | Сильный синергизм в активации Nrf2/ARE пути (антиоксидантная защита ×2.5) |
| Зелёный чай (EGCG) | Кверцетин + EGCG потенцируют апоптоз раковых клеток (in vitro) |

| Нутриент/препарат | Эффект |
|-------------------|--------|

## Особые группы риска дефицита

- **Курящие:** высокий окислительный стресс (повышенный расход флавоноидов)
- **Лица с хроническим воспалением** (артрит, простатит, цистит) — повышенная потребность
- **Аллергики:** за stabilisering тучных клеток (повышенная потребность при поллинозе)
- **Атлеты:** повышенный окислительный стресс после интенсивных тренировок
- **Лица с метаболическим синдромом:** сниженный антиоксидантный статус

## Научные факты

- **Сенолитический протокол D+Q (2019, Nature Medicine):** Кверцетин 100 мг + дазатиниб 100 мг 3 дня/нед — снижение стареющих клеток на 40% в эпителии; фаза II — улучшение функции лёгких при идиопатическом фиброзе.
- **Мета-анализ (2020, Nutrients):** Кверцетин 500 мг/сут в течение 8 нед — снижение СРБ (на 30%), ИЛ-6 (на 35%), ФНО-α (на 25%) у лиц с метаболическим синдромом.
- **No-effect для гипотиреоза (2021, Thyroid):** У лиц с нормальной функцией щитовидной железы кверцетин до 500 мг/сут не влияет на T4, T3 и ТТГ. При субклиническом гипотиреозе — не рекомендуется >500 мг/сут.
- **Профилактика рака (2018, мета-анализ):** Потребление >50 мг кверцетина/сут связано с RR=0.65 для рака лёгкого, RR=0.72 для рака поджелудочной (данные: EPIC, NHS).
- **Спортивное восстановление (2019, Int J Sport Nutr):** Кверцетин 1000 мг/сут снижает маркеры воспаления (креатинкиназу, IL-6) на 40–50% после эксцентрических нагрузок.

## Лайфхак для пользователя (Nutrition Tip)

> 🧅 **Аллергия или частые воспаления? Добавь красный лук в рацион!** Красный лук (100 г = 45 мг кверцетина) — самый богатый обычный продукт, особенно наружные слои. Ешь сырым в салатах (варка теряет 30–50%). Для терапевтического эффекта (аллергия, простатит) — кверцетин в БАД 500–1000 мг/сут + бромелайн (ананас) + пиперин (чёрный перец). Этот набор повышает биодоступность в 3–4 раза по сравнению с сухим порошком.
>
> ⚠️ **Важно:** Если принимаешь варфарин или имеешь гипотиреоз — не превышай 500 мг/сут без врача!
