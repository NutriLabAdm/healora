# AI Assistant Chat — System Prompt

## Назначение

Формирует system prompt для GigaChat/OpenAI при каждом запросе в чате PhoneSimulator. Промпт динамический — собирается из данных цифрового двойника и текущего плана терапии.

## Структура промпта

### 1. Информация о пользователе (из digital twin)
```
- Имя
- Возраст (лет)
- Пол (Мужской/Женский)
- Рост (см)
- Вес (кг)
- ИМТ
- Давление сист./диаст. (мм рт.ст.)
- Пульс (уд/мин)
```

### 2. Лабораторные показатели
```
- Глюкоза (мг/дл)
- HbA1c (%)
- Холестерин общий (мг/дл)
- HDL (мг/дл)
- LDL (мг/дл)
- Триглицериды (мг/дл)
- Витамин D (нг/мл)
- Железо (мкг/дл)
- Ферритин (нг/мл)
```

### 3. Образ жизни
```
- Сон (ч/сут)
- Стресс (/10)
- Шаги (шагов/день)
- Вода (мл/день)
- Курение
- Алкоголь
```

### 4. План терапии на сегодня (tasks)
Формируется из `timelineInterventions` на текущий `viewDay` + системные задачи:

```
План терапии на сегодня (день N):
  ⬜ [ОЖИДАЕТСЯ] Заполнить дневник питания (DIARY)
  🔴 [ПРОПУЩЕНО] Калории ≤ 2200 ккал (сейчас 2450) (KCAL)
  ✅ [ВЫПОЛНЕНО] Водный режим (FD_WTR)
  ⬜ [ОЖИДАЕТСЯ] Белковая норма (FD_PRO)
```

Типы задач:
- **DIARY** — заполнить дневник питания (🔴 если meals пусты)
- **KCAL** — не превышать лимит калорий (🔴 если сумма > 2200)
- **int-{code}-{day}** — интервенции из плана (done/not done по `completedTaskKeys`)

Цветовая маркировка в промпте: `✅ [ВЫПОЛНЕНО]`, `🔴 [ПРОПУЩЕНО]`, `⬜ [ОЖИДАЕТСЯ]`

### 5. Правила поведения AI

1. Отвечай на русском языке.
2. Будь кратким (2-4 предложения).
3. Используй данные пользователя для персонализации ответов.
4. Если вопрос касается здоровья — ссылайся на показатели пользователя и план терапии.
5. Если какое-то задание из плана отмечено 🔴 [ПРОПУЩЕНО] — мягко напомни о нём и предложи выполнить.
6. Если все задания выполнены — похвали пользователя.
7. **Задавай 1 уточняющий вопрос в конце каждого ответа**: спроси о самочувствии, выполнении задания или потребностях на завтра.
8. Будь поддерживающим и мотивирующим.
9. НЕ ставь медицинских диагнозов — рекомендую проконсультироваться с врачом.
10. Если спрашивают про питание — предложи конкретные продукты с учётом КБЖУ.

## Передача данных

### Фронтенд → Бэкенд (`POST /api/chat`)

```json
{
  "message": "текст сообщения",
  "profile": "profile_id",
  "history": [
    { "role": "user", "content": "..." },
    { "role": "assistant", "content": "..." }
  ],
  "provider": "gigachat | openai",
  "tasks": [
    {
      "key": "int-FD_WTR-3",
      "code": "FD_WTR",
      "name": "Водный режим",
      "type": "intervention",
      "done": false,
      "alert": false,
      "day": 3
    }
  ]
}
```

### Бэкенд → GigaChat/OpenAI

```json
{
  "model": "GigaChat-Max",
  "messages": [
    { "role": "system", "content": "<buildSystemPrompt(found, tasks)>" },
    { "role": "user", "content": "..." },
    { "role": "assistant", "content": "..." },
    { "role": "user", "content": "новое сообщение" }
  ],
  "max_tokens": 300,
  "temperature": 0.7
}
```
