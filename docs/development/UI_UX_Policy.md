# UI/UX Policy

## 1. Нет эмодзи — только SVG-иконки
- Все unicode-эмодзи в UI заменяются на inline `<svg>` элементы
- SVG задаются напрямую в JSX или через функцию `renderMsgText()`, которая подменяет emoji-символы на `<svg>` в строках, передаваемых в `dangerouslySetInnerHTML`
- Исключение: эмодзи в пользовательском вводе (печатаемый текст) не заменяются
- Назначенные цвета stroke/fill SVG должны быть осмысленными (зелёный для успеха, красный для ошибок, синий/фиолетовый для нейтрального и т.д.)

## 2. Высота демо-экранов — 580px
- Все контейнеры прототипов экранов внутри чата (включая `.chat-proto-chat`, `.chat-screen-viewer-body`) имеют высоту 580px
- Это обеспечивает единый вертикальный размер для всех экранов: изображения, профиля, чата и будущих прототипов

## 3. Единый стиль карточек в чате
- Карточки сообщений AI (system, tip, task, poll) имеют единую структуру: цветная левая рамка 3px, скруглённые углы 10px, внутренний отступ 8-10px
- Каждый тип карточки использует свой цвет рамки: фиолетовый (system), оранжевый (tip), по категории (task), голубой (poll)
- Время отображается мелким шрифтом справа внизу карточки

## 4. Интерактивные элементы
- Кнопки заданий (выполнено/пропущено) — SVG-иконки на цветном фоне (зелёный/красный)
- Опросы: кнопки-варианты с hover-подсветкой, выбранный вариант — фиолетовый фон, блокировка после голосования
- Состояния выполненных заданий отмечаются цветным баннером внутри карточки
