## 1. Приоритет первой итерации

**Что делаем в первую очередь?**
- A) Чат-ассистент с RAG: индексируем 23 домена → Search Agent → ответы с источниками
- B) PubMed + внешние источники: Collection Agent → наполнение базы знаний
- C) Prompt Agent: динамическая сборка промптов с инжекцией профиля + плана + знаний
- D) Knowledge Graph: граф протокол → интервенция → биомаркер

**Мой recommendation:** A → C → B → D. Начинать с Search Agent (самый быстрый ROI для чата).
