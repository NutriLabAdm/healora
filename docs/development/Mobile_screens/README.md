# Mobile Simulator — Описание экранов

## Архитектура

- **PhoneSimulator.jsx** — основной модуль телефона: шапка с навигацией по дням, переключение табов (чат/профиль/план/дневник), нижняя панель навигации.
- **PhoneSimulator.css** — стили для .phone-overlay*.
- Экраны переиспользуют DDD-модули: `ProfileView`, `DiaryView`, `PlanView`.
- Телефон позиционируется как `fixed`-оверлей (500×800px).

---

## Экран 1: Чат (Chat)

**Файл:** PhoneSimulator.jsx — раздел `phoneOverlayTab === 'chat'`

**Назначение:** AI-ассистент Healora с мультиформатным диалогом.

### Компоненты:
1. **Шапка чата** — фиолетовая полоса с названием "Healora AI", индикатор дня.
2. **Лента сообщений** — скроллируемый контейнер с сообщениями разных типов.
3. **Панель ввода** — поле текста + кнопка микрофона + кнопка отправки.

### Типы сообщений:
| Тип | Описание | Визуал |
|-----|----------|--------|
| **text (AI)** | Текст от AI, слева, серо-фиолетовый фон | Поддерживает **жирный** текст через markdown `**` |
| **text (user)** | Текст пользователя, справа, фиолетовый фон | |
| **intervention-badge** | Бейдж интервенции из плана терапии | Три состояния: ✅ Выполнено (зелёный), ❌ Пропущено (красный), ⏳ Ожидает (оранжевый). Показывает код и название интервенции |
| **food-photo** | Фото еды от пользователя | SVG-плейсхолдер тарелки еды 80×80px + подпись + КБЖУ (калории, белки, жиры, углеводы) |
| **voice** | Голосовое сообщение | Иконка микрофона + визуализация аудио (столбики гистограммы) + длительность |

### Пример диалога:
1. AI: Приветствие + Healora Score
2. AI: Список интервенций на сегодня (бейджи с реальными кодами из `chatMessages`/`timelineInterventions`)
3. AI: Запрос фото еды
4. User: Фото обеда с КБЖУ
5. AI: Анализ питания (белки/жиры/углеводы + совет)
6. User: Голосовое сообщение (0:23)
7. AI: Распознавание голоса + отчёт о прогрессе
8. User: Запрос на изменение плана
9. AI: Подтверждение изменений
10. AI: Прощание + план на завтра

### Связь с данными:
- `chatMessages` (из DigitalTwin) — интервенционные сообщения с полями `code`, `name`, `done`, `skipped`, `day`
- `timelineInterventions` — запланированные интервенции
- `profile` — данные пользователя (имя, Score, сон, шаги)
- `profile?.digital_twin_scores?.current_stars` — текущий Healora Score

## Экран 2: Профиль (Profile)

**Файл:** ProfileView.jsx

**Назначение:** Просмотр и редактирование профиля цифрового двойника.

### Секции:
- **Демография** — возраст, пол, этничность
- **Антропометрия** — рост, вес, ИМТ, талия
- **Витальные** — АД, пульс, HRV, SpO₂
- **Лабораторные** — глюкоза, холестерин, HbA1c, витамин D и др.
- **Образ жизни** — сон, стресс, шаги, вода, курение, алкоголь, активность, диета
- **Генетика** — APOE, MTHFR, лактаза, BRCA
- **Медицинская история** — лекарства, аллергии, семейный анамнез

### Режимы:
- `compact` — без "Код" колонки, для телефона
- Полный — для основной панели

## Экран 3: План (Plan)

**Файл:** PlanView.jsx

**Назначение:** План терапии с таблицей интервенций.

### Компоненты:
- **Блок врача** — пациент, врач, дата, срок (30 дней)
- **Прогресс-бар** — пройдено X/30
- **Целевые параметры** — карточки биометрических целей (старт/текущая/цель), клик — график
- **Таблица интервенций** — №, Пер (расписание в тултипе), Интервенция (20%, клик — разворот с описанием), Код, Интервал (+ % плана), редактирование
- **Данные для рекомендаций** — интерактивные чипсы полей профиля
- **Заключение врача** — текст + редактирование
- **Статус** — селектор статуса плана

### Режимы:
- `compact` — скрывает "Код", селектор шаблона, PDF/QR

## Экран 4: Дневник (Diary)

**Файл:** DiaryView.jsx

**Назначение:** Трекинг питания и самочувствия.

### Компоненты:
- **Приёмы пищи** — Завтрак, Обед, Ужин, Перекус (каждый с фото, описанием, КБЖУ, временем)
- **Самочувствие** — оценка энергии, настроения, сна, стресса, пищеварения
- **Вода** — слайдер объёма воды
- **РПП** — секция расстройств пищевого поведения (типы, частота, триггеры)
- **Комментарий** — текстовая заметка
- **Голосовая заметка** — микрофон

---

## Навигация

- **Верхние табы** — Чат / Профиль / Дневник
- **Нижняя панель** — Chat / План / Goals / Diary / Profile (SVG иконки + лейблы)
- **Навигация по дням** — ◀ День X / 30 ▶ в шапке телефона (перемотка меняет день для PlanView и DiaryView)
