Агент обновления базы знаний

Knowledge Base Update Agent — автоматическое пополнение и верификация контента

Назначение

Агент обеспечивает непрерывное пополнение и актуализацию структурированной базы знаний по нутрициологии, БАДам, лекарственным взаимодействиям и режимам питания. Источники — рецензируемые исследования, клинические рекомендации (ESPGHAN, WHO, NIH, Rospotrebnadzor) и проверенные научные датасеты.
