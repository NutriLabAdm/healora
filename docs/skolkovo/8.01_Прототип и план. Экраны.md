# Онкоскрининг. Прототип и план. Экраны

| Индекс экрана | Описание экрана + сценариев использования | Место для скриншота |
|---------------|-------------------------------------------|---------------------|
| 1 | **Главная экран (Digital Twin)** - Основной дашборд с визуализацией цифрового двойника пациента, показывающий текущие показатели здоровья, статус соблюдения протокола и рекомендации. Используется для ежедневного мониторинга здоровья, просмотра прогресса и получения персонализированных рекомендаций. | [Скриншот главной экрана] |
| 2 | **Экран чата с ИИ-ассистентом** - Интерфейс диалога с виртуальным ассистентом HEALORA для получения рекомендаций, ответов на вопросы о здоровье, поддержки и мотивации. Используется для консультаций по питанию, активности, сну и другим аспектам здоровья. | [Скриншот чата] |
| 3 | **Маршрут здоровья (Ваш Путь)** - Визуализация прогресса пациента к целевым показателям здоровья во времени. Показывает исторические данные, тренды и достижения. Используется для отслеживания долгосрочного прогресса и мотивации. | [Скриншот маршрута] |
| 4 | **Экран профиля пациента** - Информация о пациенте: демографические данные, начальные показатели, цели программы и статус. Используется для просмотра и редактирования личных данных, установки целей и просмотра истории изменений. | [Скриншот профиля] |
| 5 | **Цели программы** - Список текущих целей здоровья (вес, артериальное давление, глюкоза, сон, активность и т.д.) с прогрессом к каждой цели. Позволяет устанавливать новые цели и отслеживать достижение существующих. | [Скриншот целей] |
| 6 | **Дневник питания** - Журнал приемов пищи с анализом питательной ценности, соблюдением диетических рекомендаций и обратной связью. Используется для отслеживания потребления калорий, макронутриентов и соблюдения диетического плана. | [Скриншот дневника] |
| 7 | **Подбор протокола (Decision Flow)** - Интерактивный опрос для определения оптимального протокола лечения/профилактики на основе ответов на вопросы о состоянии здоровья, факторах риска и предпочтениях. Используется при первом настройке программы или при изменении состояния здоровья. | [Скриншот подбора протокола] |
| 8 | **Панель интервенций (нижняя панель)** - Боковая панель с доступными медицинскими, диетическими, физическими и поведенческими интервенциями, которые можно добавить в персональный план. Позволяет просматривать доступные рекомендации и перетаскивать их в план. | [Скриншот панели интервенций] |
| 9 | **Окно добавления интервенции в план** - Модальное окно, появляющееся при перетаскивании интервенции на область тела цифрового двойника, для выбора времени выполнения и добавления заметок. Используется для планирования конкретных действий в расписании дня. | [Скриншот добавления интервенции] |
| 10 | **Экран авторизации/входа** - Форма входа в систему с полями для email и пароля, а также ссылка на регистрацию. Используется для доступа к персонализированным данным и настройкам программы. | [Скриншот экрана входа] |
| 11 | **Экран регистрации** - Форма создания нового аккаунта с полями для email, пароля и, возможно, дополнительной информации о здоровье. Используется новыми пользователями для начала работы с системой. | [Скриншот регистрации] |
| 12 | **Селектор профилей** - Выпадающее меню в верхней панели для выбора между разными профилями пациентов (если у пользователя несколько профилей или в демонстрационном режиме). Используется для переключения между разными случаями или просмотра демонстрационных данных. | [Скриншот селектора профилей] |
| 13 | **Интервенция в деталях** - Всплывающая панель с подробной информацией о конкретной интервенции: описание, доказательная база, сложность выполнения, ожидаемый эффект и советы по реализации. Появляется при клике на интервенцию в плане или панели. | [Скриншот деталей интервенции] |
| 14 | **Экран редактирования плана** - Режим редактирования персонального плана здоровья с возможностью изменения времени выполнения интервенций, их частоты и добавления/удаления элементов. Используется для корректировки плана под изменяющиеся обстоятельства. | [Скриншот редактирования плана] |
| 15 | **Уведомления и напоминания** - Лента уведомлений о предстоящих интервенциях, достижении целей, необходимости выполнения действий и обратной связи по уже выполненным рекомендациям. | [Скриншот уведомлений] |
| 16 | **История выполнения плана** - Журнал выполненных и пропущенных интервенций с анализом приверженности протоколу за выбранный период времени. Используется для анализа эффективности и корректировки плана. | [Скриншот истории выполнения] |
| 17 | **Настройки учетной записи** - Экран управления профилем: изменение пароля, настройка уведомлений, подключение устройств для сбора данных (wearable), настройка конфиденциальности. | [Скриншот настроек] |
| 18 | **Экран экспорта данных** - Возможность экспортировать историю здоровья, выполненные интервенции и достижения в различных форматах (PDF, CSV) для передачи врачу или личного архива. | [Скриншот экспорта] |
| 19 | **Обучающие материалы и советы** - Библиотека статей, видео и инфографики по различным аспектам здоровья: питание, физическая активность, сон, стресс-менеджмент и профилактика заболеваний. | [Скриншот обучающих материалов] |
| 20 | **Поддержка и обратная связь** - Интерфейс для связи со службой поддержки, отправки предложений по улучшению, сообщения об ошибках и оценки качества работы системы. | [Скриншот поддержки] |
