Проблема
Решение
AI игнорирует Auto Layout
Добавьте в промпт: Apply Auto Layout to all containers. Use "Hug" for height/width, 16px padding, vertical/horizontal alignment.
Не создаёт варианты
Укажите явно: Create as Component Set with 3 named variants. Do not flatten.
Цвета/шрифты сбиваются
Задайте Figma Variables: Use local variables --primary, --bg, --text. Apply consistently.
Слайды/модалки не позиционируются
Укажите: Position modal at bottom with 24px top rounded corners. Add #00000030 overlay behind.
Прототип не цепляется
После генерации вручную переключите Interaction → While dragging / On tap → Change to + Smart Animate
